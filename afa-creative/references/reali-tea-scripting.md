# Reali-TEA 脚本法则 (Reali-TEA Scripting Playbook)

本文件是 afa-creative 的 UGC 风格视频脚本核心参考。Reali-TEA 不是一种"风格"，而是一种"反广告"的方法论——让广告看起来不像广告，从而绕过用户的心理防御。

---

## 一、 Reali-TEA 哲学

### 1.1 核心原理

人类大脑有一个"广告过滤器"——当我们识别到某个内容是广告时，会自动降低信任度和注意力。Reali-TEA 的目标是**绕过这个过滤器**，让广告内容被大脑归类为"朋友的推荐"而非"品牌的推销"。

### 1.2 四大支柱

| 支柱 | 含义 | 执行要求 |
| :--- | :--- | :--- |
| **言语不完美** | 人们说话是破碎的、重复的 | 注入填充词 `like`, `literally`, `wait—`, `okay so` |
| **行为真实感** | 打破第四面墙，微小的分心 | 直接与镜头互动，整理头发，自嘲 |
| **视觉提示** | 非完美的拍摄环境 | 手持晃动、非中心构图、环境杂乱 |
| **禁用营销词** | 消除一切广告感 | 禁止 "game-changer", "revolutionary", "must-have" |

## 二、 UGC 视频脚本结构

### 2.1 标准 15-30 秒脚本结构

| 时间段 | 元素 | 目标 | 执行要点 |
| :--- | :--- | :--- | :--- |
| **0-3 秒** | Hook (钩子) | 停止滑动 | 必须在 1 句话内制造好奇心或共鸣 |
| **3-8 秒** | Problem (问题) | 建立共鸣 | 描述受众正在经历的痛苦 |
| **8-18 秒** | Solution (方案) | 展示产品 | 自然地引入产品，展示使用过程 |
| **18-25 秒** | Proof (证据) | 建立信任 | 展示结果、数据、或他人评价 |
| **25-30 秒** | CTA (行动号召) | 驱动转化 | 简单、直接、有紧迫感 |

### 2.2 Hook 类型库（前 3 秒）

**A. 直接对话型 (Direct Address)**
- "OK so I need to tell you about this..."
- "Wait, before you scroll..."
- "I've been gatekeeping this for too long."
- "POV: you just discovered [产品]"

**B. 争议/反直觉型 (Controversial)**
- "Unpopular opinion: [行业常识] is wrong."
- "I'm going to get hate for this but..."
- "Everyone's been doing [行为] wrong."
- "Your [品类] is lying to you."

**C. 故事型 (Story)**
- "So I was [场景] when I noticed..."
- "Three months ago I was [痛苦状态]..."
- "My [朋友/同事] kept asking me about this..."
- "I didn't believe it until I tried it myself."

**D. 数据/事实型 (Data)**
- "[数字]% of [人群] don't know this."
- "I tested [数字] [品类] and this is the winner."
- "After [时间段] of using this, here are my results."
- "The [品类] industry doesn't want you to know this."

### 2.3 前 7 秒黄金法则

1. **绝对不提产品名称**：前 7 秒是建立共鸣和抛出 Hook 的，提品牌名 = 触发广告过滤器。
2. **"意外发现"叙事**：产品放在杂乱的桌子上，创作者说到痛点时顺手拿起来。禁止双手捧着产品、灯光完美的"供奉"式展示。
3. **负面 Hook 优先**：人类对负面信息的反应比正面信息更强烈。"Stop doing [mistake]" > "Try this amazing product"。

### 2.4 Problem 段落规则（3-8 秒）

**必须遵守**：
- 痛点必须是受众**正在经历**的，而非假设的
- 使用"你"而非"我们的客户"
- 描述感受，而非仅描述问题
- 最多 2 句话

**禁止**：
- 过度夸大痛苦（触发广告过滤器）
- 使用医学/法律术语（降低亲和力）
- 让人感到被攻击或羞辱

### 2.5 Solution 段落规则（8-18 秒）

**必须遵守**：
- 产品引入必须**自然**（"所以我试了这个..." > "隆重介绍我们的产品..."）
- 展示**使用过程**，而非仅展示产品外观
- 使用具体的、可感知的描述（"质地像丝绸一样滑" > "质地很好"）

**禁止**：
- 念产品说明书
- 列举超过 3 个功能点
- 使用"革命性"、"颠覆性"等过度营销词汇

### 2.6 Proof 段落规则（18-25 秒）

**证据类型优先级**（从高到低）：
1. **Before/After 对比**：最强的视觉证据
2. **具体数据**："30 天后我的 [指标] 提升了 47%"
3. **第三方背书**："连我的皮肤科医生都问我用了什么"
4. **社会认同**："已经推荐给了 5 个朋友"
5. **个人感受**："这是我用过最好的 [品类]"

### 2.7 CTA 段落规则（25-30 秒）

**遗忘式 CTA (The "Oh Yeah" CTA)**：
行动号召必须听起来像是创作者在挂断视频通话前突然想起来的一件事。
- "Oh yeah, I linked it in my bio if you want to check it out. Bye!"
- "Anyway, they're having a sale right now, link is below."
- "Comment [关键词] and I'll send you the link."

**迅速切断**：说完 CTA 后立刻切断视频，不留空白。

## 三、 长视频脚本结构 (60-90 秒)

| 时间段 | 元素 | 内容 |
| :--- | :--- | :--- |
| **0-3 秒** | Hook | 同上 |
| **3-10 秒** | Problem + Context | 更深入地描述痛苦场景 |
| **10-15 秒** | Failed Solutions | "我试过 [竞品/方法]，但是..." |
| **15-30 秒** | Discovery | "直到我发现了..." + 产品展示 |
| **30-50 秒** | Deep Demo | 详细的使用过程 + 质地/效果展示 |
| **50-60 秒** | Social Proof | 评价截图、Before/After、数据 |
| **60-75 秒** | Objection Handling | 主动回应常见疑虑 |
| **75-90 秒** | CTA | 行动号召 + 优惠信息 |

**"失败方案"段落的重要性**：提到你尝试过的其他方案（并解释为什么不够好），是 Reali-TEA 最强大的信任建设工具——它证明你是"真正的消费者"，预先回应了"为什么不用竞品？"的疑问，并通过对比让产品优势更突出。

## 四、 视觉执行规范

### 4.1 拍摄/Prompt 规则

| 元素 | 规范 |
| :--- | :--- |
| **镜头** | 手持感，轻微晃动，前置摄像头视角 |
| **光线** | 自然光优先。窗光或户外光。禁止专业打光 |
| **背景** | 真实的生活场景（卧室、浴室、厨房、办公桌） |
| **穿着** | 日常穿着，而非"拍广告"的穿着 |
| **表情** | 自然的、对话式的。禁止"库存图假笑" |
| **文本覆盖** | 必须使用平台原生字体和样式 |

### 4.2 音频规范

- **语速**：正常对话速度，不要像念稿
- **语调**：像跟朋友说话，而非做演讲
- **背景音**：轻微的环境音增加真实感
- **音乐**：如果使用，必须是平台热门音频，音量低于语音

### 4.3 剪辑节奏

- **跳剪 (Jump Cuts)**：每 3-5 秒一次，保持节奏感
- **B-Roll 插入**：产品特写、使用过程、结果展示
- **文本同步**：关键信息必须同时以文本形式出现在画面上
- **字幕**：必须有字幕（80% 用户某些时候静音观看）

## 五、 营销词汇替换表

| 禁用词汇 | 替换为 |
| :--- | :--- |
| Game-changer | "My skin looked different in like a week." |
| Revolutionary | "I don't know what kind of magic is in this, but..." |
| Must-have | "Don't quote me on this, but I think it actually works." |
| High-quality | "The texture is like... okay you need to feel this." |
| Innovative | "I've never seen anything do this before." |
| Best-seller | "Everyone keeps asking me about this." |
| Premium | "It feels expensive, if that makes sense." |

## 六、 脚本质量自检清单

在输出任何 UGC 脚本前，检查以下 10 项：

- [ ] Hook 是否在 3 秒内制造了好奇心或共鸣？
- [ ] 整个脚本是否像"朋友的推荐"而非"品牌的广告"？
- [ ] 是否使用了对话式语言（"你"而非"我们的客户"）？
- [ ] 产品引入是否自然（而非突兀的"隆重介绍"）？
- [ ] 是否展示了使用过程（而非仅展示产品外观）？
- [ ] 是否有具体的证据（数据、对比、评价）？
- [ ] CTA 是否柔和但有效（而非"Buy now!"）？
- [ ] 是否有任何营销术语需要替换为日常用语？
- [ ] 脚本长度是否适合目标平台？
- [ ] 是否考虑了静音观看的用户（文本覆盖/字幕）？

## 七、 结构化脚本输出模板

```markdown
## TikTok 脚本：[概念名称]

**目标受众**：[Audience]
**核心痛点**：[Pain Point]
**视觉风格**：手持自拍视角，自然窗光，背景是真实的卧室。

| 时间 (秒) | 视觉/动作 | 画外音/字幕 | Reali-TEA 注入 |
| :--- | :--- | :--- | :--- |
| 0-3 | 创作者调整手机，轻微皱眉 | "Okay, wait. If you're still using [竞品]..." | 动作：调整镜头。语气：略带警告 |
| 3-8 | 创作者看向镜头，叹气 | "...you literally need to stop." | 词汇：使用 "literally" |
| 8-15 | 从桌子上随意拿起产品 | "I switched to this like, two weeks ago?" | 动作：随意拿起。不确定语气 |
| 15-25 | 展示使用过程（特写但不完美） | "And my [结果] is completely different." | 禁用营销词，使用 "actually crazy" |
| 25-30 | 边走边说 | "Anyway, link in bio, bye." | 遗忘式 CTA，迅速切断 |
```
