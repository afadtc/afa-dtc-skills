# 广告文案框架与模板

> **版本**: 1.0 | **所属**: 通用创意文案知识库 | **来源**: 整合自多渠道投放场景中的通用文案知识

本文件是通用广告文案的核心参考，涵盖经典文案结构框架、按漏斗阶段的文案模板、CTA 变体库、UGC 全漏斗应用框架以及创意产出节奏指南。所有内容均为**平台无关**的通用知识，各投放平台（Meta、TikTok、Google 等）的平台特有文案规范请参见对应渠道的投放规范。

## 一、经典文案结构框架

### 1.1 PAS（Pain-Agitate-Solution）

```
Pain：直击痛点
Agitate：放大痛苦
Solution：提供解决方案
```

**适用场景**：冷流量，问题驱动型产品。当受众正在经历明确的痛苦时，PAS 是最直接有效的框架。先让受众意识到问题的存在，再放大这个问题带来的后果（不解决会怎样），最后自然引出产品作为解决方案。

### 1.2 AIDA（Attention-Interest-Desire-Action）

```
Attention：抓住注意力
Interest：激发兴趣
Desire：创造欲望
Action：推动行动
```

**适用场景**：通用框架，特别适合新品发布。AIDA 是最经典的营销漏斗文案模型，适用于几乎所有品类和平台。关键在于每一步都要有足够的"钩子"让受众继续往下读。

### 1.3 HSO（Hook-Story-Offer）

```
Hook：开头钩子
Story：品牌/用户故事
Offer：不可抗拒的报价
```

**适用场景**：UGC 风格广告，社交媒体原生内容。HSO 的核心优势在于"故事"——人类天生对故事有更高的注意力和记忆力。特别适合需要建立情感连接的品牌。

### 1.4 BAB（Before-After-Bridge）

```
Before：使用前的痛苦状态
After：使用后的理想状态
Bridge：产品就是桥梁
```

**适用场景**：效果驱动型产品（护肤、健身、教育等）。BAB 的力量在于视觉化的对比——让受众清晰地看到"现在的我"和"可能的我"之间的差距，然后产品成为连接两者的桥梁。

### 1.5 社证堆叠（Social Proof Stack）

```
数字社证："已有 50,000+ 客户"
评价社证："4.9 星好评"
权威社证："被 [媒体] 推荐"
用户社证：真实用户评价
```

**适用场景**：再营销阶段，信任建设。当受众已经对产品有初步认知但尚未转化时，堆叠多维度的社会证明可以有效消除购买犹豫。关键是每种社证类型要用不同的信任维度（数量、质量、权威、真实用户），形成"证据矩阵"。

## 二、按漏斗阶段的文案模板

### 2.1 TOFU — 认知阶段文案

**模板 1：好奇驱动**

```
主文案: [令人好奇的结果声明]
正文: 
[1-2 句痛点共鸣]
[产品如何解决，不直接推销]
[社会证明数据点]
CTA: 点击了解更多 →
```

**示例**：

```
主文案: 为什么 10 万+ 妈妈都在用这个替代传统的 [品类]？
正文:
如果你也受够了 [痛点]，你不是一个人。
我们用 [差异化方法] 重新设计了 [品类]，让 [核心利益] 变得简单。
已经有 100,000+ 家庭在用，平均评分 4.8/5。
CTA: 看看适不适合你 →
```

**模板 2：教育驱动**

```
主文案: [品类] 的 3 个常见误区（第 2 个 90% 的人都在犯）
正文:
❌ 误区 1: [常见错误]
❌ 误区 2: [常见错误]
❌ 误区 3: [常见错误]
✅ 正确做法: [你的产品/方法]
CTA: 免费获取 [品类] 指南 →
```

### 2.2 MOFU — 考虑阶段文案

**模板 3：社会证明驱动**

```
主文案: "这是我用过最好的 [品类]" — [真实用户名]
正文:
⭐⭐⭐⭐⭐ [真实评论摘录]
[产品核心卖点 1]
[产品核心卖点 2]
[限时优惠信息]
CTA: 立即购买，享受 [折扣] →
```

**模板 4：对比驱动**

```
主文案: 为什么 [你的品牌] > [品类通称]
正文:
传统 [品类]: [缺点 1] | [缺点 2] | [缺点 3]
[你的品牌]: [优点 1] | [优点 2] | [优点 3]
+ [独特卖点]
+ [保障/退款政策]
CTA: 亲自体验差距 →
```

### 2.3 BOFU — 转化阶段文案

**模板 5：紧迫感驱动**

```
主文案: ⚡ [限时优惠] — 仅剩 [时间/数量]
正文:
[产品名] — [一句话核心价值]
✅ [卖点 1]
✅ [卖点 2]
✅ [卖点 3]
🎁 限时赠送 [赠品]
⏰ [截止时间/库存信息]
CTA: 立即下单 →
```

**模板 6：风险消除驱动**

```
主文案: 不满意？全额退款，运费我们出。
正文:
我们对 [产品名] 有 100% 的信心：
✅ [卖点 1]
✅ [卖点 2]
✅ [卖点 3]
🛡️ 30 天无理由退款
📦 免费配送
⭐ [评论数] 条五星好评
CTA: 零风险试用 →
```

## 三、CTA 变体库

### 3.1 按转化目标分类

| 转化目标 | CTA 变体 | 适用漏斗阶段 |
|:---|:---|:---|
| 引流 | "了解更多 →" / "看看适不适合你 →" / "免费获取指南 →" | TOFU |
| 加购 | "加入购物车" / "先收藏，等打折" / "看看有没有你的尺码 →" | MOFU |
| 直接购买 | "立即下单 →" / "零风险试用 →" / "限时优惠，立即抢购 →" | BOFU |
| 互动 | "评论区告诉我你们想看什么" / "你觉得哪个颜色好看？" | 全漏斗 |
| 分享 | "转发给你那个需要的朋友" / "保存这个视频，迟早用得上" | TOFU/MOFU |

### 3.2 按语气风格分类

| 风格 | CTA 示例 | 适用平台 |
|:---|:---|:---|
| 正式专业 | "立即预约免费咨询" / "下载完整白皮书" | LinkedIn, Google |
| 轻松对话 | "链接在 bio 里，别怪我没告诉你" / "评论区留言，我私信你" | TikTok, Instagram |
| 紧迫感 | "最后 24 小时" / "库存只剩 XX 件" / "这个价格不会再有了" | 全平台促销期 |
| 遗忘式 | "Oh yeah, 链接在下面" / "对了，他们现在在打折" | UGC 风格内容 |

## 四、UGC 全漏斗应用框架

UGC（用户生成内容）不仅仅是"让用户拍视频"，而是一套贯穿整个营销漏斗的内容策略。

| 漏斗阶段 | UGC 类型 | 核心目标 | 内容方向 |
|:---|:---|:---|:---|
| **TOFU（认知）** | 开箱视频、第一印象 | 引发好奇、建立认知 | "我刚收到这个…" / "TikTok 让我买的…" |
| **MOFU（考虑）** | 详细测评、对比视频 | 建立信任、消除疑虑 | "用了一个月的真实感受" / "和 [竞品] 对比" |
| **BOFU（转化）** | 效果展示、限时优惠 | 推动购买决策 | "Before vs After" / "最后一天特价" |
| **售后（留存）** | 回购视频、使用技巧 | 提升复购、激发口碑 | "已经是第三瓶了" / "你们不知道的隐藏用法" |

## 五、创意产出节奏指南

### 术语定义

- **创意概念 (Creative Concept)**：一个独立的广告创意角度/Hook，如“痛点放大”或“社会认同”。
- **创意变体 (Creative Variant)**：同一概念的微调版本，如不同的标题、不同的 CTA。
- **素材条数 (Asset Count)**：最终产出的文件数量，包括同一创意的不同尺寸和平台适配。一个创意概念 × 3 种 Format × 2 种尺寸 = 6 条素材。

创意不是一次性的工作，而是持续的“假设-测试-迭代”循环。以下是不同阶段的推荐产出节奏（单位：创意概念数）。

| 阶段 | 每周新创意概念数 | 格式分布 | Hook 测试策略 |
|:---|:---|:---|:---|
| **冷启动期** | 7-10 个 | 覆盖 5+ 格式，广撒网 | 每个创意使用不同 Hook |
| **验证期** | 5-7 个 | 聚焦表现最好的 Top 3 格式 | 围绕 Top Hook 进行迭代 |
| **扩量期** | 10-15 个 | Top 格式 × 新角度/新人物 | 系统化 A/B 测试 |
| **成熟期** | 7-10 个 | 维护现有 + 探索创新 | 持续刷新，防止创意疲劳 |

**创意疲劳预警信号**：当某个创意的 CTR 连续 3 天下降超过 20%，或 CPM 上升超过 30%，说明该创意已经进入疲劳期，需要启动新一轮测试。
