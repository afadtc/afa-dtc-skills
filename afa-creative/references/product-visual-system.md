# 产品视觉系统

## 八、 产品视觉系统 (Product Visual System)

电商产品图是 DTC 品牌的核心视觉资产。afa-creative 必须能够生成完整的产品视觉包。

### 8.1 产品图类型矩阵

| 图片类型 | 用途 | 构图要求 | 光影要求 |
| :--- | :--- | :--- | :--- |
| **英雄图 (Hero Shot)** | 官网首页、广告主图 | 产品 45 度角，留出负空间，16:9 或 4:5 | 戏剧性光影，单光源，深阴影 |
| **白底图 (E-commerce Flat)** | 电商平台产品页 | 产品正面居中，纯白背景，1:1 | 均匀柔光，无阴影或极轻阴影 |
| **生活方式图 (Lifestyle)** | 社交媒体、广告 | 产品在真实使用场景中，人物互动 | 自然光，暖色调，真实感 |
| **细节图 (Detail/Macro)** | 产品页详情、轮播 | 产品局部特写，展示材质和工艺 | 微距镜头，浅景深，侧光 |
| **规模图 (Scale Shot)** | 帮助用户理解产品大小 | 产品与日常物品（手、杯子）并列 | 自然光，真实场景 |
| **开箱图 (Unboxing)** | 社交媒体、UGC | 产品从包装中取出的过程 | 自然光，手持感，Lo-Fi |

### 8.2 产品图 Prompt 构建规则

1. **材质优先**：必须明确描述产品的材质（如：`matte ceramic`, `brushed aluminum`, `organic cotton`）。
2. **反射与高光**：根据材质调整反射（如：金属需要 `specular highlights`，哑光需要 `diffused reflection`）。
3. **阴影锚定**：产品必须有接触阴影 (Contact Shadow)，否则会"漂浮"在空中。
4. **比例参照**：如果产品大小不直观，必须在画面中加入比例参照物。

### 8.3 电商平台特殊要求

| 平台 | 主图要求 | 背景要求 | 文本要求 |
| :--- | :--- | :--- | :--- |
| **Amazon** | 纯白背景 (RGB 255,255,255)，产品占 85% 面积 | 绝对纯白 | 主图禁止文本 |
| **Shopify** | 品牌自定义，推荐 1:1 或 4:5 | 品牌一致 | 允许但克制 |
| **Etsy** | 自然光生活方式图表现最佳 | 真实场景 | 允许 |
