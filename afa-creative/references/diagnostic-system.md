# 创意诊断体系

## 十二、 诊断体系 (Diagnostic System)

当创意效果不佳或生成结果不符合预期时，启动诊断模式。

### 12.1 诊断模式 1：AI 塑料感诊断 (The Slop Audit)

**触发条件**：用户反馈"图片看起来太假"、"像 AI 画的"、"不够真实"。

**诊断决策树**：
```text
图片看起来像 AI
│
├─ 色彩问题？
│  ├─ 过度饱和 -> 注入 `desaturated`, `muted tones`, 使用具体 HEX 码
│  └─ 色彩不自然 -> 锚定真实胶片色彩 `Kodak Portra 400 color science`
│
├─ 光影问题？
│  ├─ 无方向全局光 -> 注入 `single light source from upper left`, `deep shadows`
│  └─ 过度 HDR -> 注入 `natural lighting`, `no HDR`, `subtle contrast`
│
├─ 纹理问题？
│  ├─ 皮肤过于光滑 -> 注入 `natural skin texture`, `visible pores`, `freckles`
│  └─ 材质缺乏质感 -> 注入 `film grain`, `analog texture`, `slight noise`
│
├─ 构图问题？
│  ├─ 完美对称 -> 注入 `asymmetrical`, `off-center`, `candid moment`
│  └─ 背景模糊汤 -> 明确描述背景中的具体物体
│
└─ 表情问题？
   ├─ 库存图假笑 -> 注入 `mid-thought`, `genuine expression`, `looking away`
   └─ 过于完美 -> 注入 `slight imperfections`, `natural aging`, `real person`
```

### 12.2 诊断模式 2：广告疲劳诊断 (Ad Fatigue Audit)

**触发条件**：投放 skill 反馈某组广告 CTR 下降，CPA 上升。

**诊断决策树**：
```text
广告效果下降
│
├─ CTR 下降但 CVR 稳定？
│  └─ Hook 疲劳：受众对文案角度免疫
│     └─ 行动：保留视觉 Format，生成多组全新 Hook 角度
│
├─ CTR 稳定但 CVR 下降？
│  └─ 落地页不匹配：创意承诺与落地页体验脱节
│     └─ 行动：协同转化率优化对齐创意与落地页的信息
│
├─ CTR 和 CVR 同时下降？
│  └─ 视觉盲点 (Banner Blindness)：受众对整个视觉风格免疫
│     └─ 行动：保留获胜 Hook，切换到明显不同的视觉 Format
│
└─ 频率 (Frequency) 过高？
   └─ 受众池过小或预算过高
      └─ 行动：生成更多全新变体以降低单个创意的频率压力
```

### 12.3 诊断模式 3：平台政策检查 (Platform Policy Check)

**触发条件**：广告被平台拒批。

**诊断决策树**：
```text
广告被拒批
│
├─ Meta 拒批？
│  ├─ 个人特征断言 -> 改为间接表述（"你胖" -> "想更健康？"）
│  ├─ Before/After 图 -> 移除或改为"过程展示"而非"结果对比"
│  └─ 文本过多 -> 先按平台政策与版位限制减少画面内文本密度，避免文案堆叠影响审核与阅读
│
├─ Google 拒批？
│  ├─ 图像中有文本 -> 移除所有图像内文本（响应式展示广告）
│  ├─ 过度大写 -> 改为正常大小写
│  └─ 图像模糊 -> 提高分辨率，确保产品清晰
│
└─ TikTok 拒批？
   ├─ 虚假 UI 元素 -> 移除假播放按钮/假关闭按钮
   ├─ 其他平台水印 -> 重新生成无水印版本
   └─ AI 内容未披露 -> 添加"Made with AI"标签
```

### 12.4 诊断模式 4：品牌一致性诊断 (Brand Consistency Audit)

**触发条件**：用户反馈"这些图看起来不像同一个品牌"。

**诊断决策树**：
```text
品牌不一致
│
├─ creative-kit.md 是否存在？
│  ├─ 不存在 -> 优先运行“品牌视觉基建”模式
│  └─ 存在 -> 检查以下维度
│
├─ 色彩偏移？
│  └─ 检查 Prompt 中是否注入了 creative-kit.md 的 HEX 码
│
├─ 光影不一致？
│  └─ 检查 Prompt 中是否锁定了统一的光源方向和色温
│
├─ 排版混乱？
│  └─ 检查后期排版是否使用了 creative-kit.md 指定的字体
│
└─ 情绪/基调偏移？
   └─ 检查 Prompt 中是否注入了品牌情绪关键词和反模式清单
```

### 12.5 诊断模式 5：转化率诊断 (Conversion Rate Audit)

**触发条件**：创意 CTR 很高但转化率极低。

**诊断决策树**：
```text
高 CTR 低 CVR（多因一果问题，按顺序排查）
│
├─ Step 1：落地页消息一致性
│  └─ 创意承诺的内容是否在落地页首屏兑现？
│     └─ 如果不一致 → 重新对齐创意信息与落地页 Offer
│
├─ Step 2：点击诱饵/过度承诺
│  ├─ 创意是否吸引了好奇但无购买意图的人？
│  └─ 产品图是否过度美化，实物与预期不符？
│     └─ 行动：在创意中加入价格锚点或明确的产品类别信号，使用更真实的产品图
│
└─ Step 3：受众精准度
   └─ 受众是否太宽泛，导致吸引了非目标用户点击？
      └─ 行动：收窄受众定位（添加兴趣/行为层）

*注意：CTR 高 + CVR 低通常是多因一果问题，应按顺序排查，确认主因后再给出建议。*
```

### 12.6 诊断模式 6：跨平台表现差异诊断

**触发条件**：同一创意在不同平台表现差异巨大。

**诊断决策树**：
```text
跨平台表现差异
│
├─ Meta 好 TikTok 差？
│  └─ 检查是否违反了 TikTok 的 Lo-Fi 原生感原则
│     └─ 行动：重新生成手机原生视角的素材
│
├─ TikTok 好 Meta 差？
│  └─ 检查是否过于 Lo-Fi 导致在 Meta 上显得不专业
│     └─ 行动：提升视觉品质但保留真实感
│
├─ 社交好 Google 差？
│  └─ 检查 Google 展示广告是否包含了文本（硬性违规）
│     └─ 行动：生成无文本版本
│
└─ 所有平台都差？
   └─ 创意本身的 Hook 或视觉不够强
      └─ 行动：回到测试矩阵，重新构建 Hook 和 Format
```
