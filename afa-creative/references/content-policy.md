# 广告平台内容政策参考

## 十一、 广告平台内容政策参考 (Platform Content Policies)

> **温馨提示**：广告平台的内容政策时刻在更新，以下仅为常见规则的简要参考，**具体要求请以各平台官方最新政策为准**。违规可能导致广告被拒、投放延迟甚至账户受限。

### 11.1 Meta 广告内容政策参考

| 类别 | 规则 |
| :--- | :--- |
| **内容限制** | 禁止不切实际的 Before/After；禁止针对个人特征的断言（"你是不是很胖？"应改为"想要更健康？"） |
| **健康/美容** | 禁止不切实际的健康声明；禁止减肥前后对比图；补充剂广告必须包含免责声明 |
| **图像质量** | 文本覆盖尽量低于 20%；禁止过度暴露；禁止暴力/血腥 |
| **视频** | 禁止突然的音量变化；建议添加字幕（大量用户静音观看） |
| **落地页** | 广告创意必须与落地页内容一致，禁止标题党 (Bait and Switch) |

### 11.2 Google 广告内容政策参考

| 类别 | 规则 |
| :--- | :--- |
| **编辑标准** | 禁止过度大写（"FREE AMAZING DEAL TODAY"）；禁止噱头符号（"$$$ Save Big $$$"） |
| **响应式展示** | 图像必须清晰锐利；**禁止图像中包含文本**；产品必须是清晰焦点 |
| **误导性内容** | 禁止虚假声明；禁止不切实际的承诺 |

### 11.3 TikTok 广告内容政策参考

| 类别 | 规则 |
| :--- | :--- |
| **真实性** | 创意必须有 TikTok 原生感；禁止虚假的 UI 元素（假播放按钮、假关闭按钮） |
| **AI 披露** | AI 生成的内容可能需要标注"Made with AI" |
| **水印** | 禁止其他平台的水印（Instagram Logo, YouTube Logo） |

### 11.4 AI 生成内容通用披露规则

当使用 AI 生成的图像用于广告时，遵循以下最佳实践：
1. 在广告描述中包含"Image generated with AI"（如果平台要求）。
2. 绝对不要使用 AI 生成的图像来伪造真人的代言。
3. 保留所有 AI 生成资产的 Prompt 记录，以备审计。
