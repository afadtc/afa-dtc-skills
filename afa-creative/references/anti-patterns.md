# 反模式与边界处理

## 十六、 反模式与边界处理 (Anti-Patterns & Boundaries)

### 16.1 绝对禁止的操作

1. **禁止输出通用的"美丽"提示词**：永远不要使用 `beautiful`, `stunning`, `masterpiece`, `ultra-detailed`, `8K`。必须用具体的光影和材质来描述美。
2. **禁止在 Google 响应式广告中加字**：这是硬性违规，会导致拒批。
3. **禁止一次性改变多个测试变量**：在 A/B 测试迭代中，如果改变了背景，就绝对不能同时改变文案。
4. **禁止忽视平台安全区**：在生成 9:16 素材时，如果关键视觉元素位于顶部 14% 或底部 10%，视为严重失败。
5. **禁止在 AI 图像中渲染长文本**：超过 3 个词的文本必须通过后期叠加。
6. **禁止将横图裁剪为竖屏**：必须重新构思垂直构图。
7. **禁止使用营销词汇写 UGC 脚本**：`game-changer`, `revolutionary`, `must-have` 等词汇会立刻暴露广告属性。

### 16.2 常见错误纠正

| 用户请求 | 错误做法 | 正确做法 |
| :--- | :--- | :--- |
| "生成一张有大段文字的海报" | 在 AI 图像中渲染长文本 | 生成带完美留白的背景图 + 后期排版建议 |
| "把这张横图裁剪成竖屏发 TikTok" | 直接裁剪 | 重新构建 9:16 原生垂直构图 Prompt |
| "做一张超高清的完美产品图" | 使用 `8K, masterpiece, ultra-detailed` | 使用具体的相机/镜头锚定 + 反塑料感规则 |
| "做 3 个广告，一个比一个好" | 生成 3 个随机的"好看"图片 | 构建 4x3 测试矩阵，让数据决定胜者 |
| "用同一张图发所有平台" | 一图多裁 | 为每个平台重新构思原生构图 |

### 16.3 边界处理规则

1. **当用户要求生成真人照片用于广告**：必须提醒 AI 生成的人物不能用于伪造代言，建议使用 UGC 创作者的真实素材。
2. **当用户要求生成竞品 Logo 或品牌元素**：拒绝。这涉及商标侵权。
3. **当用户要求生成医疗/健康声明的视觉**：建议添加免责声明，并提醒用户自行确认平台内容政策。
4. **当创意需求超出 AI 图像生成能力**（如：复杂的信息图表、精确的数据可视化）：建议使用 Canva 或 Figma 等设计工具，afa-creative 提供视觉方向和排版建议。

### 16.4 危机模式下的创意策略

当 `crisis_mode ≠ none` 时，afa-creative 的创意策略切换到止血模式：

**当 `crisis_mode = cash_crisis` 时**：
创意策略聚焦「低成本高转化」：
1. 不建议高成本拍摄（危机期不应增加支出）
2. 聚焦 UGC 风格内容（手机拍摄、真实感、零成本）
3. 复用现有素材重新剪辑（新 Hook + 旧素材）
4. 聚焦「清仓型」创意（稀缺感、限时优惠、最后机会）
5. 不建议品牌形象类创意（危机期需要直接转化）

**当 `crisis_mode = pr_crisis` 时**：
创意策略聚焦「形象修复」：
1. 暂停所有攻击性/对比性创意（避免火上浇油）
2. 聚焦品牌价值观与社会责任类内容
3. 优先使用真实用户证言和品牌故事类素材
4. 避免幽默/讽刺语气，保持真诚、透明的语调
5. 所有创意需经过 Supervisor 审核后再发布

> **重要**：以上是「止血建议」的方向指引，不是「禁止用户做其他事」。如果用户在危机期坚持要做非止血类的事，尊重用户意愿，正常执行。

### 16.5 季节性淡季排除规则

当品牌处于季节性淡季时：
- 不自动触发危机模式（淡季销量下降是正常的）
- 使用 YoY（同比）而非环比来评估业绩趋势
- 除非用户同时有非季节性危机信号（现金流断裂、账户被封等）
