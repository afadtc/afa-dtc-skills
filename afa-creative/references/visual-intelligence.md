# 视觉智能手册 (Visual Intelligence Handbook)

本文件是创意制作流程的视觉心理学与反 AI 塑料感核心参考。所有 Prompt 构建和视觉决策应以本文件的原则为基础。

---

## 一、 视觉注意力层级 (Visual Attention Hierarchy)

人类大脑处理视觉信息具有较稳定的优先顺序，很难被主观意志完全改写。创意构图宜优先参考这一层级。

### 1.1 注意力优先级排序

| 优先级 | 视觉元素 | 进化原因 | 设计应用 |
| :--- | :--- | :--- | :--- |
| 1 (最高) | **面部/眼睛** | 镜像神经元驱动的社交信号识别 | 人物面部永远是最强的注意力锚点 |
| 2 | **运动/暗示的运动** | 威胁检测机制 | 在静态图中使用对角线、模糊边缘或动态姿势 |
| 3 | **高对比度元素** | 明暗交界处最先吸引视线 | 将核心信息放在画面中对比度最高的区域 |
| 4 | **新奇/模式中断** | 认知失调触发注意力分配 | 打破行业视觉常规 |
| 5 (最低) | **文本** | 需要调用认知资源的慢系统 | 文本永远不应该是画面的唯一焦点 |

### 1.2 "3 秒法则"构图检查

在极短浏览时间内，用户的视线路径通常会表现为：
1. **最先阶段**：先被面部、运动感或高对比度元素吸引，视线锁定在画面的核心焦点上。
2. **中间阶段**：视线沿着构图引导线移动到次要信息（产品、价值主张）。
3. **后续阶段**：视线再到达 CTA 或文本信息，完成信息传递。

### 1.3 视线引导技术

| 技术 | 描述 | 适用场景 |
| :--- | :--- | :--- |
| **引导线 (Leading Lines)** | 使用画面中的线条将视线引向焦点 | 产品英雄图、落地页 Hero Image |
| **视线方向 (Gaze Direction)** | 人物的视线方向引导观众看向同一方向 | UGC 风格广告、社交媒体帖子 |
| **大小层级 (Size Hierarchy)** | 最大的元素最先被注意到 | 粗体排版广告、信息图表 |
| **色彩孤立 (Color Isolation)** | 在单色背景中使用一个强调色突出焦点 | 产品特写、CTA 按钮 |
| **框架法 (Framing)** | 使用画面中的元素框住主体 | 生活方式图、品牌故事 |

## 二、 色彩心理学在 DTC 广告中的应用

### 2.1 色彩情绪映射

| 色彩 | 心理效应 | DTC 应用场景 | 注意事项 |
| :--- | :--- | :--- | :--- |
| **红色/赤陶色** | 紧迫感、激情、食欲 | 限时优惠、食品、CTA 按钮 | 过度使用造成视觉疲劳 |
| **蓝色/靛蓝色** | 信任、专业、冷静 | 科技、金融、健康 | 深蓝建立权威，浅蓝建立亲和力 |
| **绿色/鼠尾草色** | 自然、健康、可持续 | 有机食品、环保品牌 | 避免"绿洗"嫌疑 |
| **黑色** | 奢华、权威、极简 | 高端品牌、时尚 | 宜搭配高品质排版和留白 |
| **白色/米色** | 纯净、简约、透气 | 护肤、极简主义品牌 | 深色模式下需适配 |
| **橙色/琥珀色** | 活力、友好、行动 | CTA 按钮、年轻品牌 | 与蓝色形成互补色对比 |
| **紫色/薰衣草色** | 创造力、灵性、高端 | 美妆、创意工具 | 浅紫温和，深紫奢华 |

### 2.2 色彩对比度与可访问性

- **WCAG AA 标准**：文本与背景对比度至少 4.5:1（正常文本）或 3:1（大文本）。
- **深色模式适配**：浅色品牌主色宜准备深色模式替代色板。
- **色盲友好**：不要仅依赖红/绿色传达信息，使用形状、图标或文本辅助。

## 三、 构图法则深度指南

### 3.1 三分法则 (Rule of Thirds)

将画面分为 3x3 网格，主体放在四个交叉点之一。产品放在右下交叉点，左上留白用于文本叠加。人物面部放在左上交叉点，视线引导向右侧的产品。

### 3.2 负空间 (Negative Space)

留白不是"空"的，而是"呼吸空间"。在 DTC 广告中：
- **画面宜保留足够负空间**（尤其在需要叠加文本时）。
- 负空间的颜色应该与品牌色彩系统一致。
- 负空间可以用来暗示产品的"纯净"或"简约"属性。

### 3.3 对角线法则 (Diagonal Rule)

对角线构图创造动态感和运动暗示。适用于展示"从 A 到 B"的转变（Before/After），暗示进步或增长的概念。

## 四、 反 AI 塑料感深度指南

### 4.1 "真实感注入"三维清单

在每个 Prompt 中，宜尽量覆盖以下几个真实感维度：

**物理真实感**：
- 材质纹理：`visible fabric weave`, `slight scratches on metal surface`
- 环境互动：`condensation on glass`, `dust particles in sunlight`
- 重力和物理：`fabric draping naturally`, `liquid settling in glass`

**光影真实感**：
- 方向性光源：`single window light from left`
- 阴影一致性：阴影方向宜与主光源保持一致
- 色温一致性：暖光源 = 暖阴影，冷光源 = 冷阴影

**叙事真实感**：
- 时间线索：`morning light`, `golden hour`, `overcast afternoon`
- 使用痕迹：`well-worn leather`, `dog-eared book pages`
- 环境逻辑：背景中的物体应有存在的理由

### 4.2 相机/胶片锚定库

| 风格目标 | 推荐锚定 | 视觉特征 |
| :--- | :--- | :--- |
| **温暖的生活方式** | `Fujifilm Pro 400H`, `Contax T2` | 柔和的绿色/青色偏移，自然肤色 |
| **高端商业** | `Phase One IQ4`, `Hasselblad X2D` | 极高细节，平滑色调过渡 |
| **复古/怀旧** | `Kodak Portra 400`, `Leica M6` | 温暖色调，柔和对比度，明显颗粒 |
| **冷酷/极简** | `Sony A7R V`, `Zeiss Otus 55mm` | 锐利细节，冷色调，高对比度 |
| **Lo-Fi/UGC** | `iPhone 15 Pro`, `GoPro Hero 12` | 广角畸变，自然光，轻微过曝 |
| **戏剧性/时尚** | `Canon EOS R5`, `85mm f/1.2` | 极浅景深，奶油般散景 |

### 4.3 "AI 塑料感"自检清单

在输出任何 Prompt 前，检查以下 7 项：
- [ ] 是否使用了具体的颜色名称（而非"colorful"或"vibrant"）？
- [ ] 是否指定了明确的光源方向（而非"well-lit"）？
- [ ] 是否锚定了真实的相机/胶片（而非"high quality"或"8K"）？
- [ ] 是否描述了具体的背景物体（而非让 AI 自由发挥）？
- [ ] 是否要求了自然的不完美（纹理、颗粒、不对称）？
- [ ] 是否包含了强大的负面提示词？
- [ ] 人物表情是否有叙事性（而非"smiling"或"happy"）？

## 五、 直接响应视觉原则

广告视觉的目的不是为了获奖，而是为了卖货。

1. **结果大于过程**：展示最终成果，而非努力过程。美味菜肴 > 原始食材。
2. **具体大于宽泛**：具体细节触发身份认同。"奥斯汀 42 岁的创业者" > "一个商人"。
3. **读者优先的构图**：视线流动宜优先呈现为 主图 -> 标题 -> 正文 -> CTA。
4. **真实胜过精美**：UGC 风格的粗糙感往往带来更高转化率。

## 六、 2026 商业视觉风格词汇表

### 1. 高端极简 (Premium & Sophisticated)
**适用**：高端 DTC、设计驱动型品牌。
**关键词**：`editorial photography, luxury restraint, negative space, architectural composition, muted palette, natural materials, subtle texture, confident minimalism, soft diffused lighting.`

### 2. 温暖亲和 (Warm & Approachable)
**适用**：生活方式、健康、母婴品牌。
**关键词**：`golden hour lighting, warm tones, natural light, organic texture, authentic lifestyle context, imperfect beauty, human moments, candid, shot on 35mm film.`

### 3. 大胆破坏 (Bold & Disruptive)
**适用**：潮牌、Z 世代目标受众、强烈模式中断的广告。
**关键词**：`high contrast, unexpected color combination, surreal element, scale violation, provocative, pattern interrupt, visual tension, hard flash photography.`

### 4. 科技创新 (Technical & Innovative)
**适用**：SaaS、硬件、AI 产品。
**关键词**：`chrome textures, metallic reflections, neural network abstract, data flow, geometric precision, systematic, high-tech material, computational aesthetic, dark mode palette.`

### 5. 触感与手工 (Tactile & Handmade)
**适用**：强调工艺、反数字化的实体产品。
**关键词**：`raw ceramic, linen texture, natural wood grain, hand-thrown, wabi-sabi, craft aesthetic, material honesty, visible process marks, imperfect edges, artisan quality.`
