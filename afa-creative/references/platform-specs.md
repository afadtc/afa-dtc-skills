# 跨平台创意规范手册 (Platform Creative Specifications)

本文件是创意制作流程的平台原生适配核心参考，涵盖 Meta、Google、TikTok、LinkedIn、Pinterest 等主流平台的最新尺寸、安全区、原生视觉语言和平台政策要求。

---

## 一、 Meta (Facebook & Instagram)

### 1.1 Feed (信息流)

| 参数 | 规范 |
| :--- | :--- |
| 最佳比例 | `4:5` (1080 x 1350px)，占据手机屏幕最大垂直空间 |
| 次佳比例 | `1:1` (1080 x 1080px)，适合网格展示 |
| 最大文件大小 | 30MB (图片), 4GB (视频) |
| 视频时长 | 1 秒 - 240 分钟（具体时长应按平台规则与创意目标决定） |
| 文本限制 | 画面内文本宜保持克制，避免遮挡主体或降低可读性 |
| 格式偏好 | 轮播图（Carousel）常有较好的互动潜力，但需结合创意目标判断 |

**轮播图规范**：
- 卡片数量：按故事完整度与信息密度配置
- 每张卡片比例：1:1 或 4:5（同一组素材宜保持统一）
- 第一张图宜承担最强的 Hook 角色
- 视觉风格宜在所有卡片间保持一致

### 1.2 Stories / Reels

| 参数 | 规范 |
| :--- | :--- |
| 比例 | `9:16` (1080 x 1920px) 全屏沉浸式 |
| 顶部安全区 | 顶部 14% (约 250px) 被头像/进度条遮挡 |
| 底部安全区 | 底部 10% (约 200px) 被 CTA 按钮遮挡 |
| 核心内容区 | 中间 76% |
| 视频时长 | Stories 与 Reels 时长应按平台规则和内容节奏决定 |
| 前段开场 | 宜尽早出现视觉运动或明显的模式中断 |

### 1.3 Meta 广告平台政策要点

- 禁止不切实际的 Before/After 对比
- 禁止针对个人特征的断言（"你是不是很胖？" → "想要更健康？"）
- 健康/美容类禁止不切实际的健康声明
- 补充剂广告必须包含免责声明
- 落地页内容必须与广告创意一致

## 二、 TikTok

### 2.1 尺寸与安全区

| 参数 | 规范 |
| :--- | :--- |
| 比例 | `9:16` (1080 x 1920px) 专属 |
| 右侧安全区 | 右侧 80px 被点赞/评论/分享按钮遮挡 |
| 底部安全区 | 底部 170px 被视频描述和音乐名称遮挡 |
| 视觉重心 | 视觉重心宜避开主要遮挡区，并优先照顾中部可视区域 |
| 视频时长 | 时长应按内容节奏、信息密度与平台规则决定 |
| 最大文件大小 | 500MB |

### 2.2 TikTok 原生视觉方言

TikTok 是一个“内容优先”平台，广告应尽量贴近原生内容语感：

- **Lo-Fi 手机原生感**：过度精美的商业大片常更容易被快速划走
- **大字体原生文本覆盖**：不少用户会在静音或半静音场景下观看
- **快速剪辑**：按内容节奏保持较快切换
- **原生音效/音乐**：使用平台热门音频
- **竖屏原生**：禁止横屏裁剪

### 2.3 TikTok 广告平台政策要点

- 禁止虚假 UI 元素（假播放按钮、假关闭按钮）
- AI 生成内容可能需要标注"Made with AI"
- 禁止其他平台水印（Instagram Logo, YouTube Logo）
- 创意应尽量贴近 TikTok 原生语感

## 三、 Google Ads

### 3.1 响应式展示广告 (Responsive Display Ads)

| 参数 | 规范 |
| :--- | :--- |
| 横向尺寸 | `1.91:1` (1200 x 628px) |
| 方形尺寸 | `1:1` (1200 x 1200px) |
| 最大文件大小 | 5MB |
| **硬性规则** | **图像应尽量避免额外文本或非必要 Logo 干扰** |
| 原因 | Google 会自动叠加文本标题，图像内信息过多容易造成重叠与可读性问题 |

**图像质量要求**：
- 产品应是清晰焦点
- 背景干净简洁
- 避免拼贴或多产品杂乱展示
- 高分辨率，无模糊

### 3.2 YouTube 缩略图

| 参数 | 规范 |
| :--- | :--- |
| 比例 | `16:9` (1280 x 720px) |
| 面部表情 | 夸张的、有强烈情绪的面部特写 |
| 文本 | 文本宜尽量精简，并保证在缩略图尺寸下仍可快速识别 |
| 色彩 | 宜与 YouTube UI（白色/深灰色）形成明显对比 |
| 右下角 | 尽量不要放置重要信息（容易被视频时长标签遮挡） |

### 3.3 Google 广告平台政策要点

- 禁止过度大写（"FREE AMAZING DEAL TODAY"）
- 禁止噱头符号（"$$$ Save Big $$$"）
- 响应式展示广告图像禁止包含文本
- 禁止误导性内容和不切实际的承诺

## 四、 LinkedIn

### 4.1 尺寸规范

| 格式 | 比例 | 尺寸 |
| :--- | :--- | :--- |
| 单图帖子 | `1:1` 或 `1.91:1` | 1080x1080 或 1200x628 |
| 文档/轮播 | `4:5` 或 `1:1` | 1080x1350 或 1080x1080 |
| 视频 | `1:1` 或 `16:9` | 最长 10 分钟 |

### 4.2 LinkedIn 原生视觉方言

- **专业但不刻板**：避免"握手"或"穿西装指黑板"的库存图
- **色彩心理学**：深蓝色系、大量留白、单一强调色
- **数据可视化**：图表、统计数据、信息图表表现好
- **深色模式适配**：宜考虑 LinkedIn 日益普及的深色模式
- **PDF 轮播**：教育类内容的最佳格式

## 五、 Pinterest

### 5.1 尺寸规范

| 参数 | 规范 |
| :--- | :--- |
| 最佳比例 | `2:3` (1000 x 1500px) 垂直图像占据更多瀑布流空间 |
| 最大比例 | `1:2.1` (超过此比例会被裁剪) |
| 最小宽度 | 600px |

### 5.2 Pinterest 原生视觉方言

- **文本覆盖宜有**：用户快速滚动时，图片上的文字常是重要线索
- **无面部优先**：只展示产品或手部动作的图片被 Pin 的次数更多
- **多图拼接**：展示产品使用步骤或多种搭配的拼接图表现好
- **灵感导向**：Pinterest 用户处于"发现"模式，视觉应该激发灵感而非直接推销
- **长尾效应**：Pinterest 内容的生命周期远长于其他平台（数月 vs 数天）

## 六、 邮件营销视觉规范

### 6.1 尺寸与技术要求

| 参数 | 规范 |
| :--- | :--- |
| 最大宽度 | 600px（确保在所有邮件客户端中正确显示） |
| 头图比例 | `2:1` (600 x 300px) 或 `3:2` (600 x 400px) |
| 文件格式 | JPG (照片), PNG (图形/透明背景), GIF (动画) |
| 文件大小 | 文件大小宜尽量克制，以兼顾加载速度与清晰度 |
| Alt 文本 | 每张图片应有描述性 Alt 文本 |

### 6.2 邮件视觉特殊考虑

- **图片可能被屏蔽**：许多邮件客户端默认不加载图片，确保即使没有图片邮件仍可读
- **暗色模式**：越来越多的邮件客户端支持暗色模式，测试图片在暗色背景下的显示效果
- **Retina 适配**：为 Retina 屏幕准备 2x 分辨率的图片（1200px 宽度显示为 600px）

## 七、 跨平台适配决策树

```text
需要生成创意
│
├─ 目标是 Meta Feed？
│  └─ 4:5（1080x1350）+ 反塑料感 + 文本克制
│
├─ 目标是 TikTok / Reels？
│  └─ 9:16 (1080x1920) + Lo-Fi 原生感 + 大字幕 + 安全区
│
├─ 目标是 Google 展示广告？
│  └─ 1.91:1 (1200x628) + 1:1 (1200x1200) + 绝对无文本
│
├─ 目标是 YouTube 缩略图？
│  └─ 16:9 (1280x720) + 面部特写 + 极简文本 + 高对比
│
├─ 目标是 LinkedIn？
│  └─ 1:1 (1080x1080) + 专业感 + 数据可视化 + 深色模式适配
│
├─ 目标是 Pinterest？
│  └─ 2:3 (1000x1500) + 文本覆盖 + 灵感导向 + 无面部
│
└─ 目标是邮件？
   └─ 约 600px 宽 + 文件体积克制 + Alt 文本 + 暗色模式测试
```
