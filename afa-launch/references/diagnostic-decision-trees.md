# 启动诊断决策树

> 当启动不顺利时，避免基于恐慌和直觉的无序调整。通过**四层逻辑诊断漏斗**，从宏观到微观，系统性定位问题的根本原因。

## 1. 四层诊断漏斗

```text
四层诊断漏斗：

  第一层：宏观指标层（Macro-KPI）
  ├── 问题是"流量不足"、"转化率太低"还是"获客成本太高"？
  ├── 对比：实际销售额 vs 目标、网站流量 vs 预期、整体 CVR vs 基准
  └── 输出：问题性质定性（流量型 / 转化型 / 成本型）
       │
       ↓
  第二层：渠道创意层（Channel & Creative）
  ├── 哪个渠道、哪类创意、哪个受众出了问题？
  ├── 对比：各渠道 CPA、CTR、各创意钩子率、各受众 CVR
  └── 输出：问题渠道/创意/受众定位
       │
       ↓
  第三层：网站体验层（Funnel & UX）
  ├── 用户从哪个页面、哪个环节大量流失？
  ├── 分析：GA4 漏斗报告、热力图、加购→结账流失率
  ├── 收集：客服反馈、社媒评论、退货原因
  └── 输出：漏斗断裂点定位
       │
       ↓
  第四层：根本原因层（Root Cause）
  ├── 综合以上信息，提炼 1-3 个根本原因
  ├── 例："目标受众的痛点理解有误，导致所有创意无法引起共鸣"
  └── 输出：根因报告 + 下一步行动计划
```

## 2. 常见启动失败模式与对策

| 失败模式 | 典型症状 | 根因方向 | 对策 |
|----------|----------|----------|------|
| **流量荒漠** | 展示量极低，CPM 异常高 | 受众定义过窄 / 出价过低 / 素材被拒审 | 扩大受众 / 提高出价 / 检查素材是否符合平台政策 |
| **点击黑洞** | 展示量正常但 CTR < 0.5% | 钩子无效 / 创意与受众不匹配 | 重新测试钩子 / 调整受众定位 |
| **加购悬崖** | CTR 正常但加购率 < 2% | 落地页与广告承诺不一致 / 价格冲击 | 优化落地页一致性 / 调整定价策略 |
| **结账逃逸** | 加购正常但结账完成率 < 30% | 运费惊吓 / 支付方式不足 / 信任缺失 | 免运费门槛 / 增加支付选项 / 添加信任徽章 |
| **一单难求** | 全漏斗指标均低于基准 | 产品与市场严重脱节（无 PMF） | 启动 Kill 流程，返回产品验证 |

## 3. 决策树 A：广告无展示 / 展示量极低

```text
[症状] 广告投放 24h+，展示量 < 500
│
├── 广告是否通过审核？
│   ├── 否 → 检查素材是否符合平台政策（文字占比、敏感词、Before/After 限制）
│   │        修改后重新提交，等待 24h
│   └── 是 ↓
│
├── 出价/预算是否过低？
│   ├── 日预算 < $20 或 CBO 预算 < $50 → 提高至最低 $50/天
│   └── 预算合理 ↓
│
├── 受众规模是否过小？
│   ├── 受众 < 100K → 扩大受众：移除过多兴趣限制，使用 Broad Targeting
│   └── 受众 > 100K ↓
│
├── 是否存在受众重叠？
│   ├── 多个广告组受众重叠 > 30% → 合并广告组或使用排除规则
│   └── 无重叠 ↓
│
└── 账户是否处于学习期限制？
    ├── 新账户 / 新像素 → 降低优化目标（Purchase → ATC → View Content）
    │                      先积累数据，再逐步提升优化目标
    └── 成熟账户 → 联系平台客服排查账户异常
```

## 4. 决策树 B：有展示但无点击（CTR < 0.5%）

```text
[症状] 展示量正常（>5,000）但 CTR < 0.5%
│
├── 钩子率是否合格？（视频广告）
│   ├── 钩子率 < 20% → 钩子无效：
│   │   ├── 重新设计前 3 秒（参考钩子矩阵 §4.2）
│   │   ├── 测试不同钩子类型（恐惧/好奇/社会证明/对比）
│   │   └── 优先测试创始人第一人称开场
│   └── 钩子率 > 30% 但 CTR 仍低 ↓
│
├── 缩略图/封面是否有吸引力？（图片广告 / 视频封面）
│   ├── 封面缺乏视觉冲击 → 测试高对比度、人脸特写、文字叠加封面
│   └── 封面合理 ↓
│
├── 广告文案是否与受众匹配？
│   ├── 文案过于通用 / 行业黑话过多 → 用客户语言重写
│   │   参考阶段 0 的用户访谈原话
│   └── 文案匹配 ↓
│
└── 受众定位是否精准？
    ├── 受众过宽导致大量无关展示 → 添加核心兴趣限制或使用 Lookalike
    └── 受众合理 → 可能是产品品类本身 CTR 偏低，参考品类基准调整预期
```

## 5. 决策树 C：有点击但无转化（CVR < 1%）

```text
[症状] CTR 正常（>1.5%）但网站转化率 < 1%
│
├── 广告与落地页是否一致？
│   ├── 广告承诺 vs 落地页内容存在断裂 → 统一信息：
│   │   ├── 广告强调的卖点必须在落地页首屏出现
│   │   ├── 广告中的价格/优惠必须与落地页一致
│   │   └── 视觉风格保持连贯
│   └── 一致 ↓
│
├── 落地页加载速度是否合格？
│   ├── 移动端加载 > 3 秒 → 优化图片压缩、减少脚本、使用 CDN
│   │   （每增加 1 秒加载时间，转化率下降 7%）
│   └── 加载 < 3 秒 ↓
│
├── 价格是否造成"贴纸冲击"（Sticker Shock）？
│   ├── 广告未提及价格，用户到达后因价格放弃 →
│   │   ├── 方案 A：广告中预先披露价格，过滤低意向用户
│   │   ├── 方案 B：落地页增加价值锚定（竞品对比 / 每日成本拆解）
│   │   └── 方案 C：提供分期付款选项（Shop Pay Installments）
│   └── 价格合理 ↓
│
├── 社会证明是否充足？
│   ├── 新品无评价 / 无 UGC →
│   │   ├── 添加 Beta 测试用户评价
│   │   ├── 添加媒体/专家背书
│   │   └── 添加"已售 XX 件"实时计数器
│   └── 社会证明充足 ↓
│
└── 结账流程是否存在摩擦？
    ├── 检查项：
    │   ├── 是否强制注册？→ 开启 Guest Checkout
    │   ├── 运费是否在结账页才显示？→ 提前展示或免运费
    │   ├── 支付方式是否充足？→ 至少支持信用卡 + PayPal + Shop Pay
    │   └── 是否有安全徽章？→ 添加 SSL、退款保证、安全支付图标
    └── 结账流程顺畅 → 可能是产品本身缺乏 PMF，启动 PMF 评估
```

## 6. 决策树 D：CPA 持续高于目标

```text
[症状] 有转化但 CPA > 目标 150%
│
├── 是否仍在学习期？
│   ├── 广告组转化 < 50 次 → 等待学习期完成（通常 7 天 / 50 次转化）
│   │   期间不做重大调整，避免重置学习
│   └── 已过学习期 ↓
│
├── 是否存在"胜出者"创意？
│   ├── 无：所有创意 CPA 均高 → 回到 MVP 测试，重新测试新假设
│   └── 有：部分创意 CPA 达标 ↓
│
├── 胜出者预算是否已集中？
│   ├── 预算仍均匀分配 → 关停 CPA > 目标 130% 的创意
│   │   将预算集中到胜出者
│   └── 已集中 ↓
│
├── 受众是否已饱和？
│   ├── 频次 > 2.5 且 CPA 持续上升 → 扩展受众：
│   │   ├── 扩大 Lookalike 比例（1% → 3% → 5%）
│   │   ├── 测试新的兴趣定位
│   │   └── 测试新的地理区域
│   └── 频次 < 2.5 ↓
│
└── 是否需要跨渠道调整？
    ├── 仅在 1 个渠道测试 → 扩展到其他渠道（TikTok / Google）
    │   不同渠道的用户行为模式不同，可能找到更低 CPA 的渠道
    └── 多渠道均高 CPA → 启动 PMF 评估，可能是产品-市场匹配问题
```

## 7. 决策树 E：启动后增长停滞

```text
[症状] 首周数据良好，但第 2-3 周增长停滞或下滑
│
├── 是否出现创意疲劳？
│   ├── CTR 周环比下降 > 20% 且频次 > 2.5 →
│   │   ├── 立即启动创意焕新
│   │   ├── 基于胜出者制作 5+ 变体
│   │   └── 测试 3+ 全新钩子概念
│   └── CTR 稳定 ↓
│
├── 是否遇到预算天花板？
│   ├── 日预算增加但转化未同步增长 →
│   │   ├── 每次加预算不超过 20%
│   │   ├── 加预算后等待 3 天再评估
│   │   └── 考虑横向扩展（新广告组 / 新渠道）而非纵向加码
│   └── 预算未触顶 ↓
│
├── 市场是否出现外部变化？
│   ├── 竞品加大投放 / 季节性变化 / 平台算法调整 →
│   │   ├── 扫描竞品最新动态
│   │   ├── 参考季节性调整因子
│   │   └── 调整出价策略和受众定位
│   └── 外部环境稳定 ↓
│
└── 是否已触达可触达市场的天花板？
    ├── 品类 TAM 较小 / 受众已充分覆盖 →
    │   ├── 扩展到相邻品类受众
    │   ├── 开拓新的地理市场
    │   └── 从获客转向提升客单价和复购
    └── TAM 充足 → 综合诊断，可能是多因素叠加，逐一排查以上所有节点
```
