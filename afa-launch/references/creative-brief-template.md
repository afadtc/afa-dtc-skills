# 创意简报模板

> 本文档为产品启动流程的参考文件，提供交付给设计师、视频制作者或 UGC 创作者的标准化创意 Brief 模板。
> **使用边界**：下方模板默认按用户可见口径书写，可填写团队名称、项目负责人姓名或业务角色；不得在最终交付给外部协作者的版本中直接暴露内部模块代号、路由标签或系统字段。

---

## 一、创意 Brief 总览模板

```markdown
# 创意 Brief：[Brand Name] — [Campaign / Hypothesis Name]

**Date:** [Date]
**Prepared by:** [Team / Owner Name]
**Document Version:** [Version]

---

## 1. Background & Objective

**Brand:** [Brand Name]
**Product:** [Product Name + SKU]
**Price:** $[X]
**Launch Stage:** [Pre-launch / Launch Day / Post-launch Optimization]

**Campaign Objective:**
[One sentence: What do we want the viewer to do after seeing this ad?]
Example: "Drive first-time purchases from cold audiences within an acceptable CPA range for this launch stage"

**Target Audience:**
- Demographics: [Age, Gender, Location]
- Psychographics: [Interests, Values, Lifestyle]
- Pain Point: [The specific problem this audience faces]
- Current Solution: [What they're currently using / doing]

---

## 2. Core Message (Hypothesis Being Tested)

**Hypothesis ID:** [H1 / H2 / H3]
**Value Proposition:** [One sentence — the single most important thing to communicate]
**Supporting Points:**
1. [Proof point / feature / benefit #1]
2. [Proof point / feature / benefit #2]
3. [Proof point / feature / benefit #3]

**Tone of Voice:** [Conversational / Authoritative / Playful / Urgent]
**Brand Personality:** [a short set of adjectives]

---

## 3. Creative Specifications

### 3.1 Format Requirements

| Platform | Format | Aspect Ratio | Duration | File Size |
|----------|--------|-------------|----------|-----------|
| Meta Feed | Video | 1:1 or 4:5 | 15-30s | < 30MB |
| Meta Stories | Video | 9:16 | 15s | < 30MB |
| TikTok | Video | 9:16 | 15-60s | < 287MB |
| Google Display | Image | 1200×628 | Static | < 150KB |

### 3.2 Creative Variants Needed

| Variant | Hook Type | Format | Platform |
|---------|-----------|--------|----------|
| V1 | [Primary hook angle] | [Preferred format] | [Priority platform] |
| V2 | [Secondary hook angle] | [Preferred format] | [Priority platform] |
| V3 | [Social proof or demo angle] | [Preferred format] | [Priority platform] |
| V4 | [Alternative visual angle] | [Preferred format] | [Priority platform] |
| V5 | [Optional challenger angle] | [Preferred format] | [Priority platform] |

---

## 4. Video Script Framework

### Opening Hook (opening moment) — CRITICAL
[Exact words / actions for the opening moment]
- Must stop the scroll immediately
- No logo, no brand name, no generic intro
- Start with the pain point or a provocative statement

### Problem Agitation (early body)
[Expand on the pain — make the viewer feel understood]
- Use specific, relatable scenarios
- Use the audience's own language (from interviews)

### Solution Introduction (mid body)
[Introduce the product as the answer]
- Show the product in use
- Highlight the key differentiator
- Keep it simple — one main benefit

### Proof / Social Proof (proof segment)
[Why should they believe you?]
- Testimonial clip / text overlay
- Before / after results
- Number of customers / ratings
- Expert endorsement

### Call to Action (closing segment)
[Clear, specific, urgent]
- "Shop now and get [offer]"
- "Link in bio — [X]% off ends [date]"
- Include urgency element (limited time / limited stock)

### On-Screen Text Overlay Timeline
| Timestamp | Text |
|-----------|------|
| 0s | "[Hook text — large, bold]" |
| 5s | "[Problem text]" |
| 10s | "[Product name + key benefit]" |
| 18s | "[Social proof stat]" |
| 25s | "[CTA + offer code]" |

---

## 5. Visual Guidelines

**Must Include:**
- [ ] Product clearly visible within first 10 seconds
- [ ] Human face / hands interacting with product
- [ ] Text overlays for key messages (many watch on mute)
- [ ] Brand colors in CTA / end card
- [ ] Captions / subtitles throughout

**Must Avoid:**
- [ ] Logo in first 3 seconds (kills engagement)
- [ ] Stock footage or overly polished "brand" aesthetic
- [ ] More than 3 text lines on screen simultaneously
- [ ] Music louder than voice
- [ ] Before / after claims without disclaimer (if applicable)

**Reference Ads:**
[Include a small set of links to ads with similar style / energy that performed well]
1. [URL] — "Like this energy and pacing"
2. [URL] — "Like this hook approach"
3. [Optional URL] — "Like this product demo style"

---

## 6. Deliverables & Timeline

| Deliverable | Quantity | Due Date | Notes |
|-------------|----------|----------|-------|
| Raw footage | [X] clips | [Date] | Unedited, multiple takes |
| Edited videos | [X] versions | [Date] | With and without text overlays |
| Static images | [X] versions | [Date] | For carousel / display |
| Thumbnail options | 3 per video | [Date] | High contrast, face visible |

---

## 7. Success Metrics

This creative will be evaluated against:
- **Hook Rate:** Compare against the current creative batch, platform norms, and launch-stage expectations
- **CTR:** Compare against the current creative batch and channel baseline
- **CPA:** Judge within an acceptable acquisition cost range for this launch stage
- **ROAS:** Use as a secondary efficiency signal during launch when attribution is sufficiently stable

Minimum data before evaluation: wait until there is enough delivery volume and runtime to avoid judging on noisy early data.
```

## 二、UGC 创作者 Brief 模板（简化版）

```markdown
# UGC 创作者 Brief：[Brand Name] — [Product Name]

Hi [Creator Name],

Thanks for partnering with us. Here is everything you need for this brief.

## The Product
[A short explanation of what the product is and does]

## Your Mission
Create a short-form video that feels like you are telling a friend about this product. We do not want an overly polished ad; we want authentic, real, relatable delivery.

## The Key Message
"[One sentence value proposition]"

## Script Outline
1. **Hook (0-3s):** Start with "[suggested opening line]" — or use your own words that feel natural
2. **Problem (3-8s):** Talk about [the pain point] — have you experienced this? Use your own story
3. **Solution (8-15s):** Show the product, explain how it solves the problem
4. **Proof (15-20s):** Share your honest experience / results
5. **CTA (20-25s):** "Check it out, link in bio" or similar

## Do's
- Film vertically (9:16)
- Use good lighting (natural light is great)
- Show the product clearly
- Be yourself — your personality is why we chose you
- Film multiple takes so we have options

## Don'ts
- Do not start with "Hey guys" or the brand name
- Do not use a ring-light look that feels overly influencer-staged
- Do not read from a script — bullet points are fine
- Do not add music or filters — post-production will handle that

## Deliverables
- A small set of video versions (with different hooks but the same core message)
- Raw footage files (no editing needed)
- Deadline: [Date]

## Shipping
Product will arrive by [Date]. Please film within an agreed window after receiving it.

Questions? Please DM us anytime.
```

## 三、静态图片广告 Brief 模板

```markdown
# 静态广告 Brief：[Brand Name] — [Campaign Name]

## Format & Specs
- Sizes needed: 1080×1080 (Feed), 1080×1920 (Stories), 1200×628 (Display)
- File format: PNG or JPG
- Max file size: 150KB (Display), 30MB (Social)

## Layout Structure

### Version A: Problem-Solution
```
┌─────────────────────┐
│  [Bold headline —    │
│   the problem]       │
│                      │
│  [Product image      │
│   in context]        │
│                      │
│  [Key benefit text]  │
│                      │
│  [CTA button]        │
│  [Offer / urgency]   │
└─────────────────────┘
```

### Version B: Social Proof
```
┌─────────────────────┐
│  ★★★★★ [Rating]     │
│  "[Customer quote]"  │
│                      │
│  [Product image]     │
│                      │
│  [Brand logo small]  │
│  [CTA button]        │
└─────────────────────┘
```

### Version C: Comparison
```
┌──────────┬──────────┐
│  Before  │  After   │
│  [Image] │  [Image] │
│          │          │
│  ✗ Pain  │  ✓ Gain  │
│  ✗ Pain  │  ✓ Gain  │
├──────────┴──────────┤
│  [CTA + Offer]      │
└─────────────────────┘
```

## Copy Options (A/B Test)
- Headline A: "[Problem-focused]"
- Headline B: "[Benefit-focused]"
- Headline C: "[Social proof-focused]"
- CTA: "Shop Now" / "Try It Risk-Free" / "See Details" / [launch-specific offer CTA if applicable]
```

## 四、内部映射说明（仅供系统使用，不直接进入用户交付物）

> 如需在内部继续拆分责任，可把 Brief 准备与汇总环节映射到启动统筹流程，把后续投放细化映射到对应渠道执行流程；但任何对外发出的 Brief 都应只显示团队名称、负责人姓名或业务角色，不得直接写出内部模块 ID。
