# 启动核心框架与行业基准

## 1. 创意焕新持续优化系统

广告创意通常存在明显的“保鲜期”。创意与开场钩子常对广告表现产生很大影响，因此，创意的持续迭代往往不是“锦上添花”，而是启动阶段的重要基础动作。

### 1.1 创意焕新节奏

```text
制度化的创意迭代节奏：

  Week 1（启动周）：
  ├── 投放 MVP 测试矩阵中的初始创意
  ├── 核心目标：找到首批更有希望的"胜出者"
  └── 产出：按预算与制作能力铺开一组可比较的创意样本

  Week 2（焕新周）：
  ├── 胜出者预算可在可控范围内逐步放大
  ├── 基于胜出者制作若干变体（换背景/换演员/换节奏）
  ├── 同时测试少量全新钩子概念
  └── 产出：补充一批新创意进入测试

  Week 3+（持续迭代）：
  ├── 按创意疲劳速度与制作产能持续测试新概念
  ├── 以阶段性节奏向现有广告活动补充新素材
  ├── 监控创意疲劳信号（如 CTR 周环比明显下滑）
  └── 对疲劳创意及时替换，不恋战
```

### 1.2 创意疲劳预警指标

| 指标 | 相对稳定 | 需要关注 | 高风险 |
|------|------|------|------|
| CTR 周环比变化 | 小幅波动，仍在可接受区间 | 明显下滑，需要结合素材新鲜度判断 | 持续显著下滑，需优先排查疲劳或承接问题 |
| CPA 周环比变化 | 小幅波动，仍在可接受区间 | 明显上升，需要复核流量质量与承接 | 持续显著上升，需优先排查素材疲劳或投放结构 |
| 频次 (Frequency) | 与账户阶段基本匹配 | 持续走高，需要结合受众规模判断 | 明显偏高，且与成本恶化同时出现 |
| 钩子率趋势 | 稳定或上升 | 缓慢下降 | 急剧下降 |

### 1.3 创意类型效果基准

```text
2025-2026 DTC 启动期创意类型效果参考：

  类型                     │ 相对表现倾向                 │ 常见适用阶段
  ─────────────────────────┼──────────────────────────────┼──────────────
  创始人第一人称视频        │ 常作为较强的信任建立素材       │ 全阶段
  脚本化 UGC               │ 常兼顾原生感与转化清晰度       │ 引爆 + 优化
  问题→解决方案叙事        │ 更适合痛点清晰的品类           │ 引爆
  前后对比展示             │ 更适合结果可视化的场景         │ 引爆 + 预热
  原始 UGC                 │ 原生感强，但稳定性不一定最佳   │ 优化阶段
  传统品牌广告             │ 常更适合成熟品牌放大或形象层投放 │ 启动期需谨慎使用

  关键洞察：
  ├── 创始人视频常比传统品牌广告更容易建立信任
  ├── 脚本化 UGC 往往比完全原始的 UGC 更容易复制与放大
  └── 启动期通常更适合优先使用信任感强、表达清晰的创意类型
```

## 2. 行业基准数据（2025-2026）

```text
DTC 品牌启动期关键基准（仅作参考对照，不作统一红线）：

  指标类别          │ 具体指标                │ 参考说明
  ──────────────────┼────────────────────────┼────────────────────────────
  启动表现          │ 首月销售额 vs 预期      │ 更适合与内部目标、历史样本和阶段期望对比
  ──────────────────┼────────────────────────┼────────────────────────────
  广告创意          │ 创意对表现的贡献度      │ 创意与钩子常对结果有较大影响
```

### 2.1 三维验证评估（Go/No-Go 决策）

在启动阶段 0 结束时，使用以下三维评估决定是否进入引爆阶段：

| 维度 | 评估方法 | 可推进信号 | 暂缓信号 |
|------|----------|---------|------------|
| 需求验证 | 用户访谈 + 问卷 | 支付意愿接近目标价格区间，且痛点清晰 | 支付意愿明显低于可行价格带，且痛点模糊 |
| 产品验证 | Beta 测试 | 用户反馈、采纳与复购意向呈现积极信号 | 用户反馈偏弱，功能采纳与复购意向不足 |
| 竞品验证 | 竞品扫描 | 差异化机会明确，竞品未覆盖的需求存在 | 市场高度拥挤，且暂未发现明确差异化空间 |
