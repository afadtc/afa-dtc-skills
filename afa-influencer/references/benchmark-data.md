## 1. 平台参与度基准 (Engagement Rate Benchmarks)

参与度是衡量创作者受众质量的核心指标。不同平台和不同粉丝量级的基准差异巨大。

### 1.1 Instagram

| 粉丝量级 | 参与度基准 | 优秀标准 | 预警信号 |
| :--- | :--- | :--- | :--- |
| 纳米 (1K - 10K) | 4.0% - 5.0% | > 6.0% | < 2.0% |
| 微型 (10K - 50K) | 2.0% - 3.0% | > 4.0% | < 1.5% |
| 中型 (50K - 500K) | 1.5% - 2.0% | > 2.5% | < 1.0% |
| 大型 (> 500K) | 0.8% - 1.2% | > 1.5% | < 0.5% |

### 1.2 TikTok

| 粉丝量级 | 参与度基准 | 优秀标准 | 预警信号 |
| :--- | :--- | :--- | :--- |
| 纳米 (1K - 10K) | 8.0% - 10.0% | > 12.0% | < 5.0% |
| 微型 (10K - 50K) | 5.0% - 7.0% | > 8.0% | < 3.0% |
| 中型 (50K - 500K) | 4.0% - 5.0% | > 6.0% | < 2.5% |
| 大型 (> 500K) | 2.5% - 4.0% | > 5.0% | < 1.5% |

### 1.3 YouTube

YouTube 的参与度计算通常为 (点赞+评论) / 观看量，而非粉丝数。

| 指标 | 基准 | 优秀标准 |
| :--- | :--- | :--- |
| 参与度 (点赞+评论/观看量) | 0.3% - 0.5% | > 1.0% |
| 平均观看率 (观看量/粉丝数) | 2% - 5% | > 10% |

## 5. ROI 计算模型

### 5.1 基础 ROI 公式

```
ROI = (活动带来的总收入 - 活动总成本) / 活动总成本 × 100%
```

### 5.2 成本构成 (Total Costs)

| 成本项 | 说明 |
| :--- | :--- |
| 创作者固定费用 | 按帖子/视频支付的固定金额 |
| 联盟佣金支出 | 按销售额比例支付的佣金 |
| 产品样品成本 | 赠送产品的 COGS + 运费 |
| 折扣码利润折损 | 专属折扣码带来的利润减少 |
| 平台/代理服务费 | 联盟追踪平台或代理机构的费用 |
| 内部人力成本 | 管理创作者关系的人力投入 |

### 5.3 收入归因 (Revenue Attribution)

**直接归因**：通过专属联盟链接或折扣码追踪到的直接销售额。这是最可靠的归因方式。

**间接归因（光环效应）**：活动期间品牌词搜索量提升带来的自然流量销售增长、社交媒体粉丝增长带来的长期价值、通过白名单投放带来的额外销售额。间接归因难以精确量化，但通常占总影响的 30%-50%。

### 5.4 ROI 基准

| 合作类型 | 目标 ROI | 优秀 ROI |
| :--- | :--- | :--- |
| 纯品牌曝光合作 | > 1.0x | > 2.0x |
| 产品置换 (Seeding) | > 3.0x | > 5.0x |
| 联盟营销 | > 4.0x | > 8.0x |
| 白名单投流 | > 2.5x | > 5.0x |
| 品牌大使项目 | > 3.0x (6个月周期) | > 6.0x |
