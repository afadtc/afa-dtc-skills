# 红人合作风险管理参考指南

> 本文件是达人合作流程的核心参考文件之一，提供 FTC 披露最佳实践、知识产权保护、平台政策和抽奖活动的参考信息。
>
> **重要声明：本文件仅提供一般性参考信息，不构成法律建议。具体要求请咨询专业律师。**

---

## 1. FTC 广告披露最佳实践参考

### 1.1 核心法律原则

美国联邦贸易委员会 (FTC) 要求：只要品牌与创作者之间存在"物质联系"（Material Connection），就应向消费者清晰披露。物质联系包括但不限于：支付现金报酬、赠送免费产品、提供折扣或特殊待遇、存在雇佣或家庭关系、提供联盟佣金。

### 1.2 各平台披露标准

| 平台 | 推荐披露方式 | 位置要求 | 注意事项 |
| :--- | :--- | :--- | :--- |
| **Instagram** | 使用"Paid partnership"标签 + `#ad` | 文案前部、折叠前更稳妥 | 仅在 Stories 中使用"Paid partnership"标签通常不够，宜同时在文案中披露 |
| **TikTok** | 使用"Paid partnership"标签 + 视频内口头披露 | 视频前段更稳妥 | 仅在描述中加 `#ad` 通常不够，宜在视频中口头说明 |
| **YouTube** | 使用"包含付费推广"标签 + 视频内口头披露 | 视频前段更稳妥 | 长视频通常宜在开头和中段重复披露一次 |
| **博客/文章** | 在文章开头添加披露声明 | 第一屏或第一段前更稳妥 | 不宜放在文末或侧边栏 |

### 1.3 披露语言示例

**推荐使用的披露语言**：
- `#ad` 或 `#advertisement`
- `#sponsored` 或 `Sponsored by [Brand]`
- `Paid partnership with [Brand]`
- `[Brand] gifted me this product`（产品置换场景）

**禁止使用的模糊语言**：
- `#collab` 或 `#partner`（过于模糊，消费者可能不理解这是付费合作）
- `#sp`（缩写不够清晰）
- `Thanks to [Brand]`（不明确是否为付费关系）

## 2. 知识产权 (IP) 与内容授权

### 2.1 内容所有权

默认情况下，创作者拥有其创作内容的版权。品牌如需使用这些内容，应通过合同获得明确的授权。

### 2.2 使用许可 (Usage Rights) 分级

| 授权级别 | 使用范围 | 典型费用溢价 |
| :--- | :--- | :--- |
| **基础授权** | 仅限创作者自己的社交媒体发布 | 包含在基础费用中 |
| **品牌自有渠道** | 品牌官网、社媒、邮件中转发/嵌入 | 通常较基础授权存在明显溢价，具体幅度取决于使用范围与期限 |
| **付费广告投放** | 用于 Meta、TikTok、Google 等平台的付费广告 | 通常比品牌自有渠道授权溢价更高，需结合投放周期与预算单独谈判 |
| **全渠道授权** | 包括线下物料、包装、电视广告等 | 一般为最高等级授权，费用应按媒介范围、期限和排他性单独核算 |

### 2.3 白名单授权 (Whitelisting)

白名单授权是指品牌获得创作者的授权，通过创作者的社交媒体账号直接投放广告。这种方式的广告效果通常优于品牌自有账号的广告，因为它保留了创作者的真实性和社交证明。

**Meta 平台**：通过 Branded Content Ads，品牌可以将创作者的帖子作为广告素材，通过品牌的广告账户进行投放和优化。

**TikTok 平台**：通过 Spark Ads，品牌可以将创作者的原生视频作为广告素材，保留原始的点赞、评论和分享数据。

## 3. 营销声明与敏感词审查

### 3.1 高风险声明类别

| 声明类别 | 风险等级 | 示例 | 参考建议 |
| :--- | :--- | :--- | :--- |
| **健康功效** | 极高 | "治愈痤疮"、"减肥很多" | 应有科学证据支持，否则改为"有助于改善" |
| **绝对化表述** | 高 | "效果非常稳定"、"持久度明显提升" 一类容易被夸大解读的说法 | 可改为"经测试，部分用户表示..." |
| **对比声明** | 中 | "比 [竞品] 好很多" | 应有可验证的第三方数据支持 |
| **环保声明** | 中 | "完全环保"、"零碳" 等绝对化环境主张 | 应有权威认证支持 |

### 3.2 安全改写工作流

当创作者的文案包含可能触发平台审核红线的激进表述时，使用以下改写策略：

**原则**：在保留商业说服力的同时，替换敏感词汇，降低封号风险。

**改写示例**：
- 原文："这款面霜能消除你所有的皱纹" → 改写："在品牌测试中，部分用户反馈细纹有改善；正式对外使用前应补充真实测试范围与证据来源"
- 原文："保证在短期内明显减重" → 改写："搭配健康饮食和运动，帮助你更轻松地管理体重"

## 4. 抽奖与竞赛注意事项

### 4.1 非法赌博三要素

如果一个活动同时包含以下三个要素，则可能构成非法赌博：

1. **奖品 (Prize)**：有价值的奖励。
2. **运气 (Chance)**：获奖取决于运气而非技能。
3. **对价 (Consideration)**：参与者需要付出代价（如购买产品、支付费用）。

### 4.2 参考策略

**抽奖活动 (Sweepstakes)**：消除"对价"要素。通常应提供"无需购买的替代参与方式 (AMOE)"，如邮寄明信片或在线免费注册。

**竞赛活动 (Contests)**：消除"运气"要素。获奖通常应基于技能（如最佳照片、最有创意的视频），并由评委评判。

### 4.3 建议包含的法律声明

所有抽奖和竞赛活动通常应包含完整的官方规则，包括：活动起止日期、参与资格（年龄、地域限制）、参与方式、奖品描述和价值、获奖概率（如适用）、获奖者通知方式、免责声明（"本活动与 Instagram/TikTok 无关"）。
