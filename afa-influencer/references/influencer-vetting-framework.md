# 创作者 3C 深度评估模型与尽职调查清单

> 本文件是 afa-influencer 的核心参考文件之一，提供创作者筛选的完整方法论、评分体系和尽职调查流程。

---

## 1. 理想创作者画像 (ICP) 构建

在开始搜索之前，必须与品牌方明确以下维度，构建一个清晰的"理想创作者画像"。这一步骤是整个筛选流程的基础，直接决定了后续搜索的效率和合作的成功率。

### 1.1 受众匹配维度

创作者的粉丝画像必须与品牌的目标客户高度重合。需要确认的关键数据包括：

| 维度 | 关键问题 | 数据来源 |
| :--- | :--- | :--- |
| **年龄分布** | 粉丝的年龄段是否与目标客户一致？ | 创作者后台数据截图 |
| **性别比例** | 男女比例是否与产品受众匹配？ | 创作者后台数据截图 |
| **地理位置** | 粉丝主要集中在哪些国家/城市？ | 创作者后台数据截图 |
| **兴趣标签** | 粉丝的兴趣爱好是否与产品品类相关？ | 第三方分析工具 |
| **消费能力** | 粉丝的消费水平是否能承受产品定价？ | 内容中展示的生活方式推断 |

### 1.2 价值观契合维度

创作者的个人价值观、生活方式和公开发言必须与品牌理念一致。这是品牌安全的第一道防线。需要检查的内容包括创作者过去 12 个月内的公开发言、转发内容和参与的话题讨论，确保没有涉及种族歧视、性别歧视、极端政治倾向或其他可能引发公关危机的言论。

### 1.3 内容垂直度维度

创作者的内容领域必须与产品高度相关。一个拥有 50 万粉丝的泛娱乐博主，其带货能力可能远不如一个拥有 5 万粉丝的垂直领域专家。垂直度越高，受众的购买意图越强，转化率越高。

## 2. 3C 深度评估模型

对于每一个通过 ICP 初筛的候选人，必须通过 3C 模型进行严格的深度审查。每个维度都有明确的评分标准。

### 2.1 Content (内容质量与风格) — 权重 30%

| 评估子项 | 评分标准 (1-5 分) | 关键观察点 |
| :--- | :--- | :--- |
| **视觉质量** | 5=专业级 4=优秀 3=良好 2=一般 1=粗糙 | 画面清晰度、构图、色彩一致性 |
| **叙事能力** | 5=引人入胜 4=流畅 3=尚可 2=平淡 1=混乱 | 是否能将产品自然融入故事 |
| **品牌契合度** | 5=完美匹配 4=高度契合 3=基本契合 2=略有偏差 1=明显冲突 | 过往合作品牌的调性 |
| **原创性** | 5=高度原创 4=有创意 3=中规中矩 2=模仿痕迹重 1=搬运 | 内容是否有独特视角 |
| **发布频率** | 5=每日 4=每周3-4次 3=每周1-2次 2=不定期 1=长期不更新 | 活跃度和持续性 |

**Content 综合评分** = 各子项加权平均 × 30%

### 2.2 Community (社区质量与真实性) — 权重 40%

社区质量是最关键的评估维度，因为它直接反映了创作者对受众的真实影响力。

| 评估子项 | 评分标准 (1-5 分) | 关键观察点 |
| :--- | :--- | :--- |
| **参与度** | 5= >5% 4= 3-5% 3= 2-3% 2= 1-2% 1= <1% | (点赞+评论+分享)/粉丝数 |
| **评论质量** | 5=深度讨论 4=有意义回复 3=简短回复 2=表情为主 1=垃圾评论 | 评论是否反映真实互动 |
| **受众真实性** | 5=完全真实 4=极少可疑 3=有少量可疑 2=明显异常 1=大量假粉 | 粉丝增长曲线、僵尸粉比例 |
| **创作者回复率** | 5=积极回复 4=经常回复 3=偶尔回复 2=很少回复 1=从不回复 | 创作者是否与粉丝互动 |
| **社区氛围** | 5=积极正面 4=友好 3=中性 2=偶有争议 1=负面/有毒 | 粉丝之间的互动质量 |

**Community 综合评分** = 各子项加权平均 × 40%

**假粉检测红旗信号**：
- 粉丝数在短时间内出现不自然的暴涨（如一天增长数万）。
- 大量粉丝的头像为默认头像或无内容。
- 评论中充斥着与内容无关的通用短语或表情。
- 点赞数与评论数比例严重失调（如 10 万点赞但只有 5 条评论）。

### 2.3 Conversion (转化潜力与商业记录) — 权重 30%

| 评估子项 | 评分标准 (1-5 分) | 关键观察点 |
| :--- | :--- | :--- |
| **带货历史** | 5=多次成功案例 4=有成功案例 3=有合作经验 2=很少合作 1=无经验 | 过往赞助内容的表现 |
| **CTA 能力** | 5=自然有力 4=清晰 3=有但生硬 2=模糊 1=无 CTA | 是否能有效引导粉丝行动 |
| **商业意识** | 5=非常专业 4=理解规则 3=基本了解 2=需要教育 1=完全不懂 | 对联盟链接、折扣码的理解 |
| **平台契合度** | 5=完美匹配 4=高度适合 3=基本适合 2=略有偏差 1=不适合 | 平台是否适合产品转化 |

**Conversion 综合评分** = 各子项加权平均 × 30%

### 2.4 综合评分与决策矩阵

**总分** = Content 评分 + Community 评分 + Conversion 评分

| 总分区间 | 决策建议 |
| :--- | :--- |
| **4.0 - 5.0** | 强烈推荐：立即启动合作谈判，考虑长期品牌大使合作 |
| **3.0 - 3.9** | 推荐：适合单次推广或联盟合作，观察表现后决定是否升级 |
| **2.0 - 2.9** | 谨慎：仅考虑产品置换 (Seeding)，不建议付费合作 |
| **< 2.0** | 不推荐：不符合品牌标准，跳过 |

## 3. 尽职调查清单 (Due Diligence Checklist)

在最终确认合作前，必须完成以下检查：

### 3.1 数据验证
- [ ] **受众画像分析**：索取创作者的后台数据截图，确认受众的年龄、性别、地理位置分布是否与品牌目标市场一致。
- [ ] **假粉检测**：使用第三方工具（如 HypeAuditor, Modash, Social Blade）或手动抽查粉丝列表，识别异常的粉丝增长曲线或大量无头像/无内容的僵尸粉。
- [ ] **参与度验证**：手动计算最近 10 篇帖子的平均参与度，与平台基准进行对比。

### 3.2 品牌安全
- [ ] **历史言论审查**：搜索创作者的历史言论（包括已删除的内容，如有可能），确保没有涉及种族歧视、性别歧视、极端政治倾向等可能引发公关危机的言论。
- [ ] **竞品排他性**：确认创作者在近期（如过去 3 个月）内没有推广过直接竞争对手的产品。
- [ ] **内容调性一致性**：确认创作者的整体内容风格不会与品牌形象产生冲突。

### 3.3 商业能力
- [ ] **历史合作审查**：查看其过去 6 个月内的赞助内容，评估合作品牌的质量和内容的表现。
- [ ] **沟通专业度**：在初步沟通中评估创作者（或其经纪人）的响应速度、专业程度和合作态度。
- [ ] **内容授权意愿**：确认创作者是否愿意授予品牌内容使用权（用于广告投放、官网展示等）。

## 4. 创作者层级分类与策略匹配

不同层级的创作者适用于不同的营销目标和合作模式。

| 层级 | 粉丝量 | 典型参与度 | 最佳用途 | 推荐合作模式 |
| :--- | :--- | :--- | :--- | :--- |
| **纳米网红** | 1K - 10K | 4% - 8% | 真实口碑、高转化率 | 产品置换、联盟佣金 |
| **微型网红** | 10K - 100K | 2% - 5% | 精准触达、高性价比 | 混合薪酬、联盟佣金 |
| **中型网红** | 100K - 500K | 1.5% - 3% | 品牌曝光 + 转化 | 固定费用 + 佣金 |
| **大型网红** | 500K - 1M | 1% - 2% | 大规模品牌曝光 | 固定费用 |
| **头部网红** | > 1M | 0.5% - 1.5% | 品牌事件、联名共创 | 固定费用 + 联名分成 |

**DTC 品牌的最佳实践**：将 70% 的预算分配给微型和纳米网红（高转化、高 ROI），20% 分配给中型网红（平衡曝光与转化），10% 分配给大型/头部网红（品牌事件和社会证明）。
