# 核心框架与范式

> 本文件为当前生效的模块参考资料，供执行时按需加载；用于扩展达人合作中的核心框架、合作范式与增长原则。若与当前模块 `SKILL.md` 或 `_system/` 全局协议冲突，以系统级规则为准。

## 一、2026 创作者经济新范式：从交易到共创

> **核心认知转变**：从"一次性流量采买"到"长期关系构建与品牌共创"。

### 1.1 范式转变全景

```
旧范式（2020-2023）                    新范式（2025-2026）
───────────────────                    ───────────────────
一次性发帖交易 (One-off Posts)   →     长期品牌大使与共创 (Long-term Partnerships)
唯粉丝数论 (Follower Count)      →     唯参与度与转化论 (Engagement & Conversion)
品牌方强控制内容                 →     创作者主导的真实表达 (Creator-led Authenticity)
单一的固定费用支付               →     基于绩效的混合薪酬 (Performance-based Hybrid)
网红仅作为分发渠道               →     网红作为内容工作室与顾问 (Creators as Consultants)
手动表格管理合作                 →     AI 驱动的发现与自动化工作流 (AI-powered Workflows)
联盟营销作为边缘渠道             →     联盟营销作为核心收入驱动力 (Core Revenue Driver)
```

### 1.2 四大核心支柱

1.  **真实性优先 (Authenticity First)**：消费者对传统的、高度抛光的广告越来越免疫。创作者的价值在于他们与受众之间建立的真实信任。品牌必须允许创作者用他们自己的声音和风格来讲述产品故事，而不是强加生硬的广告脚本。
2.  **绩效驱动 (Performance-Driven)**：随着营销预算的收紧，网红营销必须证明其 ROI。这推动了从单纯的品牌曝光向基于绩效的模式转变，如联盟佣金、按转化付费或混合薪酬结构。
3.  **微型与纳米网红的崛起 (Rise of Micro/Nano Influencers)**：虽然头部网红能带来巨大曝光，但拥有 1k-100k 粉丝的微型和纳米网红通常拥有更高的参与度、更精准的受众群体和更高的转化率，且合作成本更低。
4.  **全渠道整合 (Omnichannel Integration)**：网红产生的内容（UGC）不应仅停留在他们的个人主页上。品牌需要获取内容授权，将其整合到自己的付费广告（如 TikTok Spark Ads）、邮件营销、落地页和社交媒体矩阵中，实现内容价值的最大化。

## 二、Seeding 财务可行性前置检查

> **核心原则**：在推荐产品置换 / Seeding 策略之前，**必须先验证财务可行性**。产品成本直接决定 Seeding 的规模上限和合作模式选择。跳过这一步就给出"50-100 人 Seeding"的建议，对高客单价品牌是不负责任的。

**前置检查流程**：
1. 确认产品成本（COGS + 运费，不含零售价）
2. 查下表确定 Seeding 策略上限
3. 基于上限调整金字塔配置

| 产品成本（COGS + 运费） | Seeding 规模上限 | 推荐合作模式 | 金字塔配置调整 |
| :--- | :--- | :--- | :--- |
| **< $30** | 50-100 人（标准规模） | 产品置换为主，不强制发帖 | 标准金字塔：50-100 纳米 + 10-20 微型 + 3-5 中型 + 1 头部 |
| **$30 - $100** | 10-30 人（中等规模） | 产品置换 + 佣金制混合，优先选择高匹配度创作者 | 精选金字塔：10-30 纳米 + 5-10 微型 + 2-3 中型 |
| **$100 - $500** | 3-10 人（小规模试点） | 佣金制 / 联盟营销为主，Seeding 仅限高确定性合作 | 倒三角：佣金制为主体 + 3-10 精选 Seeding + 1-2 中型品牌大使 |
| **> $500** | 不推荐大规模 Seeding | 佣金制、联盟营销、内容授权费、品牌大使月度津贴 | 零 Seeding 模型：纯佣金联盟网络 + 1-3 深度品牌大使（付费） |

**绝对不要做的事**：
- 不要在产品成本 > $100 时推荐 50-100 人规模的 Seeding——这意味着 $5,000-$50,000 的纯赠送成本，对绝大多数中小 DTC 品牌不可行。
- 不要忽略运费成本——国际运费可能使 Seeding 成本翻倍。
- 不要把零售价当成本——Seeding 的财务计算必须基于 COGS + 运费，不是零售价。

**高客单价品牌的替代策略**：
当产品成本过高不适合大规模 Seeding 时，推荐以下零库存成本的合作模式：
1. **纯佣金联盟计划**：15-30% 佣金比例（高客单价产品的佣金比例可以更高，因为绝对金额足够吸引创作者）
2. **内容授权费模式**：向创作者支付 $200-$500 的内容制作费，获取内容使用权，而非赠送产品
3. **体验式合作**：借用产品进行评测（评测后归还），适用于高价电子产品、家具等
4. **品牌大使月度津贴**：$500-$2,000/月的固定合作费 + 佣金，换取持续的内容产出

## 三、平台选择矩阵

不同平台的用户行为和转化路径差异巨大，必须根据产品特性选择最佳平台：

| 平台 | 最佳内容类型 | 转化特点 | 适合品类 | 白名单投流方式 |
| :--- | :--- | :--- | :--- | :--- |
| **TikTok** | 短视频 (15-60s) | 冲动消费、病毒式传播 | 美妆、服饰、食品、小工具 | Spark Ads |
| **Instagram** | Reels + 图文 + Stories | 视觉驱动、生活方式影响 | 时尚、家居、美妆、健康 | Branded Content Ads |
| **YouTube** | 深度评测 + 教程 | 高客单价、理性决策 | 科技、SaaS、高端护肤 | 无原生白名单 |
| **Pinterest** | 灵感图片 + 教程 | 长尾搜索流量、高购买意图 | 家居、婚礼、DIY、食品 | Idea Ads |

## 四、自动化发现工作流建议

*   **社交聆听**：监控品牌提及、行业关键词和竞品标签，发现已经对品牌或相关话题感兴趣的创作者。
*   **客户数据库挖掘**：分析现有的高 LTV 客户或忠诚度计划成员，找出其中具有社交影响力的个体，将他们转化为品牌倡导者。
*   **AI 工具辅助**：利用专业的网红营销平台（如 Aspire.io, Grin, CreatorIQ）或 AI 搜索工具，基于受众数据和内容标签进行精准匹配。
*   **竞品创作者分析**：研究竞品正在合作的创作者，识别其中可能对你的品牌也感兴趣的人选。

## 五、内容白名单与二次利用体系

创作者产生的内容不应仅停留在他们的个人主页上。品牌必须建立系统化的内容二次利用体系：

*   **白名单投流 (Whitelisting)**：获取创作者授权，通过品牌的广告账户将创作者的内容作为广告素材进行付费推广。Meta 平台使用 Branded Content Ads，TikTok 使用 Spark Ads。白名单广告常常能跑出比品牌自有素材更好的效率，但真实 uplift 幅度高度依赖创作者贴合度、素材质量、账户结构与归因口径，宜先按小样本验证，再决定是否放大。
*   **官方渠道转发**：将优质 UGC 转发到品牌官方社媒账号，作为社会证明。
*   **邮件营销嵌入**：将创作者的真实评价和使用场景图嵌入到欢迎序列、弃单挽回和促销邮件中。
*   **落地页优化**：将创作者的视频评测和使用展示嵌入到产品页和专属落地页中，提升信任度和转化率。
