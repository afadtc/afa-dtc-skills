# 品牌定位理论深度参考

> 本文件为 afa-brand 的定位构建模式提供理论支撑和 DTC 案例库。

---

## 一、经典定位理论在 DTC 中的应用

### 1.1 Ries & Trout 定位理论

**核心观点**：定位不是你对产品做的事，而是你对潜在客户心智做的事。

DTC 应用要点：
- 心智阶梯：每个品类在消费者心中只有 2-3 个位置，DTC 品牌必须找到未被占据的阶梯
- 聚焦法则：宁可在一个狭窄领域做到第一，也不要在宽泛领域做到第三
- 对立法则：如果不能做第一，就做第一的对立面
- 品类法则：如果不能在现有品类做第一，就创造一个新品类

### 1.2 April Dunford — Obviously Awesome

**核心框架**：5 步定位流程

| 步骤 | 问题 | DTC 示例 |
|:---|:---|:---|
| 竞争替代方案 | 如果你不存在，客户会怎么做？ | 去超市买普通咖啡 |
| 独特属性 | 你有什么他们没有的？ | 单一产地、72 小时内烘焙 |
| 属性价值 | 这些属性对客户意味着什么？ | 更新鲜、更独特的口味体验 |
| 目标客户 | 谁最在乎这些价值？ | 咖啡爱好者，愿意为品质付费 |
| 市场品类 | 你在什么市场中竞争？ | 精品咖啡订阅 |

### 1.3 Eugene Schwartz — 市场意识阶段

| 阶段 | 市场状态 | 最佳定位策略 | DTC 示例 |
|:---|:---|:---|:---|
| 1 - 全新 | 没人见过这个 | 简单宣告产品 | 第一个 DTC 床垫品牌 |
| 2 - 成长 | 竞争出现 | 宣称优越性 | "最舒适的床垫" |
| 3 - 拥挤 | 类似宣称泛滥 | 解释独特机制 | "3 层记忆棉技术" |
| 4 - 疲劳 | 市场见过一切 | 身份认同 | "为侧睡者设计" |
| 5 - 成熟 | 品牌忠诚决定 | 独家归属感 | "加入 10 万好眠社区" |

### 1.4 Category Design（品类设计）

**核心观点**：不要在现有品类中竞争，创造一个你能定义的新品类。

品类设计三步法：
1. **定义问题**：用新的框架重新定义客户面临的问题
2. **命名品类**：为你的解决方案创造一个新品类名称
3. **教育市场**：通过内容让市场接受这个新品类

DTC 品类设计案例：
- Glossier → "Skin First, Makeup Second"（护肤优先美妆）
- Dollar Shave Club → "Razor Subscription"（剃须刀订阅）
- Warby Parker → "Designer Eyewear at a Fraction"（平价设计师眼镜）
- Allbirds → "Sustainable Comfort Shoes"（可持续舒适鞋）

## 二、DTC 定位案例库

### 案例 1：Glossier — 创造者原型

| 维度 | 内容 |
|:---|:---|
| 定位 | 护肤优先、美妆第二的美妆品牌 |
| 原型 | 创造者 + 普通人 |
| 核心信念 | 美妆应该赋能而非掩盖 |
| 独特机制 | 社区驱动的产品开发 |
| 声音 | 亲密、包容、赋能、真实 |
| 视觉 | 粉色、极简、真实肤质、自然光 |
| 成功因素 | 社区 → 内容 → 产品的反向模式 |

### 案例 2：Allbirds — 探索者原型

| 维度 | 内容 |
|:---|:---|
| 定位 | 用自然材料做的舒适鞋 |
| 原型 | 探索者 + 照顾者 |
| 核心信念 | 好产品不需要伤害地球 |
| 独特机制 | 新西兰美利奴羊毛 + 碳足迹标签 |
| 声音 | 好奇、自然、简洁、有教育性 |
| 视觉 | 自然色调、大量留白、产品特写 |
| 成功因素 | 可持续 + 舒适的双重定位 |

### 案例 3：Dollar Shave Club — 反叛者原型

| 维度 | 内容 |
|:---|:---|
| 定位 | 打破剃须刀行业暴利的订阅服务 |
| 原型 | 反叛者 + 小丑 |
| 核心信念 | 剃须不应该这么贵 |
| 独特机制 | $1/月订阅 + 直邮到家 |
| 声音 | 大胆、幽默、挑衅、直接 |
| 视觉 | 大胆配色、创始人出镜、幽默元素 |
| 成功因素 | 病毒视频 + 价格颠覆 + 幽默品牌 |

### 案例 4：The Ordinary — 智者原型

| 维度 | 内容 |
|:---|:---|
| 定位 | 透明配方、科学护肤 |
| 原型 | 智者 + 反叛者 |
| 核心信念 | 护肤行业过度营销，成分才是真相 |
| 独特机制 | 单一成分产品 + 透明定价 |
| 声音 | 理性、教育性、透明、反营销 |
| 视觉 | 极简、医药感、白色 + 黑色文字 |
| 成功因素 | 反行业共识 + 极致透明 + 教育内容 |

## 三、定位验证工具

### 3.1 "酒吧测试"

想象你在酒吧遇到一个朋友，他问你"你的公司是做什么的？"

- 如果你的回答超过 10 秒 → 定位不够清晰
- 如果对方需要追问 → 定位不够直观
- 如果对方说"哦，就像 [竞品] 一样？" → 定位不够差异化
- 如果对方说"哦，有意思，怎么做到的？" → 定位成功

### 3.2 "反向测试"

把你的定位声明中的品牌名换成竞品的名字：
- 如果依然成立 → 定位不够差异化
- 如果明显不成立 → 定位是独特的

### 3.3 "10 岁小孩测试"

把你的定位声明读给一个 10 岁的孩子：
- 如果他能理解你做什么 → 定位足够清晰
- 如果他困惑 → 定位太复杂或太抽象

### 3.4 "所以呢？测试"

读完你的定位声明后，问自己"所以呢？"
- 如果答案是"所以你的生活会变得更好" → 定位有价值
- 如果答案是"所以...嗯..." → 定位缺乏价值感

## 五、定位策略发现流程

```
Step 1：识别真正的转变（Transformation）
├── 不是产品功能，而是客户生活的改变
├── 提问：客户使用前 vs 使用后，最大的变化是什么？
├── 提问：如果用一句话描述这个转变，怎么说？
├── 框架：从 [痛苦的现状] 到 [理想的状态]
└── 示例：
    ├── 功能描述：有机棉 T 恤 → 转变描述：穿着不伤害地球的衣服，依然好看
    ├── 功能描述：智能睡眠追踪器 → 转变描述：终于知道为什么总是累，并修复它
    └── 功能描述：订阅制咖啡 → 转变描述：每天早上的第一杯，不再将就

Step 2：映射竞争替代方案
├── 客户如果不买你的产品，会怎么做？
│   ├── 什么都不做（忍受问题）
│   ├── DIY（自己拼凑解决方案）
│   ├── 雇人（找顾问/服务商）
│   ├── 买不同品类（完全不同的解决方案）
│   └── 买直接竞品
├── 每个替代方案的弱点是什么？
└── 这些弱点就是你的定位机会

Step 3：发现独特机制（Unique Mechanism）
├── 你的产品如何以不同的方式解决问题？
├── 有没有可以命名的独特流程/方法/技术？
├── 即使没有真正独特的东西，也要找到可以命名的差异
├── 提问：
│   ├── 你的产品有什么专有的流程或方法？
│   ├── 你做了什么竞品没做的事？
│   ├── 什么反直觉的洞察让你的方法有效？
│   └── 你的"秘密配方"是什么？
└── 示例：
    ├── "3-Phase Sleep Reset Protocol"（睡眠品牌）
    ├── "Farm-to-Face 72-Hour Freshness"（护肤品牌）
    └── "AI-Matched Roast Profile"（咖啡品牌）

Step 4：评估市场成熟度（Schwartz 意识阶段）
├── Stage 1（新品类）：市场没见过这个 → 角度：简单宣告
│   └── "现在你可以 [做某事]"
├── Stage 2（成长期）：竞争出现，市场升温 → 角度：宣称优越性
│   └── "最快/最简单/最完整的 [解决方案]"
├── Stage 3（拥挤期）：多个玩家，类似宣称 → 角度：解释机制
│   └── "这就是为什么 [机制] 在其他方法失败时依然有效"
├── Stage 4（疲劳期）：市场见过一切，需要新框架 → 角度：身份认同
│   └── "为 [身份标签] 的人打造"
└── Stage 5（成熟期）：品牌忠诚度决定一切 → 角度：独家归属
    └── "加入 [部落/运动]"

Step 5：运行角度生成器
├── 逆向角度（Contrarian）：挑战行业共识
│   └── "关于 [话题] 你被告知的一切都是错的"
├── 机制角度（Mechanism）：用 HOW 而非 WHAT 领先
│   └── "[命名方法] 如何让你 [结果] 而不需要 [常见牺牲]"
├── 转变角度（Transformation）：前后对比
│   └── "从 [痛苦现状] 到 [理想结果]"
├── 敌人角度（Enemy）：共同的敌人
│   └── "别让 [敌人] 偷走你的 [有价值的东西]"
├── 速度/简单角度（Speed/Ease）：压缩时间或减少努力
│   └── "[结果] 在 [惊人短的时间] 内"
├── 精准角度（Specificity）：超级具体的人群或结果
│   └── "为 [非常具体的人群] 打造的 [非常具体的结果]"
├── 社会证明角度（Social Proof）：用证据而非宣称领先
│   └── "[具体结果] 已服务 [数量] [人群]"
└── 风险逆转角度（Risk Reversal）：把保证变成标题
    └── "[结果] 或者 [对卖方的戏剧性后果]"
```

## 六、定位声明模板

```
经典定位声明（内部使用）：
对于 [目标受众]，
[品牌名] 是 [品类] 中的 [差异化定位]，
因为 [独特机制/证据]。

一句话定位（外部使用）：
[品牌名] 帮助 [目标受众] [实现转变]，通过 [独特机制]。

品牌 Tagline（广告语）：
├── 类型 1：承诺型 — "[核心利益]，[修饰语]"
├── 类型 2：信念型 — "[品牌信念的简洁表达]"
├── 类型 3：行动型 — "[动词] + [对象]"
└── 类型 4：身份型 — "为 [身份] 而 [动词]"

定位验证清单：
├── ☐ 是否清晰？（10 岁小孩能理解你做什么）
├── ☐ 是否差异化？（与前 3 名竞品明显不同）
├── ☐ 是否可信？（有证据支撑，不是空话）
├── ☐ 是否有吸引力？（目标受众听了会想了解更多）
├── ☐ 是否可持续？（不是短期噱头，能长期坚持）
├── ☐ 是否可执行？（能指导所有营销决策）
└── ☐ 是否可测量？（能通过数据验证效果）
```

DTC 品牌常见原型组合：

| 原型 | 核心驱动 | DTC 品牌示例 | 声音特征 | 适合品类 |
|:---|:---|:---|:---|:---|
| 反叛者 Outlaw | 打破规则，挑战现状 | Dollar Shave Club | 大胆、挑衅、幽默 | 颠覆传统行业的品牌 |

| 创造者 Creator | 创新、自我表达 | Glossier | 赋能、创意、包容 | 美妆、时尚、设计 |
| 照顾者 Caregiver | 保护、滋养、服务 | Honest Company | 温暖、可信、关怀 | 母婴、健康、家居 |
| 英雄 Hero | 勇气、成就、超越 | Peloton | 激励、挑战、力量 | 健身、教育、工具 |
| 魔法师 Magician | 转化、愿景实现 | Oura Ring | 神秘、智能、变革 | 科技、健康科技 |
| 智者 Sage | 知识、真理、理解 | The Ordinary | 透明、教育、理性 | 护肤、金融、教育 |
| 天真者 Innocent | 简单、纯净、乐观 | Brandless | 简洁、纯粹、快乐 | 食品、日用品、儿童 |

原型选择流程：
├── 1. 品牌的核心驱动力是什么？（改变世界 vs 服务他人 vs 追求卓越）
├── 2. 品牌希望客户感受到什么？（兴奋 vs 安全 vs 归属 vs 自由）
├── 3. 品牌的"敌人"是什么？（无聊 vs 复杂 vs 不公 vs 平庸）
├── 4. 选择 1 个主原型 + 1 个辅原型（避免超过 2 个）
└── 5. 验证：这个原型组合是否与竞品不同？是否与目标受众共鸣？
