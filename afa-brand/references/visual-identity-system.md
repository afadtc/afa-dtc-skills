# 视觉识别系统完整规范

> 本文件提供视觉识别构建模式所需的色彩心理学、字体策略、图像风格和品牌应用规范。
>
> **职责边界**：品牌策略侧负责**定义**视觉识别系统（色彩心理学、字体策略、图像风格规范），创意制作侧负责**应用**这些规范制作具体的广告素材和营销物料（可参见相关品牌素材模板）。当用户需要调整品牌视觉规范时，应进入品牌规范梳理流程；当用户需要基于现有规范制作素材时，应交由后续创意制作流程承接。

## 一、色彩心理学深度参考

### 1.1 色彩与品牌原型的对应关系

| 品牌原型 | 推荐主色 | 推荐辅色 | 心理依据 |
|:---|:---|:---|:---|
| 反叛者 Outlaw | 黑色、红色 | 橙色、黄色 | 力量、打破规则、注意力 |
| 探索者 Explorer | 绿色、棕色 | 蓝色、米色 | 自然、冒险、自由 |
| 创造者 Creator | 紫色、橙色 | 粉色、青色 | 创意、想象力、独特 |
| 照顾者 Caregiver | 蓝色、绿色 | 白色、粉色 | 信任、健康、安全 |
| 英雄 Hero | 红色、深蓝 | 金色、黑色 | 力量、勇气、成就 |
| 魔法师 Magician | 紫色、深蓝 | 金色、银色 | 神秘、转化、智慧 |
| 智者 Sage | 蓝色、灰色 | 白色、绿色 | 知识、真理、冷静 |
| 天真者 Innocent | 白色、浅蓝 | 黄色、粉色 | 纯净、简单、乐观 |
| 普通人 Everyman | 蓝色、绿色 | 棕色、灰色 | 可靠、归属、真实 |
| 情人 Lover | 红色、粉色 | 金色、紫色 | 热情、亲密、奢华 |
| 小丑 Jester | 黄色、橙色 | 红色、绿色 | 快乐、活力、乐趣 |
| 统治者 Ruler | 深蓝、金色 | 黑色、紫色 | 权威、控制、奢华 |

### 1.2 色彩组合策略

**互补色方案（高对比）**：
- 适合：需要强烈视觉冲击的品牌
- 示例：蓝色 + 橙色、紫色 + 黄色
- 注意：不要等比例使用，主色 70% + 互补色 30%

**类似色方案（和谐）**：
- 适合：追求优雅、和谐感的品牌
- 示例：蓝色 + 蓝绿 + 绿色
- 注意：需要强调色来打破单调

**三色方案（平衡）**：
- 适合：需要丰富但不混乱的品牌
- 示例：红 + 黄 + 蓝（色轮等距三角）
- 注意：一个主导，两个辅助

**单色方案（极简）**：
- 适合：追求极简、高端感的品牌
- 示例：不同深浅的蓝色
- 注意：需要强调色（通常是对比色）来引导注意力

### 1.3 DTC 品牌色彩趋势（2025-2026）

| 趋势 | 描述 | 适合品类 |
|:---|:---|:---|
| 柔和自然色 | 大地色、米色、灰绿 | 可持续品牌、有机食品、家居 |
| 大胆撞色 | 高饱和度对比色 | 年轻品牌、潮牌、科技 |
| 极简黑白 | 黑白为主 + 一个强调色 | 奢侈品、极简品牌、时尚 |
| 渐变色 | 品牌色的渐变应用 | 科技、健康科技、SaaS |
| 复古色调 | 低饱和度、怀旧感 | 手工品牌、食品、生活方式 |

## 二、字体策略深度参考

### 2.1 字体与品牌个性的对应

| 字体类型 | 传达的个性 | DTC 常用字体 | 适合品牌 |
|:---|:---|:---|:---|
| 几何无衬线 | 现代、简洁、科技 | Inter, Poppins, Montserrat | 科技、DTC、年轻品牌 |
| 人文无衬线 | 友好、温暖、亲民 | Open Sans, Lato, Source Sans | 健康、教育、社区品牌 |
| 新怪诞无衬线 | 中性、专业、万能 | Helvetica, Arial, DM Sans | 企业、金融、多品类 |
| 古典衬线 | 传统、权威、优雅 | Times, Georgia, Garamond | 奢侈品、出版、法律 |
| 现代衬线 | 时尚、精致、高端 | Playfair Display, Didot | 时尚、美妆、高端品牌 |
| 板衬线 | 大胆、有力、复古 | Rockwell, Clarendon | 户外、运动、复古品牌 |
| 手写体 | 个人、温暖、创意 | 各种手写字体 | 手工、食品、个人品牌 |
| 等宽字体 | 技术、极客、独立 | JetBrains Mono, Fira Code | 开发者工具、独立品牌 |

### 2.2 字体配对原则

**对比配对**：标题和正文字体形成对比
- 衬线标题 + 无衬线正文（经典组合）
- 粗体展示字体 + 轻量正文字体
- 示例：Playfair Display + Source Sans Pro

**和谐配对**：同一字体家族的不同字重
- 标题用 Bold/Black + 正文用 Regular/Light
- 示例：Inter Bold + Inter Regular

**超级家族配对**：同一设计师的衬线 + 无衬线
- 示例：Merriweather + Merriweather Sans
- 示例：Roboto Slab + Roboto

### 2.3 Web 字体性能优化

| 策略 | 方法 | 影响 |
|:---|:---|:---|
| 字体数量 | 最多 2 个字体家族 | 减少 HTTP 请求 |
| 字重数量 | 每个字体最多 3 个字重 | 减少文件大小 |
| 字体格式 | 优先 WOFF2 | 最小文件大小 |
| 加载策略 | font-display: swap | 避免 FOIT |
| 子集化 | 只加载需要的字符集 | 减少 50-80% 大小 |
| 预加载 | 关键字体使用 preload | 加速首次渲染 |

## 五、品牌情绪板（Moodboard）创建指南

### 5.1 情绪板要素

| 要素 | 数量 | 来源 |
|:---|:---|:---|
| 参考图片 | 5-8 张 | Pinterest, Dribbble, 竞品网站 |
| 色彩样本 | 5-7 个 | 从参考图片中提取 |
| 字体样本 | 2-3 个 | Google Fonts, Adobe Fonts |
| 材质/纹理 | 2-3 个 | 与品牌感受相关的材质 |
| 关键词 | 3-5 个 | 描述整体视觉感受 |
| 竞品参考 | 2-3 个 | 标注"要的"和"不要的" |

### 5.2 情绪板创建流程

1. 收集 20-30 张与品牌感受相关的图片
2. 筛选到 8-10 张最能代表品牌的图片
3. 从图片中提取主要色彩
4. 选择与色彩和感受匹配的字体
5. 添加材质/纹理参考
6. 用 3-5 个关键词总结整体感受
7. 与用户确认方向

## 六、色彩系统构建规则

```
色彩系统构建规则：
├── 主色（Primary）：1 个，品牌最核心的识别色
│   ├── 占比：60-70% 的品牌视觉面积
│   └── 选择依据：品牌原型 + 品类惯例 + 差异化需求
│
├── 辅色（Secondary）：1-2 个，支撑和补充主色
│   ├── 占比：20-30%
│   └── 选择依据：与主色的和谐关系 + 功能需求
│
├── 强调色（Accent）：1 个，用于 CTA 和重要元素
│   ├── 占比：5-10%
│   └── 选择依据：与主色的对比度 + 注意力吸引力
│
├── 中性色（Neutral）：2-3 个，用于背景和正文
│   ├── 包括：深色（文字）、浅色（背景）、中间色（边框/分隔）
│   └── 选择依据：可读性 + 与品牌色的协调
│
└── 功能色（Functional）：成功/警告/错误/信息
    ├── 绿色：成功、确认
    ├── 黄色：警告、注意
    ├── 红色：错误、危险
    └── 蓝色：信息、链接

色彩规范输出格式：
| 色彩角色 | 名称 | HEX | RGB | 使用场景 |
|:---|:---|:---|:---|:---|
| 主色 | {name} | #{hex} | rgb(r,g,b) | {场景} |
| 辅色 | {name} | #{hex} | rgb(r,g,b) | {场景} |
| 强调色 | {name} | #{hex} | rgb(r,g,b) | {场景} |
| 中性-深 | {name} | #{hex} | rgb(r,g,b) | {场景} |
| 中性-浅 | {name} | #{hex} | rgb(r,g,b) | {场景} |
```

## 七、字体层级规范

```
字体层级规范输出格式：
| 层级 | 用途 | 字体 | 大小 | 字重 | 行高 |
|:---|:---|:---|:---|:---|:---|
| H1 | 页面主标题 | {font} | 36-48px | Bold/700 | 1.2 |
| H2 | 章节标题 | {font} | 28-36px | SemiBold/600 | 1.3 |
| H3 | 子标题 | {font} | 22-28px | Medium/500 | 1.4 |
| Body | 正文 | {font} | 16-18px | Regular/400 | 1.6 |
| Small | 辅助文字 | {font} | 14px | Regular/400 | 1.5 |
| Caption | 标注 | {font} | 12px | Regular/400 | 1.4 |

标题字体选择维度：
├── 衬线体（Serif）→ 传统、权威、优雅（适合奢侈品、教育、金融）
├── 无衬线体（Sans-serif）→ 现代、简洁、科技感（适合科技、DTC、年轻品牌）
├── 手写体（Script）→ 个人、温暖、创意（适合手工、美妆、食品）
└── 展示体（Display）→ 独特、大胆、有个性（适合潮牌、娱乐、创意）

DTC 常用标题字体：Inter, Poppins, Playfair Display, Montserrat, DM Sans
DTC 常用正文字体：Inter, Source Sans Pro, Lato, Open Sans, Noto Sans
```
