# 品牌原型深度参考

> 本文件为 afa-brand 的原型选择提供 12 原型的完整画像、DTC 应用指南和混合原型策略。

---

## 一、12 品牌原型完整画像

### 1. 天真者 The Innocent

| 维度 | 内容 |
|:---|:---|
| 核心欲望 | 体验天堂般的幸福 |
| 核心恐惧 | 做错事、被惩罚 |
| 品牌承诺 | 生活可以是简单、美好的 |
| 策略 | 做正确的事 |
| 声音特征 | 乐观、简单、真诚、温暖 |
| 视觉特征 | 明亮、干净、柔和色调、大量留白 |
| DTC 代表 | Honest Company, Burt's Bees |
| 适合品类 | 有机食品、天然护肤、婴儿用品、清洁产品 |
| 定价策略 | 中等偏上（纯净有溢价） |
| 内容策略 | 教育性内容、成分透明、简单生活方式 |

### 2. 探索者 The Explorer

| 维度 | 内容 |
|:---|:---|
| 核心欲望 | 发现自我的自由 |
| 核心恐惧 | 被困住、空虚、从众 |
| 品牌承诺 | 帮你发现新世界 |
| 策略 | 踏上旅程 |
| 声音特征 | 冒险、独立、好奇、不羁 |
| 视觉特征 | 自然色调、户外场景、广阔空间、运动感 |
| DTC 代表 | Allbirds, REI, Patagonia |
| 适合品类 | 户外装备、旅行用品、运动品牌、可持续品牌 |
| 定价策略 | 中高端（冒险需要投资） |
| 内容策略 | 探索故事、用户冒险、自然与可持续 |

### 3. 智者 The Sage

| 维度 | 内容 |
|:---|:---|
| 核心欲望 | 发现真理 |
| 核心恐惧 | 被欺骗、无知 |
| 品牌承诺 | 帮你理解世界 |
| 策略 | 寻求信息和知识 |
| 声音特征 | 理性、教育性、权威、冷静 |
| 视觉特征 | 冷色调、极简、数据可视化、专业感 |
| DTC 代表 | The Ordinary, Aesop |
| 适合品类 | 科技产品、教育平台、专业护肤、金融服务 |
| 定价策略 | 两极化（极致性价比或极致专业） |
| 内容策略 | 深度教育、成分解析、科学研究、行业真相 |

### 4. 英雄 The Hero

| 维度 | 内容 |
|:---|:---|
| 核心欲望 | 通过勇气证明自己的价值 |
| 核心恐惧 | 软弱、脆弱、退缩 |
| 品牌承诺 | 帮你变得更强 |
| 策略 | 变得尽可能强大和有能力 |
| 声音特征 | 激励、自信、有力、直接 |
| 视觉特征 | 强烈对比、动态构图、力量感、深色调 |
| DTC 代表 | Nike, Under Armour, Peloton |
| 适合品类 | 运动健身、个人发展、高性能产品 |
| 定价策略 | 中高端（英雄不打折） |
| 内容策略 | 挑战故事、成就展示、激励内容、用户转变 |

### 5. 反叛者 The Outlaw

| 维度 | 内容 |
|:---|:---|
| 核心欲望 | 革命或复仇 |
| 核心恐惧 | 无力、被忽视 |
| 品牌承诺 | 打破规则 |
| 策略 | 颠覆、破坏、震撼 |
| 声音特征 | 大胆、挑衅、不敬、直接 |
| 视觉特征 | 黑色为主、高对比、粗犷、非传统 |
| DTC 代表 | Liquid Death, Dollar Shave Club |
| 适合品类 | 颠覆性品牌、年轻品牌、另类生活方式 |
| 定价策略 | 颠覆性定价（打破行业惯例） |
| 内容策略 | 挑战行业、幽默讽刺、用户反叛故事 |

### 6. 魔法师 The Magician

| 核心恐惧 | 意外的负面后果 |
| 品牌承诺 | 让梦想成真 |
| 策略 | 发展愿景并实现它 |
| 声音特征 | 神秘、启发、变革、有远见 |

| 定价策略 | 高端（魔法有价） |
| 内容策略 | 产品演示、变革故事、技术揭秘、愿景分享 |

### 7. 普通人 The Everyman

| 维度 | 内容 |
|:---|:---|

## 二、混合原型策略

### 2.1 为什么需要混合原型

单一原型可能过于扁平。最成功的 DTC 品牌通常有一个主原型（70%）和一个辅原型（30%）。

### 2.2 常见的高效混合

| 主原型 | 辅原型 | 效果 | DTC 示例 |
|:---|:---|:---|:---|
| 反叛者 | 小丑 | 颠覆 + 幽默 = 病毒传播 | Dollar Shave Club |
| 智者 | 反叛者 | 知识 + 挑战 = 行业颠覆 | The Ordinary |
| 探索者 | 照顾者 | 冒险 + 关怀 = 可持续品牌 | Allbirds |
| 创造者 | 普通人 | 创意 + 包容 = 社区品牌 | Glossier |
| 英雄 | 魔法师 | 力量 + 变革 = 科技品牌 | Peloton |
| 照顾者 | 智者 | 关怀 + 知识 = 健康品牌 | Hims & Hers |
| 情人 | 创造者 | 感性 + 创意 = 奢华体验 | Jo Malone |
| 天真者 | 探索者 | 纯净 + 发现 = 有机品牌 | Honest Company |

### 2.3 应避免的混合

| 混合 | 原因 |
|:---|:---|
| 反叛者 + 天真者 | 价值观冲突：打破规则 vs 做正确的事 |
| 统治者 + 小丑 | 形象冲突：权威 vs 不正经 |
| 智者 + 小丑 | 可信度冲突：严肃知识 vs 娱乐 |
| 照顾者 + 反叛者 | 态度冲突：温柔关怀 vs 激进挑衅 |

## 三、原型选择决策树

```
你的品牌想让客户感受到什么？

├── 力量和自信
│   ├── 通过挑战自我 → 英雄
│   ├── 通过打破规则 → 反叛者
│   └── 通过掌控一切 → 统治者
│
├── 知识和理解
│   ├── 追求真理和透明 → 智者
│   ├── 追求变革和创新 → 魔法师
│   └── 追求自我表达 → 创造者
│
├── 连接和归属
│   ├── 被照顾和保护 → 照顾者
│   ├── 被接纳和理解 → 普通人
│   └── 被爱和渴望 → 情人
│
└── 自由和快乐
    ├── 探索和发现 → 探索者
    ├── 纯真和简单 → 天真者
    └── 乐趣和欢笑 → 小丑
```
