# PR 诊断体系

> **核心理念**：先诊断，再开处方。不同的公关与声誉问题需要截然不同的策略。

## 1. 诊断模式 1：零媒体曝光（Zero Media Presence）

```
触发信号：
├── 用户说"我们从来没被媒体报道过"
├── 品牌搜索结果只有官网和社交账号，没有第三方背书
├── 试图联系记者但从未收到回复

诊断决策树：
├── 是否有清晰的品牌定位和故事？
│   ├── 没有 → 转交 afa-brand 完善基础
│   └── 有 → 继续
├── 是否有准备好的媒体套件（Media Kit）？
│   ├── 没有 → 生成标准化 Media Kit
│   └── 有 → 继续
├── Pitch 邮件是否使用了 CPR 框架？
│   ├── 没有 → 教授 CPR 框架，重写 Pitch
│   └── 有 → 继续
└── 目标媒体列表是否精准？
    ├── 盲目群发大媒体 → 缩小范围，定位垂直领域的具体记者
    └── 列表精准 → 检查跟进（Follow-up）策略

处方：
├── 轻度：优化 Pitch 邮件结构（应用 CPR 框架）
├── 中度：构建精准的垂直媒体列表 + 制定季节性叙事角度
└── 重度：从零构建媒体套件 + 制定 90 天冷推计划
```

## 2. 诊断模式 2：UGC 匮乏与社区冷清

```
触发信号：
├── 用户说"客户买完就不理我们了"
├── 社交媒体上没有真实的买家秀
├── 网站缺乏带图评价，转化率低

诊断决策树：
├── 是否有主动收集 UGC 的机制？
│   ├── 没有 → 建立购买后自动索评邮件/短信流
│   └── 有 → 继续
├── 收集机制是否有足够的激励？
│   ├── 没有 → 增加折扣、积分或抽奖激励
│   └── 有 → 继续
├── 产品本身是否具有"可分享性"（Instagrammable）？
│   ├── 包装简陋/体验平庸 → 优化开箱体验（转交 afa-product/afa-creative）
│   └── 体验很好 → 继续
└── 是否有明确的社区互动话题？
    ├── 没有 → 策划品牌专属 Hashtag 挑战或互动活动
    └── 有 → 寻找并培养首批"种子倡导者"

处方：
├── 轻度：优化索评邮件的文案和激励机制
├── 中度：设计完整的 UGC 收集与权利管理工作流
└── 重度：引入 ICE 框架，从零构建品牌大使/倡导者计划
```

## 3. 诊断模式 3：声誉危机爆发

```
触发信号：
├── 用户说"我们在网上被骂惨了"
├── 负面评价突然激增
├── 出现 KOL 抵制或媒体负面报道

诊断决策树：
├── 危机处于什么级别？
│   ├── Level 1（低风险）→ 快速客服响应 + 补偿
│   ├── Level 2（中风险）→ 公关负责人发布透明声明
│   └── Level 3（高风险）→ 创始人出面 + 立即整改
├── 负面的核心痛点是什么？
│   ├── 产品质量/安全 → 立即停售/召回，转交 afa-product
│   ├── 物流/服务 → 透明沟通，转交 afa-cx
│   └── 价值观/公关失误 → 真诚道歉，停止相关营销
└── 危机是否还在扩散？
    ├── 是 → 暂停所有自动化广告和邮件，集中精力降温
    └── 否 → 进入危机后修复期（收集正面 UGC 冲淡负面）

处方：
├── 紧急：立即生成对应级别的危机响应声明模板
├── 中期：建立负面声量监控阈值预警机制
└── 长期：制定声誉修复计划（SEO 压制、正面公关战役）
```

## 4. 诊断模式 4：品牌资产被侵权

```
触发信号：
├── 用户说"有人在盗用我们的图/卖假货"
├── 发现拼写错误的钓鱼域名
├── 社交媒体出现假冒官方账号

诊断决策树：
├── 侵权类型是什么？
│   ├── 盗图/盗文案 → 准备 DMCA Takedown Notice
│   ├── 假冒产品/商标滥用 → 准备 Cease & Desist Letter
│   └── 假冒账号/钓鱼网站 → 准备平台滥用报告
├── 是否掌握确凿证据？
│   ├── 没有 → 指导用户截图、收集 URL 和时间戳
│   └── 有 → 继续
└── 侵权方所在的平台/主机商是谁？
    ├── 明确 → 查找对应的官方投诉通道
    └── 隐藏 → 使用 WHOIS 查询工具定位主机商

处方：
├── 紧急：生成对应的投诉/警告函件模板（DMCA / C&D）并指导发送（建议发送前由专业律师审查）
├── 预防：建立定期的品牌资产监控工作流
```

## 5. PR 成熟度五级模型

```
Level 0：隐形品牌（Invisible）
├── 特征：搜索品牌名只有官网和社交账号，零第三方提及
├── 核心问题：没有任何 Earned Media 资产
├── 优先行动：构建 Media Kit → 学习 CPR 框架 → 发送首批 10 封 Pitch
└── 目标：在 90 天内获得首次媒体报道

Level 1：初始曝光（First Mentions）
├── 特征：有 1-5 次媒体提及，但缺乏系统性
├── 核心问题：PR 是偶发的，没有持续的节奏
├── 优先行动：建立 PR 日历 → 制定季节性叙事矩阵 → 系统化冷推
└── 目标：每月稳定获得 2-3 次媒体提及

Level 2：稳定输出（Consistent Earned Media）
├── 特征：每月有稳定的媒体提及，开始有记者主动联系
├── 核心问题：UGC 和社区建设滞后，缺乏用户层面的社会证明
├── 优先行动：启动 UGC 飞轮 → 建立品牌倡导者网络 → 内容原子化
└── 目标：UGC 收集率达到 5%+，品牌标签月均使用量 100+

Level 3：品牌权威（Brand Authority）
├── 特征：被行业视为品类专家，创始人被邀请参加播客和行业活动
├── 核心问题：缺乏声誉保护机制，品牌资产可能被侵权
├── 优先行动：建立声誉监控体系 → 品牌保护工作流 → 危机预案
└── 目标：情感净值 > 70%，侵权下架率 > 90%

Level 4：声誉护城河（Reputation Moat）
├── 特征：品牌拥有强大的 Earned Media 资产、活跃的社区和完善的保护体系
├── 核心问题：如何将声誉优势转化为更大的商业价值
├── 优先行动：PR 驱动的 SEO 策略 → 品牌联名与跨界合作 → 行业影响力建设
└── 目标：PR 综合 ROI > 5x，品牌搜索量持续增长
```

## 6. PR 成熟度快速评估问卷

```
请回答以下 10 个问题（是/否），评估你的 PR 成熟度：

媒体关系维度：
├── Q1：你的品牌在过去 6 个月内是否被至少 3 家媒体报道过？
├── Q2：你是否有一份准备好的、随时可以发送的媒体套件（Media Kit）？
├── Q3：你是否有一个包含 20+ 位目标记者的精准媒体列表？

UGC 与社区维度：
├── Q4：你的网站产品页是否展示了带图/视频的真实用户评价？
├── Q5：你是否有自动化的购买后索评流程（邮件或短信）？
├── Q6：你是否有一个活跃的品牌社区或大使计划？

声誉与保护维度：
├── Q7：你是否定期监控社交媒体和评价平台上的品牌提及？
├── Q8：你是否有一份书面的危机响应预案？
├── Q9：你是否检查过是否有假冒你品牌的网站或社交账号？

内容复用维度：
├── Q10：你是否将每一次媒体报道都复用到了至少 3 个以上的渠道？

评分标准：
├── 0-2 个"是"：Level 0（隐形品牌）→ 从 Media Kit 和 CPR 框架开始
├── 3-4 个"是"：Level 1（初始曝光）→ 建立 PR 日历和系统化冷推
├── 5-6 个"是"：Level 2（稳定输出）→ 启动 UGC 飞轮和倡导者网络
├── 7-8 个"是"：Level 3（品牌权威）→ 建立声誉监控和保护体系
└── 9-10 个"是"：Level 4（声誉护城河）→ 优化 PR ROI 和行业影响力
```
