# UGC 与 PR 协同杠杆 (UGC × PR Synergy)

> **用途**：本文件聚焦 PR 视角下如何利用 UGC 作为媒体素材和声誉资产，用于支持媒体 Pitch、新闻稿、媒体包与声誉管理场景。
> **使用边界**：如需整理为用户可见交付物，只保留自然语言、业务角色与行动建议；涉及社区运营、素材权利管理、数据追踪等内部协同信息时，必须留在内部整理层，不在对用户的成品层直接暴露内部模块代号或内部路由。

---

## 一、UGC 作为 PR 素材的价值

### 1.1 Earned Media 放大器

UGC 是 PR 最有力的"社会证明"武器。在媒体 Pitch 中引用真实用户的声音，比品牌自说自话的新闻稿更有说服力：

- **媒体 Pitch 嵌入**：在向记者发送 Pitch 时，附上 3-5 条精选的用户好评截图或视频链接，作为"真实用户反馈"的佐证。
- **新闻稿引用**：在产品发布或里程碑新闻稿中，引用真实用户的评价（需获得授权），增强可信度。
- **记者体验包**：向记者寄送产品时，附上一份精选的 UGC 合集（打印版或数字版），展示真实用户的使用场景。

### 1.2 危机公关中的 UGC 盾牌

当品牌遭遇负面舆论时，大量真实的正面 UGC 是最有力的反击武器：

- **正面声量对冲**：在社交媒体上集中展示正面 UGC，用真实用户的声音对冲负面叙事。
- **媒体回应素材**：在回应媒体质疑时，引用真实用户的正面体验作为佐证。
- **品牌大使动员**：在危机时刻，品牌大使可以主动发声支持品牌（需要提前建立信任关系，不能临时"买水军"）。

---

## 二、UGC 权利管理的法律风险（PR 视角）

> **法律风险警告**：未经明确授权，将用户在社交媒体上发布的有机 UGC 用于品牌的付费广告、官网或媒体素材，存在严重的侵权风险。PR 团队在使用 UGC 时必须确认授权状态。

**PR 特有的授权注意事项**：

- **媒体转载风险**：如果品牌在新闻稿中引用了用户评价，媒体可能会进一步转载和传播。授权范围应明确包含"媒体报道中的引用权"。
- **危机场景下的快速授权**：提前与核心品牌大使签订包含"危机声援"条款的协议，确保在需要时能快速获得授权。
- **跨境合规**：不同国家/地区对 UGC 使用的法律要求不同（如 GDPR 下的欧洲用户数据保护），PR 团队在国际传播时需特别注意。

## 三、PR-UGC 协同工作流

在实际执行中，PR 团队通常需要与社区运营、素材权利管理和数据追踪职责协同，才能把 UGC 稳定转化为媒体资产。

### 3.1 对外可整理的协同原则

| 步骤 | PR 团队动作 | 所需协同职责 |
| :--- | :--- | :--- |
| 1. 素材发现 | 从 UGC 资产库中筛选适合 PR 使用的高质量素材 | 社区/UGC 资产管理 |
| 2. 授权确认 | 确认素材的授权范围是否覆盖媒体使用 | 素材权利管理 |
| 3. 媒体包制作 | 将精选 UGC 整合到媒体包（Media Kit）中 | 本模块执行 |
| 4. Pitch 嵌入 | 在媒体 Pitch 中引用 UGC 作为社会证明 | 本模块执行 |
| 5. 效果追踪 | 追踪含 UGC 的 Pitch 的媒体采纳率 | 数据追踪与复盘 |

### 3.2 内部映射说明（仅供系统使用，不直接进入用户交付物）

| 步骤 | PR 团队动作 | 协同方 |
| :--- | :--- | :--- |
| 1. 素材发现 | 从 UGC 资产库中筛选适合 PR 使用的高质量素材 | afa-social（资产库管理） |
| 2. 授权确认 | 确认素材的授权范围是否覆盖媒体使用 | afa-social（权利管理） |
| 3. 媒体包制作 | 将精选 UGC 整合到媒体包（Media Kit）中 | 本模块执行 |
| 4. Pitch 嵌入 | 在媒体 Pitch 中引用 UGC 作为社会证明 | 本模块执行 |
| 5. 效果追踪 | 追踪含 UGC 的 Pitch 的媒体采纳率 | afa-dashboard（数据追踪） |

> **内部交接规则（仅供系统使用）**：如果用户询问的是 UGC 收集策略、社区飞轮构建、品牌大使计划的运营细节，请在内部整理阶段回交社区运营相关模块。本模块仅负责 UGC 在 PR/媒体场景中的应用。
