# TikTok 原生广告创意模板库

> **版本**: 1.1 | **适用平台**: TikTok / TikTok Shop | **核心原则**: 原生感 > 制作感，真实 > 精致
> **通用文案框架**：PAS/HSO/BAB/AIDA 等通用文案框架、CTA 变体库、UGC 全漏斗应用框架已统一归入 **afa-creative**（参见 `copywriting-frameworks.md`）。季节性创意日历参见 afa-creative 的 `seasonal-creative-calendar.md`。通用创意测试方法论参见 afa-creative 的 `ad-testing-matrix.md`。UGC 脚本法则参见 afa-creative 的 `reali-tea-scripting.md`。本文件仅保留 TikTok 平台特有的原生创意知识。

## 一、20 大 TikTok Hook 公式（前 3 秒生死线）

> **Hook 是 TikTok 广告的生死线。前 1-3 秒决定 80% 的广告效果。**
> 以下 Hook 公式均附带 TikTok 平台特有的拍摄指导（镜头、语气、适用场景）。

### 类型 A：问题痛点型

**A1 — "你是不是也…" 共鸣式**
```
"你是不是也受够了每次洗完头发就像稻草？"
"你是不是也觉得化妆品越买越多，皮肤越来越差？"
"你是不是也在为找不到合适的 [品类] 而头疼？"
```
**适用**: 解决方案型产品 | **语气**: 闺蜜聊天 | **镜头**: 对镜自拍，表情略带无奈

**A2 — "我以前也…直到…" 转折式**
```
"我以前也觉得防晒霜都很油腻，直到我发现了这个…"
"我以前花了几百块买 [竞品品类]，直到朋友推荐了这个 $29 的替代品…"
"我以前以为 [常见误解]，直到我试了这个方法…"
```
**适用**: 替代品/升级品 | **语气**: 分享发现 | **镜头**: 先展示旧产品，再切换新产品

**A3 — "别再…了" 警告式**
```
"别再用毛巾擦脸了，你的毛孔会感谢你的"
"别再花冤枉钱买 [品类] 了"
"别再犯这个 [品类] 的错误了"
```
**适用**: 教育型内容 | **语气**: 专家忠告 | **镜头**: 直视镜头，手势强调

**A4 — "如果你还在…你需要看这个" 条件式**
```
"如果你还在用传统的 [旧方案]，你需要看这个"
"如果你的 [问题] 一直没解决，可能是因为这个原因"
"如果你每个月在 [品类] 上花超过 $50，这个视频能帮你省一半"
```
**适用**: 升级换代产品 | **语气**: 理性建议 | **镜头**: 对比展示

### 类型 B：好奇悬念型

**B1 — "这个 [产品] 让我…" 结果悬念式**
```
"这个 $15 的小东西让我的厨房彻底变了样"
"这个面霜让我的同事以为我打了水光针"
"这个 app 让我一个月省了 $200"
```
**适用**: 效果显著的产品 | **语气**: 惊喜分享 | **镜头**: 先展示结果，再揭秘产品

**B2 — "我终于找到了…" 发现式**
```
"我终于找到了不会脱妆的粉底液"
"找了三年，终于找到了完美的旅行包"
"我终于知道为什么韩国女生皮肤那么好了"
```
**适用**: 长期痛点解决方案 | **语气**: 如释重负 | **镜头**: 兴奋展示产品

**B3 — "POV: 你发现了…" 沉浸式**
```
"POV: 你发现了一个能让你的 [问题] 消失的神器"
"POV: 你的朋友问你皮肤怎么突然变好了"
"POV: 你终于找到了那个 TikTok 上爆火的 [产品]"
```
**适用**: 年轻受众 | **语气**: 第二人称沉浸 | **镜头**: 主观视角

**B4 — "等等…这也太…了吧" 惊叹式**
```
"等等…这个颜色也太好看了吧"
"等等…这个价格是认真的吗？"
"等等…这个效果是真的吗？"
```
**适用**: 视觉冲击力强的产品 | **语气**: 真实惊讶 | **镜头**: 开箱/试用瞬间

### 类型 C：社会证明型

**C1 — "TikTok 让我买的…" 平台原生式**
```
"又是 TikTok 让我买的东西，但这次真的值"
"TikTok made me buy it，而且我已经回购三次了"
"好吧，我终于买了 TikTok 上那个爆火的 [产品]"
```
**适用**: 已有一定热度的产品 | **语气**: 从众但真实 | **镜头**: 开箱

**C2 — "XX 万人都在用的…" 数据式**
```
"50 万人都在用的睡眠喷雾，到底有没有用？"
"这个卖了 100 万件的 [产品]，我来测评一下"
"全网好评 4.9 分的 [产品]，真的有那么好吗？"
```
**适用**: 高销量产品 | **语气**: 测评者 | **镜头**: 产品 + 评论截图

**C3 — "我 [身份] 朋友推荐的…" 权威式**
```
"我做皮肤科医生的朋友推荐我用这个"
"我健身教练说这个蛋白粉是他用过最好的"
"我做了 10 年美发的姐姐说这个洗发水真的不一样"
```
**适用**: 需要专业背书的品类 | **语气**: 转述推荐 | **镜头**: 日常使用场景

### 类型 D：对比冲突型

**D1 — "A vs B" 直接对比式**
```
"$15 的 vs $60 的，你猜哪个更好用？"
"网红推荐的 vs 我自己发现的，差距也太大了"
"第一次用 vs 用了一个月，变化太明显了"
```
**适用**: 性价比产品/效果产品 | **语气**: 客观测评 | **镜头**: 分屏对比

**D2 — "我把 [贵的] 换成了 [便宜的]" 替代式**
```
"我把 $200 的 La Mer 换成了这个 $30 的，效果居然一样"
"我把健身房会员换成了这个家用设备，省了一大笔"
"我把三个 app 换成了这一个，效率翻倍"
```
**适用**: 平替产品 | **语气**: 精明消费者 | **镜头**: 两个产品并排

### 类型 E：行动指令型

**E1 — "停！你需要知道…" 紧急式**
```
"停！如果你正在考虑买 [品类]，先看完这个"
"停下来！这可能是你今年看到的最有用的视频"
"等一下！在你买 [竞品] 之前，你需要知道这个"
```
**适用**: 高客单价决策品 | **语气**: 紧急提醒 | **镜头**: 突然出现，手势制止

**E2 — "保存这个视频" 收藏式**
```
"保存这个视频，下次你需要 [场景] 的时候会感谢我的"
"先收藏，这个 [品类] 的选购指南你迟早用得上"
"Mark 一下，这是我用过最好的 [品类] 清单"
```
**适用**: 教程/指南类 | **语气**: 实用建议 | **镜头**: 清单展示

## 二、7 大原生视频格式模板

### 格式 1：UGC 开箱测评（15-30 秒）

```
[0-3s] Hook（选择上方 20 个公式之一）
[3-5s] 产品开箱/首次展示
[5-10s] 第一印象反应（真实表情 + 简短评价）
[10-18s] 使用演示（关键功能展示 1-2 个）
[18-25s] 效果展示 / Before-After
[25-30s] 总结评价 + CTA
```

**拍摄要点**：自然光，手机前置镜头；不化全妆（或只化日常妆）；背景为家里真实环境（卧室/浴室/厨房）；语速正常偏快，像跟朋友说话；表情真实反应，不要演。

**CTA 变体**："链接在 bio / 小黄车里" / "评论区告诉我你们想看什么色号" / "关注我看一周后的效果更新"

### 格式 2：问题-解决方案（15-30 秒）

```
[0-3s] Hook — 痛点共鸣（A1-A4 公式）
[3-8s] 放大痛点 — 展示问题场景（失败/困扰的画面）
[8-12s] 转折 — "直到我发现了这个…"
[12-20s] 产品展示 + 使用演示
[20-25s] 效果证明（Before-After / 数据 / 反应）
[25-30s] CTA + 紧迫感
```

**拍摄要点**：痛点部分要有真实感（真的在经历这个问题）；转折要自然，不要太突兀；效果展示要有视觉冲击力。

### 格式 3：GRWM / 日常融入（30-60 秒）

```
[0-3s] Hook — "跟我一起准备出门 / 做早餐 / 整理房间"
[3-15s] 日常流程展示（自然融入产品使用）
[15-25s] 产品特写 + 简短介绍（像是顺便提到）
[25-40s] 继续日常流程 + 使用效果自然展现
[40-50s] 整体效果展示
[50-60s] 轻松 CTA — "对了，这个 [产品] 链接在…"
```

**拍摄要点**：产品是日常的一部分，不是主角；不要刻意推销，像是无意间分享；背景音乐选择当下热门的轻松曲目。

### 格式 4：Before-After 对比（15-20 秒）

```
[0-2s] Hook — "Day 1 vs Day 30" / "Before vs After"
[2-5s] Before 状态（真实、不加滤镜）
[5-8s] 使用过程快进（产品出镜）
[8-12s] After 状态（同角度、同光线）
[12-15s] 产品特写 + 名称
[15-20s] CTA
```

**拍摄要点**：Before 和 After 必须同角度、同光线；不要用滤镜（或两张都用同一个滤镜）；时间跨度要真实标注。

### 格式 5：教程 / How-To（30-60 秒）

```
[0-3s] Hook — "3 步搞定 [目标]" / "你一直做错的 [事情]"
[3-8s] 步骤 1（文字叠加 + 演示）
[8-15s] 步骤 2（文字叠加 + 演示）
[15-22s] 步骤 3（文字叠加 + 演示）
[22-28s] 最终效果展示
[28-35s] 产品信息 + CTA
```

**拍摄要点**：每步都要有文字叠加（很多人静音观看）；步骤要简单，3-5 步最佳；节奏要快，不要拖沓。

### 格式 6：Stitch / Duet 回应式（15-30 秒）

```
[0-3s] 原视频片段（问题/观点/挑战）
[3-5s] 你的反应 — "好，让我来回答这个"
[5-15s] 你的回应内容（融入产品）
[15-25s] 产品展示 + 效果证明
[25-30s] CTA + 邀请互动
```

**拍摄要点**：选择热门的原视频（增加曝光）；回应要有价值，不要纯广告；保持对话感。

### 格式 7：TikTok Shop 直播切片（15-30 秒）

```
[0-3s] 直播中的精彩瞬间（惊喜反应/产品演示高光）
[3-10s] 产品核心卖点快速展示
[10-18s] 价格/优惠信息
[18-25s] 用户评论/好评截图
[25-30s] "去我主页/小黄车下单" CTA
```

**拍摄要点**：从直播录像中剪辑最精彩的片段；保留直播的真实感和互动感；加上字幕和价格标注。

## 三、10 大原生广告格式（深度版）

### 格式 1：产品对比（Product Comparison）

**适用场景**：有明确竞品或替代方案的产品

**结构**：
```
[Hook] "别再买 X 了，试试这个"
[展示竞品] 快速展示竞品的问题
[展示产品] 演示你的产品如何更好
[对比] 并排对比关键差异
[CTA] "链接在这里，自己对比一下"
```

**品类示例**：美妆（"我把 $60 的精华和这个 $25 的对比了一下…"）；食品（"普通蛋白粉 vs 这个，看看溶解度差距"）；家居（"传统拖把 vs 这个自动清洁的，差距太大了"）。

**关键要点**：不要贬低竞品，而是展示差异；用视觉演示而非口头描述；对比维度要选消费者最关心的。

### 格式 2：产品功效演示（Product Benefit）

**适用场景**：功效可视化的产品

**结构**：
```
[Hook] 先点出痛点，不提产品
[痛点放大] 让观众产生共鸣
[产品登场] 自然引入产品
[演示] 实际使用过程
[结果] 展示效果
[CTA] 自然收尾
```

**关键要点**：先说痛点，再说方案（不要上来就推产品）；调动情感，不只是功能描述。核心技巧："他不是说'这个能清理胡渣'，而是说'你女朋友是不是每次看到洗手台上的胡渣都要疯了？'"

### 格式 3：产品评测（Product Review）

**适用场景**：所有品类，尤其适合 UGC

**结构**：
```
[Hook] "说实话，我一开始也不信…"
[背景] 分享使用前的状态
[体验] 详细描述使用过程
[反驳替代方案] "你可以花 $30/月买 X，但是…"
[结果] 真实的使用效果
[推荐] 真诚的推荐（带保留意见更可信）
```

**关键要点**：真实 > 完美，带点小缺点的评测更可信；长视频（3-5 分钟）在 TikTok 上也能表现好；评论区互动是评测视频的关键转化点。

### 格式 4：创始人故事（Founder Story）

**适用场景**：有真实创业故事的品牌

**结构**：
```
[Hook] 第三人称开场 "她曾经为十亿级品牌做配方…"
[痛点] 创始人遇到的问题
[转折] 为什么决定自己做
[旅程] 创业过程的关键节点
[产品] 自然引出产品
[愿景] 品牌使命
```

**关键要点**：创始人故事是 TikTok 上最强的品牌建设工具之一；可以多次重新包装（不同角度、不同细节）；每 1-2 秒切换画面（创始人、产品、幕后、评论截图）；可以有机发布 + 付费推广双重使用。

### 格式 5：限时优惠（Offer-Based）

**适用场景**：TikTok Shop 促销、捆绑优惠

**结构**：
```
[Hook] "千万别买 54 包的…"
[反转] "因为现在 90 包和 54 包一个价！"
[展示] 产品展示 + 价格对比
[紧迫感] "不知道还能持续多久"
[CTA] "链接在购物车"
```

**关键要点**：配合 TikTok Shop 折扣功能使用；适合已经被其他格式"种草"的用户；简单直接，不需要复杂制作。

### 格式 6：争议性内容（Controversial）

**适用场景**：愿意制造话题的品牌

**结构**：
```
[Hook] 制造争议点
[展示] 展示引发争议的产品/用法
[评论引导] 引发两极化讨论
[自然传播] 争议驱动分享
```

**关键要点**：争议不等于冒犯，而是制造"50/50 分裂"；案例：母女穿同款裙子 → $230,000 销售额；评论区的争论 = 免费的算法推动力；不适合所有品牌，需要评估品牌调性。

### 格式 7：打包发货（Pack & Order With Me）

**适用场景**：所有实体产品品类

**结构**：
```
[Hook] "这个订单是给 [名字] 的…"
[打包过程] 展示仔细打包的过程
[产品介绍] 边打包边介绍产品
[个人化] 手写感谢卡等细节
[发货] 贴标签、放入快递袋
```

**关键要点**：ASMR 版本、旁白版本、快节奏版本都有效；建立亲密感和期待感；有品牌做了 1,000+ 个打包视频作为核心策略；制作成本极低，可以大量产出。

### 格式 8：日常融入（Get Ready With Me）

**适用场景**：美妆、护肤、服饰、保健品、咖啡等日常用品

**结构**：
```
[Hook] "今天试试我在 [活动] 买的东西…"
[日常场景] 早晨/晚间/出门准备
[自然使用] 在日常流程中使用产品
[真实反应] 第一次使用的真实感受
[推荐] 自然的使用心得
```

**关键要点**：像给朋友 FaceTime 一样拍摄；越随意越好，不需要完美妆容/布景；可以结合多个产品（但突出主推产品）。

### 格式 9：纯音乐展示（Music-Only Showcase）

**适用场景**：视觉冲击力强的产品

**结构**：
```
[热门音乐] 配合当前流行音乐
[产品展示] 多角度展示产品
[使用场景] 展示产品在场景中的效果
[文字叠加] 关键卖点用文字呈现
```

**关键要点**：不需要配音，降低制作门槛；适合产品本身就很"酷"的品类；可以快速批量制作。

### 格式 10：三个理由（Three Reasons Why）

**适用场景**：所有品类

**结构**：
```
[Hook] "三个理由你不应该买这个产品…"（反转 Hook）
[理由 1] 第一个"不该买"的理由（实际是优点）
[理由 2] 第二个理由
[理由 3] 第三个理由
[CTA] 自然收尾
```

**关键要点**：反转 Hook（"不该买"→ 实际是优点）效果最好；列表格式天然适合 TikTok 的快节奏；可以无限变体：3 个错误、3 个技巧、3 个秘密…

## 四、TikTok Shop 直播脚本框架

### 直播开场（前 5 分钟）

```
[0:00-0:30] 欢迎 + 今日预告
"欢迎来到直播间！今天给大家带来 [主题]，
有 [数量] 款产品要分享，而且有独家直播价！
先点个关注，不然等下找不到我们了"

[0:30-2:00] 暖场互动
"先问大家一个问题：你们平时 [相关问题] 是怎么解决的？
打在公屏上，我来看看"
（读评论，互动）

[2:00-5:00] 第一款产品预热
"好，今天第一款要给大家看的是…
（先不展示，制造悬念）
这个产品在我们店铺已经卖了 [数量]，
好评率 [百分比]，今天直播间有特别价"
```

### 单品讲解框架（每品 5-10 分钟）

```
[展示] 产品外观 + 包装
[痛点] "你们有没有遇到过 [问题]？"
[方案] "这个产品就是专门解决这个的"
[演示] 现场使用/试用（最关键的环节）
[证明] 展示评论截图 / Before-After
[价格] "正常价 $XX，今天直播间只要 $XX"
[紧迫] "库存只有 XX 件 / 这个价格只有今天"
[CTA] "点击下方小黄车，第 X 号链接"
```

## 六、TikTok 创意测试方法论

### 6.1 TikTok Creative Center 研究法

**Step 1：Top Ads 研究**
- 进入 Creative Center → Top Ads
- 按品类、地区、时间段筛选
- 关注 CTR 排名前 10% 的广告
- 记录 Hook 类型、时长、文案

**Step 2：竞品广告研究**
- 使用 TikTok Ad Library 搜索竞品
- 使用第三方工具（如 Pipiads、Minea）按 GMV 排序
- 关注实际产生销售的广告，而非仅有播放量的

**Step 3：Hook 矩阵构建**
- 为每个产品准备至少 7 个不同 Hook
- 覆盖 A/B/C/D/E 五类 Hook 类型
- 每周测试 2-3 个新 Hook
- 记录每个 Hook 的 CTR 和 CVR 数据

### 6.2 TikTok 创意测试变量隔离

| 测试轮次 | 测试变量 | 控制组 | 变体 A | 变体 B | 变体 C |
|:---:|:---|:---|:---|:---|:---|
| 第 1 轮 | **Hook 类型** | 痛点式 (A1) | 好奇式 (B1) | 社证式 (C1) | 对比式 (D1) |
| 第 2 轮 | **视频格式** | UGC 测评 | 问题-解决 | Before-After | 教程 |
| 第 3 轮 | **CTA 类型** | "链接在 bio" | "评论区留言" | "点击小黄车" | "限时优惠" |
| 第 4 轮 | **创作者类型** | 品牌自拍 | 年轻女性 UGC | 专家/KOL | 情侣/家庭 |
| 第 5 轮 | **视频长度** | 15 秒 | 30 秒 | 45 秒 | 60 秒 |

### 6.3 TikTok 测试预算分配

```
每轮测试预算: $200-$500
每个变体: $50-$125
测试周期: 3-5 天
最低数据量: 每变体 1,000+ 展示
胜出标准: 
  - TOFU: 最低 CPM + 最高 CTR
  - MOFU: 最高 CTR + 最低 CPC
  - BOFU: 最高 CVR + 最低 CPA
```

### 6.4 TikTok 创意产出节奏

| 阶段 | 每周新创意数 | 格式分布 | Hook 测试 |
|:---|:---|:---|:---|
| 冷启动 | 7-10 个 | 覆盖 5+ 格式 | 每个创意不同 Hook |
| 验证期 | 5-7 个 | 聚焦 Top 3 格式 | 迭代 Top Hook |
| 扩量期 | 10-15 个 | Top 格式 × 新角度 | 系统化 A/B 测试 |
| 成熟期 | 7-10 个 | 维护 + 创新 | 持续刷新 |

## 七、TikTok 原生创意第一原则

> **在 TikTok 上，最好的广告看起来不像广告。**

**Reali-TEA 真实感速查表**（完整版请参见 afa-creative 的 `reali-tea-scripting.md`）：

| 维度 | 规则 | 示例 |
|:---|:---|:---|
| **语言** | 使用口语化表达，避免营销腔 | "说实话这个东西真的救了我" vs "革命性产品改变您的生活" |
| **犹豫感** | 加入真实的犹豫和思考 | "一开始我也觉得是智商税…" |
| **不完美** | 允许轻微的口误、停顿 | 自然的"嗯"、"就是那个…" |
| **环境** | 真实生活场景，非摄影棚 | 卧室、厨房、浴室、车内 |
| **镜头** | 手持、略微偏移、非完美构图 | 自拍视角、前置摄像头 |
| **节奏** | 每 1-2 秒切换画面/角度 | 快节奏剪辑，保持注意力 |
| **CTA** | 非推销式，自然引导 | "链接在评论区，别怪我没告诉你" vs "立即购买" |
