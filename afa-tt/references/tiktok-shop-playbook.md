# TikTok Shop 运营手册

> 本文件为 TikTok 广告模块的内部参考文件，用于 TikTok Shop US/UK/SEA 场景下的开店、运营、流量与双轨协同策略整理。
> 如需整理为用户可见交付物，必须删除内部路由标签、模块代号和系统字段，只保留自然语言、业务角色与行动建议。
> 如需追溯职责边界、最终裁决或与其他协议冲突时的优先级，以当前模块 `SKILL.md` 与 `_system/` 全局协议为准。

## 一、TikTok Shop 生态全景

### 1.1 三大流量入口

| 入口 | 流量占比 | 特点 | 优先级 |
|:---|:---:|:---|:---:|
| **短视频购物车** | 常为主要来源之一 | 内容驱动，算法推荐，爆发力强 | ⭐⭐⭐⭐⭐ |
| **直播购物** | 常为重要来源之一 | 实时互动，转化率高，需持续投入 | ⭐⭐⭐⭐ |
| **商城搜索** | 常为补充且稳定的来源 | 主动搜索，意图明确，稳定增长 | ⭐⭐⭐ |

### 1.2 TikTok Shop vs 独立站对比

| 维度 | TikTok Shop | Shopify 独立站 |
|:---|:---|:---|
| 流量来源 | 平台内推荐 + 搜索 | 付费广告 + SEO + 直接 |
| 用户意图 | 发现式购物（低意图） | 搜索式购物（高意图） |
| 转化路径 | 视频/直播 → 小黄车 → 下单 | 广告 → 落地页 → 加购 → 结账 |
| 客单价 | 常偏中低客单 | 灵活，通常可覆盖更宽价格带 |
| 退货率 | 常相对更高 | 常相对更低 |
| 利润率 | 较低（平台佣金 + 低客单） | 较高（自主定价） |
| 品牌建设 | 弱（平台主导） | 强（自主品牌体验） |
| 数据所有权 | 有限 | 完全拥有 |

### 1.3 最佳策略：TikTok Shop + 独立站双轨并行

```
TikTok Shop 角色:
├── 获客入口（低门槛试购）
├── 爆品测试场（快速验证市场需求）
├── 品牌曝光渠道（内容种草）
└── 新客获取漏斗顶部

独立站角色:
├── 品牌主阵地（完整品牌体验）
├── 高客单转化（Bundle / 订阅）
├── 客户数据中心（Email / SMS 列表）
└── 复购与 LTV 最大化
```

## 二、TikTok Shop 开店完整 SOP

### 2.1 资质准备

**美国站 (US) 要求**:
- 美国公司注册（LLC / Corporation）
- EIN 税号
- 美国银行账户（或 Payoneer / PingPong）
- 仓库地址（可用海外仓）
- 品牌商标（非硬性前置，但通常强烈建议尽早准备）

**跨境卖家方案**:
- 通过 TikTok Shop 跨境通道申请
- 需要中国/香港公司营业执照
- 支持 Payoneer 收款
- 发货时效要求：应按目标市场与平台当前政策，承诺可稳定履约的送达窗口

### 2.2 店铺设置清单

```
□ 店铺基本信息
  ├── 店铺名称（与品牌一致）
  ├── Logo（高清，白底或透明底）
  ├── 店铺简介（包含品牌故事和核心卖点）
  └── 联系方式（客服邮箱 + 电话）

□ 物流设置
  ├── 发货地址（仓库地址）
  ├── 退货地址
  ├── 配送模板（免邮门槛设置）
  └── 发货时效承诺

□ 支付设置
  ├── 银行账户绑定
  ├── 税务信息填写
  └── 结算周期确认

□ 政策页面
  ├── 退换货政策（建议按目标市场、平台要求与履约能力设定清晰可执行的时限与条件）
  ├── 隐私政策
  ├── 服务条款
  └── 配送政策
```

### 2.3 产品上架优化

**标题优化公式**:
```
[核心关键词] + [材质/成分] + [核心卖点] + [适用场景/人群]

示例:
✅ "Vitamin C Brightening Serum - 20% Pure L-Ascorbic Acid - Dark Spot Corrector for All Skin Types"
❌ "Best Serum Ever!! Amazing Product You Need!!"
```

**产品图片要求**:
| 图片位置 | 要求 | 建议 |
|:---|:---|:---|
| 主图 | 白底，产品主体应足够突出且保持清晰留白 | 使用平台常见且清晰的主图比例，无文字叠加 |
| 图 2 | 产品细节/成分 | 展示质感和关键成分 |
| 图 3 | 使用场景 | 真实使用场景照片 |
| 图 4 | Before-After / 效果 | 真实效果对比 |
| 图 5 | 尺寸/规格参考 | 与常见物品对比大小 |
| 补充图位 | 用户好评截图 / 包装 | 社会证明 + 开箱体验 |

**产品描述结构**:
```
第 1 段: 核心价值主张（解决什么问题）
第 2 段: 若干关键卖点（可用 emoji + 短句）
第 3 段: 使用方法 / 适用场景
第 4 段: 成分/材质/规格
第 5 段: 品牌承诺（退款保障/质量保证）
```

**定价策略**:
```
TikTok Shop 常见于中低客单价格带，但实际价格仍应由品类、毛利结构、履约成本与内容转化效率共同决定。
定价公式: 应在覆盖平台佣金、退货成本、履约费用与合理利润空间后，形成可持续的价格带，而不是套用统一倍数。

价格锚定技巧:
├── 设置"原价"（划线价），折扣力度应与真实日常售价和活动策略一致
├── 设置多规格（小/中/大），引导中间选项
├── Bundle 套装比单品有明显价格优势
└── 新客专享价（首单优惠）
```

## 三、GMV Max 广告策略

### 3.1 GMV Max 核心概念

GMV Max 是 TikTok Shop 的 AI 驱动广告系统，类似 Meta 的 ASC：
- **自动优化**：AI 自动选择最佳受众、出价和版位
- **目标**：最大化 GMV（商品交易总额）
- **素材来源**：自动抓取店铺视频 + 直播片段

### 3.2 GMV Max 启动 SOP

**Phase 1: 准备期（投放前）**
```
□ 确保已有足够数量的产品相关短视频可供系统学习
□ 产品页面优化完成（标题、图片、描述、评论）
□ 设置与毛利、回收期和经营目标一致的 ROI 目标
□ 准备与测试风险承受度相匹配的日预算
```

**Phase 2: 启动期（初始学习窗口）**
```
设置:
├── 目标: GMV Maximization
├── ROI 目标: 以偏保守且可持续的起点让系统学习
├── 日预算: 采用与测试阶段匹配的起始预算
├── 产品: 选择少量主推产品作为首批测试对象
└── 素材: 让系统自动选择

注意:
├── 不要频繁调整，应给系统留下足够学习窗口
├── 初始学习阶段出现波动通常属于正常现象
└── 关注 GMV 和 ROI，不要只看 ROAS
```

**Phase 3: 优化期（学习后观察窗口）**
```
根据数据调整:
├── ROI 达标 → 可逐步提高预算，但增幅应与稳定度和承压能力匹配
├── ROI 略低 → 可适度放宽目标或延长观察窗口，再决定是否继续学习
├── ROI 远低 → 检查产品页面和视频质量
└── 某产品表现差 → 移除，换其他产品

优化方向:
├── 持续发布新视频（频率按素材产能与疲劳速度安排）
├── 优化产品页面（标题、图片、评论）
├── 调整价格策略（测试不同折扣力度）
└── 增加 Bundle 选项
```

**Phase 4: 扩量期（相对稳定后）**
```
扩量策略:
├── 预算扩量: 按稳定度分批增加，避免一次性放量过快
├── 产品扩量: 逐步添加更多产品
├── 内容扩量: 增加 UGC 创作者数量
└── 直播扩量: 增加直播频率和时长

高级技巧:
├── 分时段投放（高转化时段加预算）
├── 配合直播时段加大投放
├── 利用平台活动（大促期间加大投入）
└── 跨渠道协同（TikTok 种草 → 独立站转化）
```

### 3.3 GMV Max 诊断规则

| 问题 | 可能原因 | 解决方案 |
|:---|:---|:---|
| 花不出去钱 | ROI 目标过高 / 素材不足 | 降低 ROI 目标 / 增加视频数量 |
| ROI 低于目标 | 产品页面差 / 价格不合理 | 优化产品页 / 调整定价 |
| 点击高但不转化 | 产品页与视频不匹配 | 确保视频展示的就是实际产品 |
| 展示量突然下降 | 素材疲劳 / 竞争加剧 | 更新视频素材 / 调整预算 |
| 退货率高 | 产品描述不准确 / 质量问题 | 修正描述 / 改善产品质量 |

## 四、Affiliate 机器运营手册

### 4.1 Affiliate 生态概览

```
品牌方
├── 设置佣金计划
├── 提供产品样品
├── 审核创作者
└── 管理合作关系

创作者/达人
├── 申请合作
├── 创作内容（视频/直播）
├── 挂载商品链接
└── 赚取佣金

平台
├── 匹配品牌和创作者
├── 追踪销售归因
├── 处理佣金结算
└── 提供数据分析
```

### 4.2 佣金策略设计

**佣金率参考（按品类）**:

| 品类 | 建议佣金率 | 说明 |
|:---|:---:|:---|
| 美妆护肤 | 常可给较高佣金 | 高复购，可给更积极的激励 |
| 服饰配件 | 常为中位佣金带 | 中等利润率 |
| 家居生活 | 常为中低至中位佣金带 | 客单价较高 |
| 食品保健 | 常可给较高佣金 | 高复购 + 高利润 |
| 电子配件 | 常偏保守 | 利润率较低 |
| 宠物用品 | 常为中位佣金带 | 高情感连接 |

**分层佣金策略**:
```
开放计划（所有创作者可申请）:
├── 基础佣金: 采用可持续的基础档位
└── 目的: 广撒网，获取大量 UGC

定向计划（邀请特定创作者）:
├── 高佣金: 采用更有竞争力的激励档位
├── 免费样品
└── 目的: 获取高质量内容

独家计划（头部达人）:
├── 最高佣金: 仅在利润与合作价值可覆盖时使用更高激励档位
├── 免费样品 + 创作费
├── 独家折扣码
└── 目的: 品牌背书 + 爆款打造
```

### 4.3 创作者招募 SOP

**Step 1: 筛选标准**
```
基础条件:
├── 粉丝数达到平台当前可合作门槛
├── 近期有稳定发布，且内容仍在活跃更新
├── 内容与品类相关
└── 无违规记录

优选条件:
├── 互动率应处于该体量创作者中的可接受或较优水平（往往比粉丝数更重要）
├── 有过 TikTok Shop 带货经验
├── 内容风格与品牌调性匹配
└── 评论区互动积极
```

**Step 2: 批量邀请模板**
```
Hi [创作者名]! 👋

我是 [品牌名]，我们做 [一句话品牌描述]。

看了你的内容，觉得你的风格特别适合我们的产品！
想邀请你加入我们的 TikTok Shop Affiliate 计划：

🎁 免费产品样品（价值 $XX）
💰 [XX]% 佣金（每卖一单你赚 $XX）
📈 我们会用 Spark Ads 推广你的视频（额外曝光）

有兴趣的话，在 TikTok Shop 搜索 [品牌名] 
→ 点击"联系商家"就可以申请合作

期待合作！🙌
```

**Step 3: 样品寄送管理**
```
样品寄送规则:
├── 小体量创作者: 通常先寄送少量主推产品做首轮验证
├── 中腰部创作者: 可根据内容能力寄送更丰富的产品组合（必要时含新品）
├── 头部创作者: 仅在合作确定性与预算可覆盖时再提高寄样规格与配套物料
└── 所有样品可附带产品说明卡与品牌故事卡

跟进时间线:
├── 寄出后一段合理时间内: 确认收到
├── 收到并体验后: 适度提醒发布或反馈素材计划
├── 发布后较短窗口内: 感谢 + 初步反馈
└── 发布后完成首轮数据沉淀后: 回顾结果并讨论是否续约
```

### 4.4 Affiliate 数据监控

**关键指标**:
| 指标 | 计算方式 | 健康基准 |
|:---|:---|:---|
| 创作者激活率 | 发布内容的创作者 / 总合作创作者 | 应保持在可持续且持续改善的水平 |
| 平均佣金成本 | 总佣金支出 / 总 GMV | 应与毛利结构相匹配 |
| 创作者 ROI | 创作者带来的 GMV / (样品成本 + 佣金) | 应高于品牌可接受的回收要求 |
| 内容产出率 | 总视频数 / 总合作创作者 | 应与样品投入和合作深度相匹配 |
| 爆款率 | 高播放视频 / 总视频数 | 更适合作为内容势能参考，而非单一门槛 |

## 五、TikTok Shop 运营日历

### 5.1 日常运营节奏

```
每日:
├── 检查订单和发货状态
├── 回复客户消息（按平台服务要求与团队 SLA 尽快处理）
├── 监控 GMV Max 广告数据
└── 查看新的 Affiliate 申请

每周:
├── 发布适量新短视频（按素材产能与频道节奏安排）
├── 审核和邀请新创作者
├── 分析上周数据（GMV / ROI / 退货率）
└── 优化产品页面（标题 / 图片 / 价格）

每月:
├── 月度数据复盘
├── 调整佣金策略
├── 更新产品线（上新 / 下架）
├── 规划下月内容日历
└── 评估 Affiliate 表现（续约 / 终止）
```

### 5.2 大促运营 SOP

**大促前预热窗口**:
```
□ 确定参与的促销活动
□ 准备促销产品和库存
□ 设计促销价格（折扣力度应与毛利、库存和品牌策略一致）
□ 联系头部 Affiliate 提前预热
□ 准备足够覆盖活动周期的专属视频素材
□ 设置 GMV Max 预算（按大促承压能力与库存状态放大）
```

**大促前临近窗口**:
```
□ 发布预热视频（倒计时 / 剧透）
□ 通知所有 Affiliate 促销信息
□ 测试直播设备和流程
□ 确认库存和物流准备
□ 设置促销价格和优惠券
```

**大促当天**:
```
□ 按团队产能安排高强度直播时段
□ 按活动节奏分批发布短视频
□ GMV Max 预算拉满
□ 实时监控库存
□ 实时回复客户消息
```

**大促后短期收尾窗口**:
```
□ 及时发货（承诺时效内）
□ 处理售后问题
□ 数据复盘
□ 感谢表现好的 Affiliate
□ 总结经验教训
```

## 六、TikTok Shop 常见问题与解决方案

| 问题 | 原因分析 | 解决方案 |
|:---|:---|:---|
| 店铺审核不通过 | 资质不全 / 品类限制 | 补充资质 / 更换品类 / 联系客服 |
| 产品被下架 | 违反平台政策 / 侵权 | 修改产品信息 / 提供授权证明 |
| 订单取消率高 | 发货慢 / 库存不足 | 提前备货 / 使用海外仓 |
| 退货率高 | 产品与描述不符 / 质量差 | 优化描述准确性 / 改善产品质量 |
| Affiliate 不出单 | 佣金低 / 产品不吸引人 | 提高佣金 / 优化产品卖点 |
| GMV Max 效果差 | 素材不足 / ROI 目标不合理 | 增加视频 / 调整 ROI 目标 |
| 直播间没人 | 没有预热 / 时段不对 | 提前发预热视频 / 调整直播时间 |
| 差评多 | 产品质量 / 物流问题 | 改善产品 / 更换物流商 / 主动联系解决 |
