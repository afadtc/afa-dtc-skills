# AFA-TT Diagnostic Rules — TikTok 广告诊断规则库

> **版本**: 1.0 | **诊断方法**: CPM-CTR-CVR 诊断链 + 创意疲劳检测 + TikTok Shop 专项诊断

---

## 一、CPM-CTR-CVR 三级诊断链

### 1.1 诊断流程

```
Step 1: 确定异常指标
├── CPA 过高？→ 进入诊断链
├── ROAS 过低？→ 进入诊断链
├── 花不出去钱？→ 跳转 Section 二
└── 数据正常？→ 优化建议

Step 2: 定位瓶颈层级
├── CPM 异常 → Level 1 诊断
├── CTR 异常 → Level 2 诊断
├── CVR 异常 → Level 3 诊断
└── 多层异常 → 按 CPM → CTR → CVR 顺序逐层修复
```

### 1.2 Level 1: CPM 诊断（展示成本）

**健康基准（按品类）**:
| 品类 | 正常 CPM | 警告线 | 危险线 |
|:---|:---:|:---:|:---:|
| 美妆护肤 | $6-$12 | $15 | $20+ |
| 服饰配件 | $5-$10 | $13 | $18+ |
| 家居生活 | $4-$8 | $10 | $15+ |
| 食品保健 | $5-$10 | $12 | $17+ |
| 电子配件 | $7-$14 | $18 | $25+ |
| 宠物用品 | $5-$10 | $13 | $18+ |

**CPM 过高诊断树**:
```
CPM > 基准 1.5x
├── 检查受众规模
│   ├── 受众 < 100 万 → 扩大受众（减少兴趣限制）
│   └── 受众正常 → 下一步
├── 检查竞争环境
│   ├── 大促/节假日期间 → 正常波动，调整预期
│   └── 非大促期间 → 下一步
├── 检查广告质量得分
│   ├── 视频完播率 < 15% → 优化 Hook（前 3 秒）
│   ├── 互动率 < 1% → 优化内容吸引力
│   └── 得分正常 → 下一步
├── 检查出价策略
│   ├── 手动出价过低 → 提高出价 20-30%
│   ├── 自动出价 → 提高预算让系统有更多空间
│   └── 出价正常 → 下一步
└── 检查账户健康
    ├── 近期有违规 → 修复违规，等待恢复
    └── 账户正常 → 可能是素材问题，更换创意
```

**CPM 过低诊断**:
```
CPM < 基准 0.5x（看似好事，但需警惕）
├── 检查流量质量
│   ├── CTR 正常但 CVR 极低 → 多因素排查：流量质量→落地页体验→受众精准度→消息一致性
│   └── CTR 和 CVR 都正常 → 恭喜，继续保持
├── 检查版位
│   ├── 是否投放了低质量版位 → 排除 Pangle 等
│   └── 版位正常 → 继续监控
└── 检查地域
    ├── 是否混入了低 CPM 地区 → 检查地域设置
    └── 地域正常 → 继续监控
```

### 1.3 Level 2: CTR 诊断（点击率）

**健康基准**:
| 广告类型 | 正常 CTR | 警告线 | 优秀线 |
|:---|:---:|:---:|:---:|
| In-Feed 视频广告 | 0.8-2.0% | < 0.5% | > 3.0% |
| Spark Ads | 1.0-3.0% | < 0.7% | > 4.0% |
| TikTok Shop 广告 | 1.5-4.0% | < 1.0% | > 5.0% |
| Collection Ads | 1.0-2.5% | < 0.6% | > 3.5% |

**CTR 过低诊断树**:
```
CTR < 基准下限
├── 检查 Hook（前 3 秒）
│   ├── 3 秒完播率 < 40% → Hook 无效，更换 Hook
│   │   ├── 尝试不同 Hook 类型（痛点/好奇/社证/对比）
│   │   ├── 前 1 秒必须有视觉冲击或语言 Hook
│   │   └── 参考 creative-templates.md 的 20 大 Hook 公式
│   └── 3 秒完播率 > 40% → Hook 有效，问题在后续
├── 检查创意与受众匹配度
│   ├── 受众太宽泛 → 创意无法精准打动 → 收紧受众或定制创意
│   └── 受众精准 → 下一步
├── 检查 CTA 清晰度
│   ├── 无明确 CTA → 添加清晰的行动指引
│   ├── CTA 太弱 → 增加紧迫感/价值感
│   └── CTA 清晰 → 下一步
├── 检查广告文案
│   ├── 文案太长/无吸引力 → 精简 + 加入利益点
│   └── 文案正常 → 下一步
└── 检查缩略图/封面
    ├── 封面不吸引人 → 更换为高对比/有文字的封面
    └── 封面正常 → 综合优化或更换创意
```

**CTR 过高但 CVR 低**:
```
CTR > 基准 2x 但 CVR < 基准 0.5x
├── 可能是"标题党"效应
│   ├── 创意承诺与产品不匹配 → 调整创意，确保一致性
│   └── 创意吸引了错误受众 → 调整受众定向
├── 落地页/产品页问题
│   ├── 加载慢 → 优化页面速度
│   ├── 价格冲击 → 在创意中提前暗示价格区间
│   └── 信任缺失 → 添加评论/社会证明
└── 技术问题
    ├── 追踪是否正常 → 检查 Pixel 事件
    └── 链接是否正确 → 确认落地页 URL
```

### 1.4 Level 3: CVR 诊断（转化率）

**健康基准**:
| 转化类型 | 正常 CVR | 警告线 | 优秀线 |
|:---|:---:|:---:|:---:|
| 加购率（ATC） | 5-12% | < 3% | > 15% |
| 发起结账率 | 3-8% | < 2% | > 10% |
| 购买转化率 | 1.5-4% | < 1% | > 5% |
| TikTok Shop 转化率 | 2-6% | < 1.5% | > 8% |

**CVR 过低诊断树**:
```
CVR < 基准下限
├── 检查产品页面
│   ├── 主图质量差 → 更换高质量产品图
│   ├── 标题不清晰 → 优化标题（关键词 + 卖点）
│   ├── 描述不充分 → 补充详细描述和卖点
│   ├── 评论少/差 → 获取更多正面评论
│   └── 页面正常 → 下一步
├── 检查价格因素
│   ├── 价格高于竞品 → 调整定价或增加价值感
│   ├── 无促销/折扣 → 设置限时优惠
│   ├── 运费过高 → 设置免邮门槛
│   └── 价格合理 → 下一步
├── 检查信任因素
│   ├── 无退款政策 → 添加明确的退款保障
│   ├── 品牌知名度低 → 增加社会证明元素
│   └── 信任正常 → 下一步
├── 检查创意-产品一致性
│   ├── 创意展示的产品与页面不同 → 确保一致
│   ├── 创意承诺的效果过于夸大 → 调整创意
│   └── 一致性正常 → 下一步
└── 检查技术因素
    ├── 页面加载慢 → 优化速度
    ├── 支付方式不全 → 增加支付选项
    └── 移动端体验差 → 优化移动端
```

## 二、花不出去钱（Under-Delivery）诊断

```
日花费 < 预算 50%
├── 检查受众规模
│   ├── 预估覆盖 < 50 万 → 扩大受众
│   │   ├── 减少兴趣标签限制
│   │   ├── 扩大年龄范围
│   │   └── 增加地域
│   └── 受众充足 → 下一步
├── 检查出价
│   ├── 手动出价 → 提高出价 30-50%
│   ├── 成本上限过低 → 提高上限或切换自动出价
│   └── 自动出价 → 下一步
├── 检查创意状态
│   ├── 创意审核中 → 等待（通常 24 小时内）
│   ├── 创意被拒 → 修改后重新提交
│   └── 创意已通过 → 下一步
├── 检查账户状态
│   ├── 账户受限 → 联系客服解除
│   ├── 支付问题 → 更新支付方式
│   └── 账户正常 → 下一步
└── 检查竞争环境
    ├── 大促期间竞争激烈 → 提高出价或等待
    └── 非大促 → 更换创意 + 扩大受众
```

## 三、创意疲劳检测与应对

### 3.1 疲劳信号识别

| 信号 | 指标变化 | 严重程度 |
|:---|:---|:---:|
| CTR 连续 3 天下降 | 日降幅 > 10% | ⚠️ 早期 |
| CPM 连续上升 + CTR 下降 | CPM 涨 20%+ & CTR 降 15%+ | 🔶 中期 |
| 频率 > 3.0 | 同一用户平均看到 3 次+ | 🔴 严重 |
| 互动率骤降 | 点赞/评论/分享减少 50%+ | 🔴 严重 |
| CPA 连续 5 天上升 | 日涨幅 > 5% | 🔴 严重 |

### 3.2 疲劳应对策略

```
⚠️ 早期疲劳（CTR 开始下降）:
├── 更换 Hook（保留后续内容不变）
├── 更换封面图/缩略图
├── 调整文案
└── 预计恢复: 1-2 天

🔶 中期疲劳（CPM 上升 + CTR 下降）:
├── 上新创意（至少 3 条新素材）
├── 调整受众（排除已转化用户）
├── 降低预算 30%，等待新素材
└── 预计恢复: 3-5 天

🔴 严重疲劳（频率 > 3 + CPA 持续上升）:
├── 暂停当前广告组
├── 用全新创意 + 全新受众重建
├── 考虑更换创意角度/格式
├── 重新测试 Hook 和 CTA
└── 预计恢复: 5-7 天
```

### 3.3 创意储备规则

```
创意储备公式:
每 $1,000 日预算 → 至少 5 条活跃创意 + 3 条储备创意

创意更新节奏:
├── 每周上新 2-3 条创意
├── 每月淘汰表现最差的 30% 创意
├── 每季度更新创意方向和风格
└── 大促前 2 周准备 10+ 条新创意
```

## 四、TikTok Shop 专项诊断

### 4.1 GMV Max 诊断

| 问题 | 诊断路径 | 解决方案 |
|:---|:---|:---|
| ROI 低于目标 | 产品页 → 价格 → 视频质量 → ROI 目标设置 | 优化产品页 / 调整价格 / 更新视频 / 降低 ROI 目标 |
| 花不出去钱 | ROI 目标 → 素材数量 → 产品数量 | 降低 ROI 目标 / 增加视频 / 添加更多产品 |
| 点击高不转化 | 视频-产品匹配 → 价格 → 评论 → 物流 | 确保一致性 / 调整价格 / 获取评论 / 优化物流 |
| 退货率高 | 产品描述 → 产品质量 → 视频夸大 | 修正描述 / 改善质量 / 调整创意 |

### 4.2 直播诊断

| 指标 | 健康基准 | 诊断方向 |
|:---|:---|:---|
| 观看人数 | 同时在线 50+ | 预热不足 / 开播时间不对 / 标题不吸引 |
| 平均观看时长 | > 2 分钟 | 内容不够吸引 / 互动不足 / 节奏太慢 |
| 互动率 | > 5% | 缺乏互动引导 / 内容单调 |
| 转化率 | > 2% | 产品展示不充分 / 价格不够吸引 / CTA 不清晰 |
| GPM（千次观看 GMV） | > $50 | 综合问题，逐项排查 |

### 4.3 Affiliate 诊断

| 问题 | 诊断路径 | 解决方案 |
|:---|:---|:---|
| 创作者不发布内容 | 动力 → Brief → 产品 → 沟通 | 提高佣金 / 优化 Brief / 更换产品 / 加强跟进 |
| 内容播放量低 | 内容质量 → Hook → 话题 → 发布时间 | 提供创作指导 / 分享爆款案例 / 建议最佳发布时间 |
| 播放高但不出单 | CTA → 产品页 → 价格 → 链接 | 优化 CTA / 改善产品页 / 调整价格 / 检查链接 |
| 退货率高 | 创作者夸大 → 产品问题 → 受众不匹配 | 更新 Brief / 改善产品 / 筛选创作者 |

## 五、季节性诊断调整

| 时期 | CPM 预期变化 | CTR 预期变化 | 建议 |
|:---|:---:|:---:|:---|
| 1-2 月（淡季） | -20-30% | 持平 | 低成本测试新创意和受众 |
| 3-4 月（回暖） | 持平 | +10% | 开始扩量，准备夏季 |
| 5-6 月（平稳） | 持平 | 持平 | 稳定投放，优化效率 |
| 7-8 月（暑期） | +10-15% | +5% | 适度扩量，返校季主题 |
| 9-10 月（预热） | +15-25% | -5% | 控制成本，储备创意 |
| 11 月（BFCM） | +40-80% | +20% | 全力投放，最大化 GMV |
| 12 月（节日） | +30-50% | +15% | 礼物主题，圣诞前冲刺 |
