# AFA-TT Scaling SOP — TikTok 广告分阶段扩量手册

> **版本**: 1.0 | **方法论**: 分阶段纪律性扩量 + 创意储备驱动

---

## 一、扩量前提条件检查

### 1.1 必须满足的条件

```
□ 至少 3 个广告组连续 3 天 CPA 低于目标
□ 至少有 50 次转化事件（退出学习期）
□ Pixel 数据健康（Match Quality > 6/10）
□ 至少有 5 条经过验证的创意素材
□ 落地页/产品页转化率稳定
□ 库存充足（至少支撑 2 倍当前销量）
□ 客服/物流能力可承受增长
```

### 1.2 扩量风险评估

| 风险因素 | 低风险 | 中风险 | 高风险 |
|:---|:---|:---|:---|
| 当前 CPA vs 目标 | < 70% | 70-90% | > 90% |
| 连续盈利天数 | > 7 天 | 3-7 天 | < 3 天 |
| 活跃创意数量 | > 10 条 | 5-10 条 | < 5 条 |
| 受众规模 | > 500 万 | 100-500 万 | < 100 万 |
| 季节性 | 淡季/平稳 | 回暖期 | 大促前 |

## 二、四阶段扩量路径

### 阶段 1: 验证期（$50-$200/天）

**目标**: 找到可重复的盈利公式

**操作 SOP**:
```
Week 1-2:
├── 每天 $20-$50 × 3-5 个广告组
├── 测试 3 种受众类型:
│   ├── 宽泛受众（无定向）
│   ├── 兴趣受众（2-3 个兴趣组合）
│   └── 相似受众（1% 购买者）
├── 每组 3-5 条创意
├── 出价策略: 最低成本（自动出价）
└── 评估标准: 3 天后看 CPA

判断规则:
├── CPA < 目标 70% → 标记为"明星"，准备扩量
├── CPA 在目标 70-100% → 继续观察 2 天
├── CPA > 目标 120% → 暂停，分析原因
└── 无花费 → 提高出价或扩大受众
```

**关键指标**:
| 指标 | 目标 |
|:---|:---|
| 日花费 | $50-$200 |
| CPA | < 目标值 |
| ROAS | > 盈亏平衡点 |
| 转化数/天 | 5-15 |

### 阶段 2: 初步扩量（$200-$1,000/天）

**目标**: 放大已验证的盈利组合

**操作 SOP**:
```
Week 3-4:
├── 对"明星"广告组逐步增加预算
│   ├── 每次增加不超过 20%（与全局扩量标准统一）
│   ├── 增加频率: 每 2-3 天一次
│   └── 如果 CPA 上升 > 20%，回退到上一级预算
├── 复制"明星"广告组到新系列
│   ├── 使用 CBO（系列预算优化）
│   ├── 初始预算 = 单组最佳预算 × 广告组数
│   └── 让 TikTok 算法分配预算
├── 开始测试更多创意
│   ├── 每周上新 3-5 条创意
│   ├── 基于"明星"创意的变体（换 Hook/换 CTA/换角度）
│   └── 同时测试全新方向
└── 扩展受众
    ├── 相似受众从 1% → 2-3%
    ├── 增加新的兴趣组合
    └── 测试宽泛受众 + 强创意

回退规则:
├── 连续 2 天 CPA > 目标 130% → 降低预算 30%
├── 单日 CPA > 目标 200% → 暂停该广告组
└── 整体 ROAS < 盈亏平衡 → 回退到阶段 1 预算
```

**关键指标**:
| 指标 | 目标 |
|:---|:---|
| 日花费 | $200-$1,000 |
| CPA | < 目标值 × 1.1 |
| ROAS | > 盈亏平衡 × 1.2 |
| 转化数/天 | 15-50 |
| 活跃创意数 | 10+ |

### 阶段 3: 规模化扩量（$1,000-$5,000/天）

**目标**: 系统化规模化，建立创意机器

**操作 SOP**:
```
Month 2-3:
├── 账户结构升级
│   ├── 测试系列: 30% 预算（持续测试新创意和受众）
│   ├── 扩量系列: 50% 预算（已验证的组合）
│   ├── 再营销系列: 15% 预算（网站访客/加购/互动）
│   └── TikTok Shop 系列: 5% 预算（如适用）
│
├── 创意机器建设
│   ├── 每周产出 5-10 条新创意
│   ├── 建立 UGC 创作者网络（5-10 个稳定合作）
│   ├── 创意类型多样化:
│   │   ├── UGC 测评（40%）
│   │   ├── 问题-解决方案（20%）
│   │   ├── Before-After（15%）
│   │   ├── 教程/How-to（15%）
│   │   └── 品牌故事（10%）
│   └── Hook 矩阵: 每条创意测试 3 个 Hook
│
├── 受众扩展
│   ├── 相似受众 3-5%
│   ├── 宽泛受众（让算法找人）
│   ├── Spark Ads（利用有机内容）
│   └── 跨平台受众（上传 Meta/Google 客户数据）
│
└── 自动化规则设置
    ├── CPA > 目标 150% 持续 2 天 → 自动降预算 30%
    ├── CPA < 目标 70% 持续 3 天 → 自动提预算 20%
    ├── CTR < 0.5% → 自动暂停
    └── 频率 > 3.0 → 自动暂停

回退规则:
├── 周 ROAS < 盈亏平衡 → 砍预算 40%，回到阶段 2
├── 创意储备 < 5 条 → 暂停扩量，专注创意生产
└── 退货率 > 20% → 暂停扩量，解决产品/物流问题
```

**关键指标**:
| 指标 | 目标 |
|:---|:---|
| 日花费 | $1,000-$5,000 |
| CPA | < 目标值 × 1.15 |
| ROAS | > 盈亏平衡 × 1.15 |
| 转化数/天 | 50-200 |
| 活跃创意数 | 20+ |
| 创意更新频率 | 每周 5-10 条 |

### 阶段 4: 成熟运营（$5,000+/天）

**目标**: 稳定高效运营，持续增长

**操作 SOP**:
```
Month 4+:
├── 多账户策略
│   ├── 主账户: 核心扩量
│   ├── 测试账户: 新创意/新受众测试
│   └── 备用账户: 风险分散
│
├── 全渠道 TikTok 策略
│   ├── In-Feed Ads（核心）
│   ├── Spark Ads（UGC 放大）
│   ├── TikTok Shop Ads（直接转化）
│   ├── GMV Max（自动化）
│   ├── Affiliate 机器（规模化 UGC）
│   └── 直播（高 GPM 场景）
│
├── 创意工业化
│   ├── 每周产出 10-20 条新创意
│   ├── UGC 创作者网络 20+ 人
│   ├── 内部创意团队 + 外部创作者协同
│   ├── 创意数据库: 分类、标签、表现追踪
│   └── 创意灵感: 每日监控竞品和趋势
│
├── 数据驱动优化
│   ├── 每日: 检查核心指标，处理异常
│   ├── 每周: 创意表现分析，上新/淘汰决策
│   ├── 每月: 全面复盘，策略调整
│   └── 每季: 大方向评估，预算重新分配
│
└── 风险管理
    ├── 预算不超过总营收的 25-30%
    ├── 单一广告组不超过总预算的 20%
    ├── 保持 2-3 个备用账户
    └── 创意储备至少 2 周用量
```

## 三、扩量常见问题与解决方案

### 3.1 扩量后 CPA 飙升

```
原因分析:
├── 预算增加过快（> 50%/次）→ 算法重新学习
├── 受众饱和 → 频率上升
├── 创意疲劳 → CTR 下降
└── 竞争加剧 → CPM 上升

解决方案:
├── 立即: 回退预算到上一个稳定水平
├── 短期: 上新创意 + 扩展受众
├── 中期: 建立更大的创意储备
└── 长期: 多元化流量来源
```

### 3.2 扩量后转化率下降

```
原因分析:
├── 流量质量下降（受众变宽）
├── 落地页承载能力不足
├── 产品页面没有针对新受众优化
└── 库存/物流问题影响用户体验

解决方案:
├── 检查受众质量（对比新老受众的 CVR）
├── A/B 测试落地页变体
├── 针对不同受众定制产品页
└── 确保运营能力跟上增长
```

### 3.3 花不出去预算

```
原因分析:
├── 受众太窄
├── 出价太低
├── 创意审核未通过
├── 竞争激烈期间（大促）
└── 账户受限

解决方案:
├── 扩大受众范围（减少限制条件）
├── 提高出价 30-50%
├── 检查创意是否符合平台政策
├── 调整投放时段
└── 联系 TikTok 客服检查账户状态
```

## 四、创意储备与扩量的关系

### 4.1 创意储备公式

```
最低创意储备 = 日预算 / $200 × 3 条

示例:
├── $500/天 → 至少 8 条活跃创意
├── $1,000/天 → 至少 15 条活跃创意
├── $3,000/天 → 至少 45 条活跃创意
└── $5,000/天 → 至少 75 条活跃创意

创意更新速度 = 活跃创意数 × 30% / 周

示例:
├── 15 条活跃 → 每周上新 5 条
├── 45 条活跃 → 每周上新 14 条
└── 75 条活跃 → 每周上新 23 条
```

### 4.2 创意不足时的扩量策略

```
如果创意储备不足:
├── 暂停扩量，专注创意生产
├── 利用 Spark Ads 放大有机内容（不需要新创意）
├── 使用 TikTok 的动态创意优化（DCO）
├── 基于现有创意做变体（换 Hook/换 CTA/换音乐）
└── 启动 Affiliate 计划获取 UGC 内容
```
