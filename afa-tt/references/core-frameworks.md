## 二、2026 TikTok 广告新范式

### 2.1 范式转变：从"投广告"到"做内容"

| 旧范式（2023-2024） | 新范式（2025-2026） |
|---|---|
| 广告 = 打断用户 | **广告 = 原生内容** |
| 高制作 = 高转化 | **真实感 > 制作质量** |
| 精准定位 = 核心 | **创意 = 新的定位** |
| 手动出价 = 控制 | **AI 协同 = 效率** |
| 直投网站 = 唯一路径 | **TikTok Shop + 直投 = 双轨** |
| 品牌自产内容 | **Affiliate + UGC = 内容引擎** |

### 2.2 TikTok 生态全景

```
┌─────────────────────────────────────────────────┐
│                TikTok 商业化生态                  │
├──────────────┬──────────────┬───────────────────┤
│  TikTok Ads  │ TikTok Shop  │   Organic/SEO     │

│ • Spark Ads  │ • 短视频购物  │ • 话题标签策略     │
│ • GMV Max    │ • Affiliate   │ • 评论区运营       │

```

### 2.3 TikTok vs Meta vs Google 定位

| 维度 | TikTok | Meta | Google |
|---|---|---|---|
| **用户意图** | 发现 & 娱乐 | 社交 & 发现 | 搜索 & 意图 |
| **最强阶段** | 认知 → 兴趣 | 兴趣 → 考虑 | 考虑 → 购买 |
| **创意风格** | 原生、真实、快节奏 | 多样化、可精致 | 文字为主、购物图片 |
| **最佳品类** | 视觉冲击力强、可演示 | 全品类 | 高搜索量品类 |
| **典型 ROAS** | 常见于波动较大的中位区间 | 常见于相对更稳的中高区间 | 常见于更稳定的中高区间 |
| **扩量天花板** | 中高（受创意消耗影响） | 高 | 中（受搜索量限制） |

## 三、TikTok Ads 账户架构

### 3.1 Business Center 架构

```
Business Center（商务中心）
├── Ad Account #1（主账户 - 直投网站）
│   ├── Campaign: Prospecting
│   │   ├── Ad Group: Broad（宽泛定位）
│   │   ├── Ad Group: Interest Stack（兴趣叠加）
│   │   └── Ad Group: Lookalike（相似受众）
│   ├── Campaign: Retargeting
│   │   ├── Ad Group: Website Visitors 7d
│   │   └── Ad Group: Engaged Users 14d
│   └── Campaign: Testing
│       └── Ad Group: Creative Test（创意测试）
├── Ad Account #2（TikTok Shop 账户）
│   ├── Campaign: GMV Max
│   └── Campaign: Product Shopping Ads
├── TikTok Account（品牌账号）
├── Pixel（追踪像素）
└── Members（团队成员）
    ├── Admin（管理员）
    ├── Advertiser（投手）
    └── Analyst（分析师）
```

### 3.2 2026 推荐账户结构

**原则：简化结构 + AI 协同 + 创意为王**

#### 直投网站结构（Standard Ads）

| 层级 | 设置 | 说明 |
|---|---|---|
| **Campaign 1** | Conversions - Purchase | 主转化系列 |
| → Ad Group 1 | Broad Targeting，宽年龄带，自动版位 | 让算法找人，不限制兴趣 |
| → Ad Group 2 | Interest Stack（少量核心兴趣叠加） | 给算法方向性引导 |
| → Ad Group 3 | Lookalike（基于购买者种子） | 种子受众扩展 |
| **Campaign 2** | Conversions - Purchase | 再营销系列 |
| → Ad Group 1 | 近期 Website Visitors（排除已购买） | 近期高意向人群 |
| → Ad Group 2 | 近期 Engaged Users（排除已购买） | 互动过但未购买 |
| **Campaign 3** | Conversions - ATC/IC | 创意测试系列 |
| → Ad Group 1 | Broad，轻量测试预算 | 快速验证创意 |

#### TikTok Shop 结构（GMV Max + Shopping Ads）

| 层级 | 设置 | 说明 |
|---|---|---|
| **Campaign 1** | GMV Max | AI 全自动优化 |
| → 自动选品 | 系统选择最优商品 | 适合 SKU 多的品牌 |
| **Campaign 2** | Product Shopping Ads | 手动控制 |
| → Ad Group 1 | 主推商品 | 手动选择高转化商品 |
| → Ad Group 2 | 新品测试 | 新上架商品 |
| **Campaign 3** | Live Shopping Ads | 直播引流 |
| → Ad Group 1 | 直播间引流 | 直播期间投放 |

### 3.3 预算分配框架

#### 按阶段分配

| 阶段 | 日预算强度 | 直投网站 | TikTok Shop | 创意测试 |
|---|---|---|---|---|
| **冷启动早期** | 以可控测试预算起步 | 通常占大头 | 作为补充验证 | 保留基础测试份额 |
| **验证期** | 随稳定度逐步放大 | 仍常是主要部分 | 占比可逐步提升 | 适度提高测试份额 |
| **扩量期** | 按承压能力继续放量 | 与 TikTok Shop 可能趋于接近 | 视闭环表现提升占比 | 保持持续测试 |
| **成熟期** | 预算强度更高 | 视站内外回收效率动态调整 | 视闭环效率动态调整 | 不应被完全压缩 |

#### 按漏斗分配（直投网站部分）

| 漏斗层 | 预算方向 | 目标 |
|---|---|---|
| Prospecting（拉新） | 通常占主要部分 | 新客获取 |
| Retargeting（再营销） | 保留中等份额 | 转化犹豫用户 |
| Creative Testing（测试） | 保留基础但持续的份额 | 验证新创意 |

## 四、季节性 TikTok 广告策略

> **激活条件**：当 Hub 传入 `seasonal_mode = off_season` 或 `seasonal_mode = pre_season` 时，本模块自动激活。当用户提到"淡季""TikTok 广告效果变差""要不要暂停 TikTok"时，也应主动引用本模块。

### 4.1 TikTok 淡季预算策略

TikTok 的算法与 Meta 类似，依赖持续的转化数据来优化投放。暂停投放会导致算法"失忆"，重启时需要重新学习。

```
TikTok 淡季预算原则：

1. 不要轻易 Go Dark
   → 最低维持预算应由旺季日预算、学习成本和现金流承压能力共同决定
   → 长时间完全暂停往往会导致算法重新学习，重启成本通常高于保守维持

2. 淡季预算分配
   ├── Retargeting：通常应保留较高优先级
   ├── 创意测试：通常应保留稳定份额
   ├── 品牌内容推广：按品牌认知目标保留一定预算
   └── Spark Ads / UGC 推广：按社会认同与素材复用价值安排

3. 预热期（旺季前的准备窗口）
   → 预算可按稳定度逐步提升，而不是一次性放量
   → 重新激活 Broad 受众
   → 测试旺季创意方向
   → 确保 Pixel 数据已累积到足以支撑放量判断的水平
```

### 4.2 淡季目标转移

| 维度 | 旺季 | 淡季 |
|------|------|------|
| **Campaign 目标** | Conversions / GMV Max | Traffic / Video Views / Engagement |
| **核心 KPI** | ROAS, CPA, GMV | CPM 效率, 视频完播率, 粉丝增长, 互动率 |
| **创意方向** | 产品导向、促销导向、直接 CTA | 品牌故事、教育内容、幕后花絮、UGC |
| **受众策略** | Broad + Lookalike（转化优化） | Interest-based + 视频互动受众 |
| **TikTok Shop** | 全力推 GMV Max + 直播 | 维持基础 listing，测试新品 |
| **Affiliate** | 大量达人带货 | 维持核心达人关系，测试新达人 |

### 4.3 淡季创意测试窗口

TikTok 的淡季 CPM 往往低于旺季，因此常是相对更适合低成本测试新创意的窗口；但具体差值仍要结合品类竞争度、节庆密度与账户历史表现判断。

```
淡季创意测试计划：

每周测试预算：应占淡季总预算中的稳定测试份额

测试维度（按优先级）：
  1. 新 Hook 变体（前几秒）
     → TikTok 用户通常会在极短时间内决定是否继续看
     → 淡季可测试足够数量的 Hook 变体，旺季保留表现最强的一批

  2. 新内容格式
     → 绿幕评论回复、拆箱、对比、教程、GRWM
     → 淡季测试哪种格式在你的品类最有效

  3. 新创作者/达人风格
     → 淡季招募和测试新的 UGC 创作者
     → 建立旺季的创作者储备池

  4. 新产品角度
     → 功能 vs 情感 vs 社会认同 vs 问题解决
     → 淡季验证哪个角度转化最好

旺季前的收口窗口：
  → 逐步收敛新测试
  → 锁定表现最强的一批创意与 Hook
  → 准备可覆盖旺季节奏的素材库
  → 确保创作者产能可以支撑旺季需求
```

### 4.4 Affiliate 和达人关系维护

淡季不能断开与达人和 Affiliate 的联系，否则旺季重新建立关系成本更高。

```
淡季达人/Affiliate 策略：

1. 核心达人维护（通常由少数高贡献达人贡献大部分 GMV）
   → 保持定期沟通
   → 提前分享新品信息和旺季计划
   → 提供独家折扣码维持合作关系
   → 邀请参与产品开发/反馈

2. 新达人招募和测试
   → 淡季 CPM 较低时，测试新达人的成本通常也更友好
   → 按团队产能持续测试一批新达人
   → 建立旺季达人储备池，并让可用达人规模明显高于淡季基线

3. 内容储备
   → 与达人合作拍摄品牌故事、教育内容
   → 积累 UGC 素材库
   → 为旺季准备"弹药库"

4. Affiliate 佣金策略
   → 淡季可适当提高佣金率（激励达人在低需求期持续推广）
   → 旺季回调到标准佣金率
```

### 4.5 季节性下降时的正确反应

| 错误反应 | 为什么错 | 正确做法 |
|---------|---------|----------|
| 暂停所有 TikTok 广告 | 算法失忆 + 达人关系断裂 | 降至维持预算，保留表现最稳的一批创意 |
| 追求淡季高 ROAS | 自然低需求期强求效率 | 接受淡季 ROAS 可能下降，关注品牌指标 |
| 停止发布有机内容 | 算法惩罚不活跃账号 | 保持稳定的有机更新节奏 |
| 解散达人团队 | 旺季重建成本极高 | 维持核心达人，淡季测试新达人 |
| 停止 TikTok Shop 运营 | 失去 Shop 权重和评分 | 维持基础 listing 和评价管理 |
