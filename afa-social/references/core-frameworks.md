# 核心框架

> 本文件为社交增长模块的内部参考文件，用于沉淀社交电商内容策略、平台选择、社交搜索优化与内容复用机制的核心方法论。
> 如需整理为用户可见交付物，必须删除内部路由标签、模块代号和系统字段，只保留自然语言、业务角色与行动建议。

## 1. 2026 社交电商新范式
- **社交搜索成为新 SEO**：Z 世代和 Alpha 世代正在用 TikTok 和 Instagram 替代 Google 进行产品搜索。
- **AI 生成内容的边界**：虽然 AI 可以辅助脚本和文案，但**真实的人脸、真实的声音和真实的产品展示**是建立信任的唯一途径。
- **短视频电商闭环**：TikTok Shop 和 IG Shopping 缩短了转化路径。内容不再只是"引流"，而是直接的"销售终端"。
- **社区驱动增长 (CLG)**：从"向受众广播"转变为"与社区共创"。最好的营销人员是你的客户。

## 2. 电商内容策略框架
不要盲目追逐趋势。一个健康的电商社交账号需要平衡三个内容支柱：

### 2.1 三大内容支柱
1. **教育与价值 (Education & Value) - 40%**
   - **目的**：建立权威，解决痛点，展示产品"如何工作"。
   - **形式**：教程、成分解析、常见问题解答、行业迷思破除。
2. **娱乐与连接 (Entertainment & Connection) - 40%**
   - **目的**：吸引注意力，建立品牌个性，引发共鸣。
   - **形式**：趋势音乐、幕后花絮、创始人故事、相关 Meme。
3. **销售与转化 (Sales & Conversion) - 20%**
   - **目的**：直接驱动购买。
   - **形式**：限时优惠、产品发布、强力 UGC 评价、对比视频。

### 2.2 平台优先级矩阵
不要试图在所有平台上都做到完美。选择一个主攻平台，其他平台作为分发渠道。

| 平台 | 核心机制 | 电商角色 | 适用品牌类型 |
|:---|:---|:---|:---|
| **TikTok** | 兴趣图谱 (发现) | 病毒式获客、冲动购买 | 视觉冲击力强、价格<$50、年轻受众 |
| **Instagram** | 社交图谱 (关系) | 品牌美学、社区留存、高客单价转化 | 美妆、时尚、生活方式、高客单价 |
| **YouTube Shorts** | 搜索意图 (长尾) | 深度教育、长期 SEO 流量 | 科技、复杂产品、需要深度解释的品类 |
| **Pinterest** | 视觉搜索 (规划) | 早期意图捕捉、高 AOV 转化 | 家居、婚礼、礼品、DIY |

## 3. 社交搜索 SEO (Social Search Optimization)
Z 世代和 Alpha 世代正在用 TikTok 和 Instagram 替代 Google 进行产品搜索。你的内容必须针对平台内搜索进行优化。

### 3.1 TikTok SEO 核心策略
TikTok 的搜索算法依赖三个信号层：**口播词汇 (Speech-to-Text)**、**屏幕文本 (On-Screen Text)** 和 **标题/标签 (Caption & Hashtags)**。三者必须协同包含目标关键词。

**关键词研究方法**：
1. **搜索栏自动补全**：在 TikTok 搜索栏输入核心词，记录所有自动补全建议。
2. **评论区挖掘**：查看竞品热门视频的评论，提取用户使用的自然语言。
3. **"Others searched for" 面板**：搜索结果页底部的相关搜索词。
4. **Google Trends 交叉验证**：确认关键词的搜索趋势是否上升。

**优化清单**：
- 在视频前 5 秒**口播**目标关键词（TikTok 的语音转文字会索引）。
- 在视频中添加包含关键词的**屏幕文本**。
- 标题中自然融入 2-3 个关键词（不要堆砌）。
- 使用 3-5 个相关 Hashtag（混合大词和长尾词）。
- 视频描述中包含完整的关键词短语。

### 3.2 Instagram SEO 核心策略
Instagram 的搜索依赖 **用户名 (Username)**、**简介 (Bio)**、**标题 (Caption)** 和 **Alt Text**。
- **用户名和简介**：包含品类关键词（如 "@brandname | Clean Skincare"）。
- **标题**：前两行包含核心关键词，因为这是搜索索引的重点。
- **Alt Text**：为每张图片/视频添加描述性的 Alt Text。
- **位置标签**：如果有实体店或目标地区，始终添加位置标签。

### 3.3 Pinterest SEO 核心策略
Pinterest 本质上是一个视觉搜索引擎。SEO 是 Pinterest 增长的核心。
- **Pin 标题**：包含精确的搜索关键词（如 "Minimalist Living Room Decor Ideas"）。
- **Pin 描述**：150-300 字的关键词丰富描述。
- **Board 名称**：使用搜索友好的名称（如 "Small Apartment Organization Ideas" 而非 "My Inspo"）。
- **Rich Pins**：启用产品 Rich Pins，自动同步价格和库存信息。

## 4. 一鱼多吃矩阵 (Content Repurposing Matrix)
高效的内容团队不会为每个平台单独创作。他们从一次拍摄中提取 8+ 个资产。

### 4.1 核心原则
**一次拍摄，多次收割**。每次内容拍摄都应该被视为一次"素材采集"，而不是"制作一条视频"。

### 4.2 从 1 条长视频到 8+ 资产的拆解路径
| 源素材 | 衍生资产 | 平台 | 调整要点 |
|:---|:---|:---|:---|
| 1 条 60s 产品演示视频 | 原版 60s | TikTok | 原生发布 |
| | 15s 精华剪辑（最佳 Hook + 结果） | IG Reels | 调整安全区 |
| | 15s 精华剪辑（不同 Hook） | YT Shorts | 添加结尾卡片 |
| | 静态截图 + 文字覆盖 | Pinterest Pin | 竖版 2:3 比例 |
| | 3 张轮播图（步骤拆解） | IG Carousel | 教育型叙事 |
| | GIF/短循环 | 邮件营销 | 嵌入产品邮件 |
| | 评论回复视频（回复热门评论） | TikTok | 利用原始视频的热度 |
| | 幕后花絮（拍摄过程） | IG Stories | 真实感、亲近感 |

### 4.3 批量拍摄日 SOP
一次高效的拍摄日可以产出 2-4 周的内容：
1. **拍摄前（1-2 天）**：确定 10-15 个脚本/大纲，准备产品、道具、服装。
2. **拍摄日上午**：集中拍摄所有需要相同布景/光线的视频。先拍最重要的。
3. **拍摄日下午**：换场景/服装，拍摄剩余视频。多拍 B-roll（产品特写、手部动作、环境镜头）。
4. **拍摄后（2-3 天）**：批量剪辑，按平台调整格式，排期发布。

**关键提示**：每个视频至少拍摄 3 个不同的 Hook 开头。这样在后期可以直接进行 Hook 测试，而不需要重新拍摄。
