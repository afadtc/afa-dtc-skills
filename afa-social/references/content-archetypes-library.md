# 电商内容原型库 (Content Archetypes Library)

本指南详细拆解了 15+ 种经过验证的高转化电商内容原型，包括适用场景、结构框架、Hook 示例和视觉执行要点。

## 1. 产品发布序列 (Product Launch Sequence)
产品发布不是一条帖子，而是一个建立期待并将兴趣转化为销售的内容序列。

### 适用场景
- 新品上市、限量版发售、重大功能升级。

### 结构框架
- **D-7 (预热)**："有些新东西要来了..." / 仅展示局部特写。
- **D-3 (幕后)**："研发了 2 年..." / 展示过程。
- **D-1 (揭晓)**：完整产品揭晓、功能、价格。
- **D-0 (发布日)**："它来了。" / 开箱、评价、购物链接。
- **D+3 (社会认同)**：客户反应、首批评价、"卖得很快"。

### 视觉执行
- 预热期使用低光、特写或模糊处理。
- 揭晓期使用高对比度、清晰的产品英雄图。

## 2. 开箱体验 (Unboxing)
触发期待和奖励回路。观众感同身受地体验"新产品兴奋感"。

### 适用场景
- 包装精美的产品、订阅盒、高客单价商品。

### 结构框架
- **展示密封包裹** -> **缓慢打开 (ASMR)** -> **真实反应揭晓** -> **快速展示产品** -> **最爱单品 + CTA**。

### Hook 示例
- "我的 [品牌] 订单刚到 😍"
- "看看 [产品] 是否名副其实"

## 3. Before/After 转变
对于任何能产生可见变化的产品来说，这是最强大的格式。

### 适用场景
- 护肤、清洁、收纳、健身、美发产品。

### 结构框架
- **展示 Before（痛点）** -> **引入产品** -> **展示使用过程** -> **揭晓 After（戏剧性差异）** -> **CTA**。

### Hook 示例
- "我不敢相信这个差异"
- "POV：你终于找到了真正有效的东西"

### 关键规则
- 应保持真实。夸大或虚假的转变会破坏信任。视觉上宜“低承诺，高交付”。

## 4. Get Ready With Me (GRWM)
美妆、时尚和生活方式品牌的原生格式。产品自然融入日常。

### 适用场景
- 服装、化妆品、护肤品、配饰。

### 结构框架
- **从"凌乱"开始** -> **逐步进行日常** -> **自然展示产品** -> **快速说出喜爱之处** -> **最终造型揭晓** -> **CTA**。

### Hook 示例
- "和我一起准备去 [场合]"
- "只用 [品牌] 产品的 GRWM"

## 5. 购物分享 (Haul)
展示来自一个品牌或一次购物的多个产品。

### 适用场景
- 快时尚、美妆集合店、家居装饰。

### 结构框架
- **展示所有战利品** -> **逐一介绍** -> **试穿/演示** -> **给出诚实评价** -> **"最佳"排名** -> **CTA**。

### Hook 示例
- "我的 [品牌] Haul —— 我有点买疯了 😅"
- "评价我从 [品牌] 买的所有东西"

## 6. 客户故事 (Customer Story)
比任何广告都强大，因为它是真实的人和真实的故事。

### 适用场景
- 解决严重痛点的产品（如祛痘、缓解疼痛、提高效率）。

### 结构框架
- **介绍客户及痛点** -> **如何发现产品** -> **使用体验** -> **结果/转变** -> **推荐** -> **CTA**。

## 7. 补货提醒 (Restock Alert)
同时创造紧迫感和社会认同。售罄证明了需求。

### 适用场景
- 经常断货的热销单品。

### 结构框架
- **"它回来了"** -> **提醒为何售罄** -> **为何值得买** -> **紧迫感元素** -> **CTA**。

### Hook 示例
- "它回来了。那个在 [X 小时] 内售罄的 [产品]"。

## 8. 对比 / "这个 vs 那个" (Comparison)
帮助买家在产品之间做出选择，或将你的产品与替代品进行定位。

### 适用场景
- 市场上有明确竞品，或产品线内有多个相似选项。

### 结构框架
- **展示两个选项** -> **在若干具体标准上对比** -> **展示使用情况** -> **给出诚实结论** -> **CTA**。

### Hook 示例
- "[你的产品] vs [替代品]：哪个真正更好？"
- "我测试了两个，这样你就不用测了"

### 关键规则
- 保持公平。如果与竞品对比，承认竞品的优点。片面的对比会显得有偏见。

## 9. 教程 / 操作指南 (Tutorial / How-To)
展示如何使用产品以体现其价值。

### 适用场景
- 需要一定学习成本的产品（如复杂护肤步骤、工具、软件）。

### 结构框架
- **说明结果** -> **展示所需工具** -> **逐步演示** -> **最终结果** -> **专业提示** -> **CTA**。

### Hook 示例
- "如何在 [X] 分钟内 [达到结果]"
- "停止犯 [常见错误]。这样做。"

## 10. 创始人故事 (Founder Story)
建立深厚的情感连接。

### 适用场景
- 品牌初期、里程碑时刻、应对公关危机。

### 结构框架
- **个人经历的痛点** -> **现有解决方案为何不够好** -> **决定创造什么** -> **挣扎/旅程** -> **现状** -> **品牌主张**。

## 11. 趋势劫持 (Trend Jacking)
利用热门音乐、格式或文化时刻。

### 适用场景
- 提高品牌知名度、触达年轻受众。

### 关键规则
- 应与品牌相关；在趋势仍有讨论热度时尽快行动；改编而非照搬；如果不合适就跳过。

## 12. 打破迷思 (Myth Busting)
将品牌定位为权威。

### 适用场景
- 行业内存在普遍误解的品类（如营养补充剂、护肤成分）。

### 结构框架
- **陈述常见迷思** -> **解释为何错误（科学/逻辑）** -> **正确的做法** -> **你的产品如何契合** -> **CTA**。

## 13. 评论回复 (Comment Reply)
利用 TikTok/IG 的视频回复评论功能。

### 适用场景
- 解答常见问题、回应质疑、放大正面评价。

### 策略
- 回复常见问题、质疑或高度赞扬的评论。这证明你在倾听，并为产品教育提供了自然的切入点。

## 14. 包装/幕后 (Pack an Order With Me)
展示对细节的关注和小型企业的真实感。

### 适用场景
- 手工品牌、定制产品、注重包装体验的品牌。

### 结构框架
- **念出客户名字** -> **挑选产品** -> **精心包装 (ASMR)** -> **放入感谢卡** -> **封箱** -> **CTA**。

## 15. 争议性观点 (Controversial Opinion / Hot Take)
产生大量评论（同意和反对都是参与度）。

### 适用场景
- 挑战行业现状的颠覆性品牌。

### 结构框架
- **陈述不受欢迎的观点** -> **解释原因（逻辑论证）** -> **你的立场/解决方案** -> **CTA**。

### Hook 示例
- "不受欢迎的观点：你不需要 10 步护肤法。你需要 3 个产品。最多。"
